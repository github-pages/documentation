Exporting a service definition

On the Export service definition page, you can export the definitions of technical services you provided and save the data to an XML file. You can edit the XML file as desired or use the file as a sample for new service definitions. The format and usage of the XML tags in the file are described in detail in the Technology Provider's Guide.

To export technical service definitions:

  1. Select the check boxes of one or several technical services whose definitions you want to export. If you select several services, their definitions are written to the same file.
  2. Click Export.
  3. Choose whether to save the definitions to an XML file, or to open them instantly in an editor of your choice.
