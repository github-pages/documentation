Adding payment types

On the New payment type page, you can define a payment type offered by the selected payment service provider (PSP). The payment types defined here can later be made available to supplier and reseller organizations. The suppliers and resellers, in turn, can select the payment types they want to make available to customers.

You can add payment types, but not remove a type already defined for the PSP. The names of the payment types can be freely defined. The payment types themselves are provided by the PSP.

To add a payment type:

  1. In the Name field, enter a name for the payment type as it is to be shown to suppliers, resellers, and customers.

    The name can consist of a maximum of 100 characters.

  2. In the Payment type field, enter the identifier of the payment type as defined by the PSP.
  3. To save the payment type, click Save.
    Note: You can no longer remove a payment type once saved.

You are returned to the page for configuring the PSP.
