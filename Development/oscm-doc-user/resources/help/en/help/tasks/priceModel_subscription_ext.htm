Defining a subscription price model

On the Define for subscription page, you can define or update the price model for a specific subscription of a selected customer.

You have selected a subscription to a service that is charged for using an external billing system. The price models for calculating the charges are also defined and managed in this billing system. You need to import the appropriate price model for the subscription in the platform. You can view the price model but not modify it on the platform. To change the price model, you need to import it again after the relevant information has been updated in the external billing system.

To define or update a subscription price model:

  1. In the Define the price model section,click Import to import the price model from the external system. The billing adapter, that connects the platform with the external billing system, is responsible for providing the correct price model. You can view the price model by clicking Display.
  2. To save the price model, click Save.
The price model changes for the subscription take effect immediately.
