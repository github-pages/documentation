Managing a service

On the Manage service page, you can view and update the details of the services which you are authorized to sell.

The following information on a service is available:

  • The service name and short description which are shown in the service catalog on a marketplace when you activate the service.
  • The service description which offers details of the features offered by the service. It is displayed on a marketplace when customers request details of the service.
  • The license agreement which must be read and accepted by customers when they subscribe to the service.
  • The details of the price model defined by the supplier of the service:
    • If an external billing system is used for calculating the charges for the service, you can view the charging conditions and prices as defined in this system in a separate window by clicking Display.
    • If the platform's native billing system is used, the service can be free of charge or with costs. If customers are charged for using the service, the charging conditions and prices are shown. Click here for more details. All prices are without VAT.

The information can only be viewed, except for the license agreement. You can update the text for your customers, if required. As a prerequisite, the service must be deactivated. Existing subscriptions of your customers are not affected by the update. Any changes you make are effective for new subscriptions only. To provide the service to customers for subscription, you must activate it.

To view and update the details of a service:

  1. From the Service ID list, select the service for which to view or update the details.

    The list contains all services you are authorized to sell.

  2. In the License agreement field, update the license agreement text for the service, if required. The field is initially preset with the license agreement as defined by the supplier of the service. It is up to you whether you use this license agreement or specify a different one.
    Note: Make sure to use only the controls provided in the editor for formatting your text. Manually entered HTML tags and references to external style sheets or content may result in errors in the page layout.
  3. To save your changes, click Save. The text is stored in the language defined by your user profile.
  4. To provide the license agreement in another language, change the language in which you currently work to the required language in your user profile using the Edit profile menu option in the Account menu. You can then enter the corresponding translations.

The service is displayed with the updated definition.

To make the service available for subscription to customers, you must first specify the marketplace to which you want to publish it. You can then activate the service.
