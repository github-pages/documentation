Editing a custom attribute

On the Edit custom attribute page, you can edit or delete a selected custom attribute.

To edit the custom attribute:

  1. Change the data by editing the fields.
    • Attribute ID: Identifier or name for the attribute. Attribute IDs are restricted to 40 characters and must not contain any of the following characters: ! " # $ % & ' * + , / : ; < > \ ^ `
    • Default value: Default value of the attribute. The default value is restricted to 255 characters. It is used as long as no specific value for an individual customer or an individual subscription is entered. If the value is to be specified by the customer, the corresponding field is preset with this default value during the subscription process.
    • User option: Check box to be selected if the attribute is to be offered to a customer's administrator or subscription manager when he subscribes to one of your services. This allows the administrator or subscription manager to enter the value for the attribute.
    • Mandatory: Check box to be selected if you want to make the field for the attribute value mandatory for the customer's administrator or subscription manager.
  2. To save your changes, click Save.

You are returned to the page for managing custom attributes. The updated definition of the custom attribute is instantly displayed.

An updated attribute name or an updated default value will be included in the billing data for customers the next time the data is generated. If no value is specified for an attribute, the attribute name will be inserted into the billing data without a value.

To delete the custom attribute:

  • Click Delete and confirm the deletion.

The custom attribute is instantly deleted, and you are returned to the Manage custom attributes page. The attribute will no longer be included in the billing data for a customer the next time the data is generated.
