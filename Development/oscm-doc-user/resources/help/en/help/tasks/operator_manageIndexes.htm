Updating the search index

The search index is automatically and periodically created by the system. Users on marketplaces make use of the index when searching for subscriptions or services.

In some cases, for example, after a recovery of the database, the search index may become inconsistent. In order to solve this problem, you can use the Update search index option to explicitly re-create the search index. The search index is re-built based on the current data in the database. The existing index structure is overwritten.

To start the re-creation of the search index, click Update on the Update search index page.
