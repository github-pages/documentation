Updating configuration settings

On the Update configuration settings page, you see a list of the current configuration settings as stored in the database. In the installation, you adapted the settings to your environment, in particular server names, ports, paths, and user IDs. You can update and edit the current settings as required.

The configuration settings are also provided in the configsettings.properties file located in the following subdirectory of the directory to which you extracted the oscm-install-pack.zip file when you installed the system:

<install_pack_dir>/databases/bes_db
