Defining a subscription price model

On the Define for subscription page, you can define or update the price model for a specific subscription of a selected customer. For a subscription to a service using the platform's native billing system, you can edit the price model directly on this page. For a subscription to a service using an external billing system, you can import a new or changed price model from the external system.

You can define subscription price models for your direct customers and for your brokers' customers.

The subscription price model you define or update takes priority over the service price model or a customer price model that is defined. However, you cannot change the cost calculation option. Click here for basic information on price models.

If you update a subscription price model with a service using the native billing system, the modified settings are valid for the current and future billing periods. This means that, for example, all user assignments or billable events that occurred in using the subscription in the current billing period before the change will be charged according to the new price model.

Usually, you will change a subscription price model only in agreement with the relevant customer. For changing a subscription price model, it does not matter whether the relevant service is activated or deactivated.

To define a subscription price model:

  1. In the list in the upper page area, select the subscription for which you want to define the price model.

    You can select any active subscription to your services. The subscriptions of your direct customers and your brokers' customers are included.

  2. Depending on the billing system that is defined for the underlying technical service, proceed as follows:
    • If the platform's native billing system is used, define the price model in the subsequent sections of the page. Click here for details.
    • If an external billing system is used, import the appropriate price model from this system. Click here for details.
The price model changes for the subscription take effect immediately.
