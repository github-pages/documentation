Viewing billing system details

On the View billing systems page, you can view the details of the billing systems that the platform operator has set up for the platform. These billing systems are available for selection when defining a technical service.

When accessing the View billing systems page, the details of the billing system that is currently configured as the default are expanded. Your platform operator is responsible for specifying a different system as the default, if required.

The details displayed are specific to each billing system. Click Test Connection to make sure that the connection to a billing system can be established. If it cannot be established, contact your platform operator who can check and correct the connection properties. The billing system may have been shut down or other communication problems occurred.
