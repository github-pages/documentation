Viewing a service

On the View service page, you can view the details of the services which you are authorized to offer on a marketplace.

The following information on a service is available:

  • The service name and short description which are shown in the service catalog on a marketplace when you publish the service.
  • The service description which provides details of the features offered by the service. It is displayed on a marketplace when customers request details of the service.
  • The license agreement which must be read and accepted by customers when they subscribe to the service.
  • The details of the price model defined by the supplier of the service:
    • If an external billing system is used for calculating the charges for the service, you can view the charging conditions and prices as defined in this system in a separate window by clicking Display.
    • If the platform's native billing system is used, the service can be free of charge or with costs. If customers are charged for using the service, the charging conditions and prices are shown. Click here for more details. All prices are without VAT.

To view the details of a service:

From the Service ID list, select the service for which to view the details.

The list contains all services you are authorized to offer.

The details of the service are instantly displayed.

To make the service available to customers for subscription, you must first specify the marketplace to which you want to publish it. You can then activate the service.
