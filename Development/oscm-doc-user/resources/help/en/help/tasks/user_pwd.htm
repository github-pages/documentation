Changing your password

On the Change password page, you can change your password stored on the platform.

Note: If your organization uses an external system for user authentication, contact your administrator for information on how to change your password.

To change your password:

  1. Enter your current password and the new password, and repeat the new password in the corresponding fields.

    The minimum password length is 6 characters, the maximum length is 100 characters.

    When you type the password, the Password strength field indicates how secure your password is. The color green means that your password is strong and secure. A strong and secure password consists of at least 8 characters, and contains numbers, letters, and special characters.

  2. To save the new password, click Change.

The new password is saved. Use it when logging in again.

Note: When changing the password for a technology provider responsible for a service controller, make sure to change it first in the APP (Asynchronous Provisioning Platform) Web interface before the technology provider changes it in the ESCM administration portal. Otherwise, he can no longer in to the service controller interface to change it there.
