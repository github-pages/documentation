Adding tracking code

On the Add tracking code page, you can add or remove tracking code to or from your marketplace Web pages.

Tracking code allows you to monitor the marketplace traffic by measuring the frequency in which the marketplace is accessed by visitors. You are free to choose your preferred Web analytics software. The menu option enables you to insert the desired HTML code for tracking the access into each marketplace Web page. When a page is loaded, the tracking script is executed and you can evaluate the analytics data.

To add tracking code:

  1. From the Marketplace list, select the marketplace to which you want to add tracking code.

    The list contains all marketplaces to which you have access as a marketplace manager.

  2. Enter the HTML code. You can use snippets of JavaScript code, if required.

    Caution: The code may break the layout of your marketplace pages, and the layout changes may invalidate a customized branding of your marketplace.

    Be aware that you are responsible for a correct page layout.

  3. To save your changes, click Save.

The code is added to all pages of your marketplace. The change takes effect the next time you log in to the marketplace or refresh your Web browser.

To remove tracking code:

  1. From the Marketplace list, select the marketplace from which to remove the tracking code.

    The list contains all marketplaces to which you have access as a marketplace manager.

  2. Remove the HTML code.
  3. To save your changes, click Save.

The code is removed from the pages of your marketplace. The change takes effect the next time you log in to the marketplace or refresh your Web browser.
