Adding a custom attribute

On the Add custom attribute page, you can add a custom attribute for your customers or for your subscriptions, respectively.

To add a custom attribute:

  1. Type an identifier or name for the attribute in the Attribute ID field. Attribute IDs are restricted to 40 characters and must not contain any of the following characters:

    ! " # $ % & ' * + , / : ; < > \ ^ `

  2. Enter a default value for the attribute in the Default value field.

    You can enter a maximum of 255 characters.

    The default value is used as long as no specific value for an individual customer or an individual subscription is entered. If the value is to be specified by the customer (see step 3 below), the corresponding field is preset with this default value during the subscription process.

  3. Select the User option check box if the attribute is to be offered to a customer's administrator or subscription manager when he subscribes to one of your services. This allows the administrator or subscription manager to enter a value for the attribute.
  4. Select the Mandatory check box if you want to make the specification of the attribute value mandatory for your customer's administrator or subscription manager during the subscription process. The check box is only available if you selected User option in step 3 above.
  5. To save your entries, click Save.

You are returned to the page for managing custom attributes. The new custom attribute is instantly displayed.

The name of the new attribute and its value will be included in the billing data for customers the next time the data is generated. If no value is specified for an attribute, the attribute name will be inserted into the billing data without a value.
