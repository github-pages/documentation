Managing subscription attributes

On the Manage subscription attributes page, you can define attribute values for custom attributes, provided such attributes have been defined for the subscriptions of your customers and your brokers' customers and you are authorized to enter the required values. The page also lists custom attributes whose values are to be entered by the customers' administrator or subscription manager during the subscription process. You cannot change the values for these attributes.

To enter values for subscription attributes:

  1. In the list in the upper page area, select the subscription for which you want to edit the custom attribute values. The current attribute values for the selected subscription are shown in the lower page area.
  2. Overwrite a value in the Value field. You can enter a maximum of 255 characters.
  3. To save your entries, click Save. The Save button is not available if the page lists only custom attributes whose values are to be entered by your customers' administrator or subscription manager during the subscription process.

The values will be included in the billing data for the customer the next time the data is generated. If no value is specified for an attribute, the attribute name will be inserted into the billing data without a value.
