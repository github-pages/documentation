Localizing the service information

On the Localize service description page, you can provide the service name for customers, the short description, and the detailed description of the marketable service in all supported languages. If defined for the service, you can also translate the name of any defined custom tab that is shown when customers display details on their subscriptions.

For localizing the elements, the marketable service must be deactivated. Existing subscriptions of your customers are not affected by the localization. Any changes you make are effective for new subscriptions only. To provide the service to customers for subscription, you must activate it.

Brokers and resellers as well as their customers can immediately view the service description in the languages in which you provide it. Existing subscriptions to services sold by brokers and resellers are not affected by the localization. Any changes you make are effective for new subscriptions only.

Note: If you do not localize the service elements, the English descriptions are displayed to customers.

To localize the elements:

  1. Select the source language. For reference purposes, the service name for customers, the short description, the detailed description, and the name of the custom tab are displayed in the source language.
  2. Select the target language. The list contains all supported languages.
  3. Enter the translations.
    Note: Make sure to use only the controls provided in the editor for formatting your text. Manually entered HTML tags and references to external style sheets or content may result in errors in the page layout.
  4. To confirm your translations, click OK.

After activating the service, the localized elements are visible to your customers, provided they have the corresponding language set in their profile.

Brokers and resellers as well as their customers can instantly view your localized elements.
