Managing suppliers

On the Manage suppliers page, you can add or remove a supplier to/from the list of suppliers who are allowed to define marketable services for a specific technical service.

To manage suppliers:

  1. Select the technical service for which you want to manage the suppliers. The suppliers appointed for the selected technical service are listed on the page.
  2. To appoint a supplier, type the organization ID in the Organization ID field and click Add. Contact the supplier or your platform operator for the supplier's organization ID.
  3. To remove a supplier from the list, select the corresponding check box and click Delete.

    If customer subscriptions exist for the marketable services of the supplier, a corresponding message is displayed. It lists the instance IDs of the underlying application instances. If you want to view the instance IDs of your technical services, you can create an instance report.
