Managing currencies

After installation, you need to add the currency or currencies to be supported. These currencies will be available to suppliers when defining the price models for marketable services. The selected currencies are used by the integrated rating and billing engine when calculating subscription usage charges.

To add a currency:

  1. In the ISO currency code field, enter the ISO code of the currency to be supported. For example, enter EUR for the Euro currency.
  2. Click Add.

The currency is added, if it does not yet exist.
