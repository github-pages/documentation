Managing categories

On the Manage categories page, you can create, update, and delete categories for a marketplace.

Categories can be used for organizing and classifying services on a marketplace. For example, they may group different types of application software, such as word processing, desktop publishing, or spreadsheet software.

As a marketplace owner, you can create any number of categories for your marketplace. Suppliers, brokers, and resellers can assign these categories to the services they publish to the marketplace. Marketplace visitors can use the categories for browsing the service catalog and filtering the services, provided that you have enabled the display of categories. Categories are also taken into consideration by the full-text search feature on the marketplace.

As a marketplace owner, you can decide to use the categories for grouping the services displayed on the marketplace home page. Click here for details.

A category can be provided in all supported languages. For each category, you define a language-independent ID and the actual term to be displayed in each language. If a term for a specific language is not found, the English term is displayed by default. If the English term is not defined either, the language-independent category ID is used.

To create a category:

  1. From the Marketplace list, select the marketplace for which you want to create categories. The list contains all marketplaces to which you have access as a marketplace manager.
  2. Select the Language for which you want to provide a category.

    The list contains all supported languages. You can add terms for additional languages later.

  3. Click Add to define a new category.
  4. Enter a language-independent ID for the category in the Category ID column.

    The ID is restricted to 40 characters and must be unique within the category IDs defined for the marketplace.

  5. Enter a term for the category that is to be displayed in the selected language.

    The term is restricted to 100 characters. It must be unique within the selected language.

  6. Click Add if you want to add an additional category. Otherwise, click Save to store the new category.

To update a category:

Updating a category allows you to change the category ID and the terms to be displayed for the category in all supported languages.

  1. From the Marketplace list, select the marketplace for which you want to update a category.

    The list contains all marketplaces to which you have access as a marketplace manager.

  2. To update the language-independent ID of a category, enter the new ID in the Category ID column.

    The ID is restricted to 40 characters and must be unique within the category IDs defined for the marketplace.

  3. To localize a category or update a localized term, select the required Language and enter the localized term in the corresponding column.

    The term is restricted to 100 characters. It must be unique within the selected language.

  4. Repeat Step 2 or Step 3 to update an additional category.
  5. Click Save to store your changes.

The categories are instantly updated. For marketplace visitors, the update takes effect with their next login to the marketplace.

The assignment of a category to a service by a supplier, broker, or reseller is not affected by the update. Usually, you will inform your suppliers, brokers, and resellers when you update a category. They are not notified automatically.

To delete a category:

  1. From the Marketplace list, select the marketplace for which you want to delete a category.

    The list contains all marketplaces to which you have access as a marketplace manager.

  2. Select the check box in the Delete column for the category to be deleted. You can select one or multiple categories.
    Note: Do not delete the first three categories defined for the marketplace if you have decided to display services on the marketplace home page grouped by categories.
  3. Click Save. If there are services published to the marketplace that have the category assigned, a corresponding warning is displayed.
  4. Confirm the deletion.

The category is instantly deleted. Visitors of the marketplace who use the category for filtering the services receive an empty service list. With their next login to the marketplace, the category is no longer provided to them as a filter.

If suppliers, brokers, or resellers had assigned the deleted category to their services, the assignment is removed. The suppliers, brokers, and resellers are notified by email about the deletion.
