Service usage error

An error has occurred when trying to use the service because defined restrictions have been violated:

  • The maximum number of concurrent users of the subscription has been reached. For example, 5 users are allowed to use the service concurrently, all of them are currently logged in, and you have tried to login to the service as the 6th user.
  • The subscription has been terminated. It is no longer available, thus you can no longer access the service.
