Recovering your password

On the Recover password page, you start the process for defining a new password for your user ID.

To recover your password:

  1. Enter your user ID.

    If you have forgotten your user ID, contact your administrator who can look up the IDs of all users registered for your organization.

  2. Enter the characters as visible in the Word verification area. Be aware that the characters are case-sensitive.
  3. To start the process for defining a new password, click Continue.

You are sent an email with a URL for entering a new password. Click the URL provided in the email to define a new password and finish the recovery process.

Note: If you do not receive this email, you have entered a wrong user ID. Note that you do not receive an error message when entering a wrong user ID. Check your user ID and try again with the correct one.
