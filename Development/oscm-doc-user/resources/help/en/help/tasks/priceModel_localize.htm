Localizing price model elements

On the Localize price model description page, you can provide and save price model elements in any of the supported languages:

  • If you define the price model for a marketable service that is charged for using the platform's native billing system, you can localize the description and the license agreement.
  • If you define the price model for a marketable service that is charged for using an external billing system, you can localize the license agreement only. Additional price model elements must be localized in the external system.

If you do not localize the price model elements, the English texts are displayed to customers.

You can only localize the elements if the marketable service for which you provide the translations is deactivated. Existing subscriptions are not affected by the localization. Any changes you make are visible to customers for new subscriptions only.

Brokers you have granted the permission to sell the service as well as their customers can instantly view the price model elements in the languages in which you provide them. Existing subscriptions to services sold by brokers are not affected by the localization. Any changes you make are effective for new subscriptions only.

Resellers can provide their own license agreement. Thus, localizing the license agreement does not have any effect on their service offering.

To localize price model elements:

  1. Select the source language. The price model elements you can localize are displayed in this language.
  2. Select the target language. The list contains all supported languages.
  3. Enter the translation for the price model elements.
    Note: Make sure to use only the controls provided in the editor for formatting your text. Manually entered HTML tags and references to external style sheets or content may result in errors in the page layout.
  4. To confirm your translations, click OK.
  5. To save your changes, click Save on the Define for service, Define for customer, or Define for subscription page from where you started the localization.

After activating the service, the localized elements are visible to customers who have the corresponding language set in their profile.

Brokers and resellers as well as their customers can instantly view the changes you have made.
