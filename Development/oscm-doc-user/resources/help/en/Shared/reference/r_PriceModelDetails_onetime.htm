One-time fee

A one-time fee defines an amount a customer has to pay for a subscription in the first billing period.

A one-time fee is added to the total charges for the first billing period. It is independent of the number of users, events, or other usage data.

If a one-time fee is defined for a service to which a customer upgrades or downgrades a subscription, it is added to the total charges for the customer, even if the service from which the customer migrates also defines a one-time fee.
