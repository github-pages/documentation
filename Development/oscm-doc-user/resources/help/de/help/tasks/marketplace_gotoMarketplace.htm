Auf Marktplatz wechseln

Die Seite Auf Marktplatz wechseln erlaubt Ihnen den direkten Zugriff auf einen Marktplatz.

So wechseln Sie auf einen Marktplatz:

  1. Wählen Sie in der Liste Marktplatz den Marktplatz, auf den Sie wechseln möchten.

    Die Liste stellt alle Marktplätze zur Auswahl, auf denen Sie Services veröffentlichen können oder deren Eigentümer sie sind.

  2. Klicken Sie Wechseln, um auf den Marktplatz zuzugreifen.

Der Marktplatz wird sofort angezeigt, und Sie sind automatisch angemeldet. Abhängig von Ihren Browser-Einstellungen wird der Marktplatz auf dem aktuellen Tab, einem neuen Tab oder in einem neuen Browser-Fenster angezeigt.
