Währungen verwalten

Nach der Installation müssen Sie die Währungen definieren, die auf Ihrer Plattform unterstützt werden sollen. Serviceanbietern stehen diese Währungen beim Definieren ihrer Preismodelle zur Verfügung. Die Währungen werden beim Berechnen der Nutzungsgebühren verwendet.

So fügen Sie eine Währung hinzu:

  1. Geben Sie im Feld ISO-Währungscode den ISO-Code der Währung ein, die unterstützt werden soll. Beispiel: EUR für Euro
  2. Speichern Sie die Währung mit Hinzufügen.
Falls die Währung noch nicht im System existiert, wird sie hinzugefügt.
