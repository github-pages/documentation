Rechnungsdaten exportieren

Die Seite Rechnungsdaten exportieren verwenden Sie für die folgenden Aufgaben:

  • Als Serviceanbieter oder Wiederverkäufer können Sie die Rechnungsdaten eines oder mehrerer Kunden für einen bestimmten Zeitraum exportieren.

    Die exportierten Daten können Sie für Rechnungen an Ihre Kunden verwenden.

  • Als Serviceanbieter, Vermittler, Wiederverkäufer oder Marktplatzeigentümer können Sie die anteiligen Umsätze Ihrer Organisation für einen bestimmten Zeitraum exportieren.

    Als Plattform-Betreiber können Sie die anteiligen Umsätze aller Organisationen mit einer bestimmten Rolle für einen bestimmten Zeitraum exportieren.

    Die exportierten Daten können Sie nutzen, um die Umsatzbeteiligung - je nach vertraglicher Vereinbarung - von den anderen Parteien einzufordern oder an die anderen Parteien zu zahlen.

Die Rechnungsdaten können in einer XML-Datei gespeichert oder in einem Editor Ihrer Wahl angezeigt werden. Sie können die Rechnungsdaten nach Bedarf bearbeiten und zum Beispiel an ein Buchhaltungssystem weiterleiten.

Hinweis: Eine detaillierte Beschreibung der XML-Elemente finden Sie im Handbuch Developer's Guide.

Abrechnungsdaten werden jedes Monat nur für Services die den nativen Abrechnungssystem von OSCM anwenden generiert.

So exportieren Sie Rechnungsdaten von Kunden:

  1. Wählen Sie Kundenrechnungsdaten im Feld Rechnungsdatentyp.

    Es werden alle Kunden aufgelistet, die Sie selbst angelegt haben oder die mindestens einen Ihrer Services abonniert haben.

    Wenn Sie als Serviceanbieter angemeldet sind, werden Ihnen auch die Kunden Ihrer Vermittler aufgelistet, die einen Ihrer Services abonniert haben.

  2. Markieren Sie die Kunden, deren Rechnungsdaten Sie exportieren möchten.
  3. Geben Sie das Anfangs- und Enddatum des Zeitraums an, für den Sie die Rechnungsdaten exportieren möchten. Klicken Sie dazu die Felder Von und Bis und wählen Sie das gewünschte Datum im bereitgestellten Kalender.
  4. Entscheiden Sie, ob die Daten in einer Datei gespeichert (<datum>BillingData.xml) oder direkt in einem Editor Ihrer Wahl angezeigt werden sollen.

So exportieren Sie die Umsatzbeteiligungen von Organisationen:

  1. Wählen Sie Umsatzbeteiligungen im Feld Rechnungsdatentyp.
  2. Wählen Sie im Feld Organisationsrolle die Rolle aus, für die Sie die anteiligen Umsätze exportieren möchten. Abhängig von der Rolle Ihrer Organisation können Sie Serviceanbieter, Vermittler, Wiederverkäufer oder Marktplatzeigentümer wählen.
  3. Geben Sie das Anfangs- und Enddatum des Zeitraums an, für den Sie die Daten exportieren möchten. Klicken Sie dazu die Felder Von und Bis und wählen Sie das gewünschte Datum im bereitgestellten Kalender.
  4. Entscheiden Sie, ob die Daten in einer Datei gespeichert (<datum>BillingData.xml) oder direkt in einem Editor Ihrer Wahl angezeigt werden sollen.
