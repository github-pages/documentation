Kundeninformationen anzeigen

Auf der Seite Kunden anzeigen können Sie die aktuellen Organisationsdaten anfordern, die für Ihre Kunden definiert sind.

Die folgenden Informationen können angezeigt werden

  • Name der Organisation.
  • E-Mail-Adresse der Organisation für den allgemeinen E-Mail-Verkehr.
  • Telefonnummer der Organisation.
  • URL der Webseite der Organisation.
  • Adresse der Organisation.
  • Land, in dem die Organisation ansässig ist.
  • Beschreibung der Organisation.

So zeigen Sie Informationen zu einem Kunden an:

Wählen Sie in der Liste Organisations-ID den Kunden, zu dem Sie Informationen anzeigen lassen möchten.

Es werden alle Kunden aufgelistet, die bereits einen Ihrer Services abonniert haben oder die Sie selbst registriert haben, unabhängig davon, ob bereits ein Service abonniert wurde.

Wenn Sie als Serviceanbieter angemeldet sind, sehen Sie auch die Kunden Ihrer Vermittler, die bereits einen Ihrer Services abonniert haben.

An den Namen der Kundenorganisation ist die ID der Organisation angehängt.

Die Informationen werden sofort angezeigt.
