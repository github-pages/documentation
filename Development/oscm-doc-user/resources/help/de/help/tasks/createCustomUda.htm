Eigenes Attribut anlegen

Auf der Seite Eigenes Attribut anlegen können Sie ein eigenes Attribut für Ihre Kunden oder die Abonnements Ihrer Kunden anlegen.

So legen Sie ein eigenes Attribut an:

  1. Geben Sie im Feld Attribut-ID eine ID oder einen Namen für das Attribut an. Die ID darf aus maximal 40 Zeichen bestehen. Die folgenden Zeichen sind nicht erlaubt:

    ! " # $ % & ' * + , / : ; < > \ ^ `

  2. Geben Sie im Feld Vordefinierter Wert einen Standardwert für das Attribut ein.

    Sie können maximal 255 Zeichen eingeben.

    Der Standardwert wird verwendet, sofern kein spezifischer Wert für einen bestimmten Kunden oder ein bestimmtes Abonnement angegeben wird. Wenn der Wert eines eigenen Attributs vom Kunden eingegeben werden soll (siehe nächster Schritt), ist das entsprechende Feld beim Abonnieren des Service mit dem Standardwert vorbelegt.

  3. Markieren Sie die Option Benutzeroption, wenn Sie einem Administrator oder Abo-Manager des Kunden beim Abonnieren Ihres Service die Möglichkeit bieten möchten, den Attributwert selbst einzugeben.
  4. Markieren Sie die Option Pflichtangabe, wenn die Eingabe des Attributwerts durch den Administrator oder Abo-Manager des Kunden obligatorisch sein soll. Die Option steht nur zur Verfügung, wenn Sie die Option Benutzeroption im vorherigen Schritt markiert haben.
  5. Schließen Sie Ihre Eingaben mit Speichern ab.

Die vorherige Seite zur Verwaltung von eigenen Attributen wird wieder angezeigt. Sie sehen das neue Attribut.

Der Name des neuen Attributs und die zugehörigen Attributwerte werden in die Rechnungsdaten übernommen, sobald diese neu generiert werden. Wenn kein Wert für ein Attribut angegeben ist, wird der Attributname ohne zugehörigen Wert in die Rechnungsdaten übernommen.
