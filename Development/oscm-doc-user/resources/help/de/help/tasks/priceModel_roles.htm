Rollenspezifische Preise definieren

Auf der Seite Rollenspezifische Preise definieren können Sie für jede Servicerolle, die im zugrunde liegenden technischen Service definiert ist, eine zusätzliche Gebühr festlegen. Servicerollen können dazu verwendet werden, Benutzern bestimmte Privilegien zu gewähren. In der zugrunde liegenden Anwendung werden diese Servicerollen dann auf die entsprechenden Rechte abgebildet.

Für jede Servicerolle können Sie einen Preis definieren. Der Preis wird bei der Berechnung der Nutzungsgebühren für einen Abrechnungszeitraum zum Grundpreis für Benutzer addiert.

Hinweis: Rollenspezifische Preise sind nur für Preismodelle von Services relevant, für deren Kostenberechnung das plattformeigene Abrechnungssystem verwendet wird. Preismodelle von externen Abrechnungssystemen definieren ihre eigenen Elemente, auf deren Basis die Kosten für einen Services berechnet werden.

So definieren Sie Preise für Servicerollen:

  1. Wählen Sie die Servicerolle, für die Sie Preise festlegen möchten.
  2. Geben Sie im Feld Preis für Benutzer einen Wert ein, falls die Nutzungsbebühren pro Benutzer mit der Servicerolle monatlich, wöchentlich, täglich oder stundenweise in Rechnung gestellt werden sollen.
  3. Geben Sie einen Preis für jede Serviceoption an.
  4. Bestätigen Sie Ihre Eingaben mit Anwenden und geben Sie für eine weitere Servicerolle Preise ein. Speichern Sie Ihre Eingaben mit OK. Sie werden zur Seite Service-Preismodell definieren, Kunden-Preismodell definieren oder Abo-Preismodell definieren weitergeleitet.
  5. Schließen Sie auf der Seite Service-Preismodell definieren, Kunden-Preismodell definieren oder Abo-Preismodell definieren die Definition Ihres Preismodells mit Speichern ab.
