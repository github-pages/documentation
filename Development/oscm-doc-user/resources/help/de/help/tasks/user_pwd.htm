Passwort ändern

Auf der Seite Passwort ändern können Sie Ihr aktuelles Passwort ändern.

Hinweis: Falls Ihre Organisation ein externes Authentifizierungssystem zur Benutzerverwaltung verwendet, wenden Sie sich zum Ändern Ihres Passworts an Ihren Administrator.

So ändern Sie Ihr Passwort:

  1. Geben Sie Ihr aktuelles Passwort und das neue Passwort in die entsprechenden Felder ein, und wiederholen Sie danach das neue Passwort.

    Das Passwort muss aus mindestens 6 Zeichen bestehen. Es darf nicht länger als 100 Zeichen sein.

    Beim Eingeben des Passworts zeigt Ihnen das Feld Passwortsicherheit, wie sicher Ihr Passwort ist. Die Farbe grün bedeutet, dass Sie ein sicheres Passwort eingegeben haben. Ein sicheres Passwort besteht aus mindestens 8 Zeichen und verwendet eine Kombination aus Zahlen, Buchstaben und Sonderzeichen.

  2. Speichern Sie das neue Passwort mit Ändern.

Das neue Passwort wird gespeichert. Verwenden Sie es, wenn Sie sich das nächste Mal anmelden.

Hinweis: Wenn Sie dass Passwort eines Technologie-Managers ändern, der für einen Service Controller zuständig ist, müssen Sie sicherstellen, dass das Passwort zuerst in APP (Asynchronous Provisioning Platform) geändert wird.
