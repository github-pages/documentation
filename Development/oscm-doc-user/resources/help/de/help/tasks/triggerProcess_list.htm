Prozesse verwalten

Bestimmte Aktionen von Organisationen können von einem externen Prozesssteuerungssystem gesteuert und überwacht werden.

Ist eine Genehmigung für eine Aktion erforderlich, wartet die Aktion mit ihrer Ausführung so lange, bis sie im Prozesssteuerungssystem genehmigt oder abgelehnt ist. Ist keine Genehmigung erforderlich, wird die Aktion sofort ausgeführt.

Voraussetzung für die Steuerung von Aktionen durch externe Prozesse ist ein Notification Service. Dieser Service bildet die Schnittstelle zwischen der Plattform und dem Prozesssteuerungssystem. Informationen zu diesem Service erhalten Sie von Ihrem Plattform-Betreiber.

Wenn eine von einem Prozesssteuerungssystem überwachte Aktion ausgeführt wird, wird auf der Seite Prozesse verwalten der aktuelle Status der Aktion angezeigt. Folgende Werte sind möglich:

  • Gestartet: Die Aktion wurde auf der Plattform initialisiert.
  • Wartet auf Antwort: Die Aktion muss im Prozesssteuerungssystem genehmigt werden. Erst danach wird sie ausgeführt.
  • Genehmigt: Die Aktion wurde im Prozesssteuerungssystem genehmigt, und sie wurde ausgeführt. Wenn Sie beispielsweise einen Service abonnieren, wurde das Abonnement angelegt.
  • Abgelehnt: Die Aktion wurde im Prozesssteuerungssystem nicht genehmigt und wird daher nicht ausgeführt.
  • Benachrichtigt: Die Aktion wurde erfolgreich ausgeführt, und das Prozesssteuerungssystem wurde entsprechend benachrichtigt.
  • Die Aktion wurde von dem externen System nicht angenommen: Die Verbindung zum Prozesssteuerungssystem konnte nicht hergestellt werden. Die Aktion wird daher nicht ausgeführt.
  • Die Aktion konnte nicht ausgeführt werden: Bei der Ausführung der Aktion ist ein Fehler aufgetreten, nachdem sie im Prozesssteuerungssystem genehmigt wurde. Die Aktion wird nicht ausgeführt.
  • Abgebrochen: Die Aktion wurde abgebrochen.

So zeigen Sie Aktionen an:

  • Wenn Sie nicht als Administrator angemeldet sind, listet die Seite Prozesse verwalten Ihre eigenen prozessgesteuerten Aktionen auf. Die Spalte Beschreibung zeigt die Aktion. Die Spalte Status zeigt den aktuellen Status der Aktion.
  • Wenn Sie als Administrator angemeldet sind, können Sie wählen, ob alle prozessgesteuerten Aktionen der Benutzer Ihrer Organisation oder nur Ihre eigenen aufgelistet werden sollen.

So brechen Sie Ausführung einer Aktion ab:

Markieren Sie die Aktion und klicken Sie Abbrechen. Die Ausführung einer Aktion kann nur abgebrochen werden, wenn die Aktion im Status Wartet auf Antwort ist.

Der Status wechselt zu Abgebrochen. Die Aktion muss im Prozesssteuerungssystem nicht mehr genehmigt werden. Sie wird nicht ausgeführt und kann aus der Liste gelöscht werden. Das Prozesssteuerungssystem wird über den Abbruch informiert und kann die Bearbeitung der Genehmigungsanforderung einstellen.

So löschen Sie eine Aktion aus der Liste:

Markieren Sie die Aktion und klicken Sie Löschen. Sie können eine Aktion nur löschen, wenn sie sich nicht im Status Wartet auf Antwort befindet.
