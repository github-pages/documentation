Texte anpassen

Auf der Seite Texte anpassen können Sie:

  • Die aktuellen Texte exportieren, die auf der Benutzeroberfläche sichtbar sind und in automatisch generierten E-Mails verwendet werden, sowie geänderte oder übersetzte Texte importieren.
  • Einen Hyperlink zu einer Webseite mit dem Impressum Ihres Marktplatzes angeben. Der Hyperlink wird am unteren Rand jeder Marktplatzseite eingeblendet.
  • Den Text Ihrer Datenschutzerklärung sowie Ihrer allgemeinen Geschäftsbedingungen eingeben. Die Datenschutzerklärung und die allgemeinen Geschäftsbedingungen müssen von Ihren Kunden akzeptiert werden, wenn sie sich auf Ihrem Marktplatz registrieren. Hyperlinks für die Datenschutzerklärung und die allgemeinen Geschäftsbedingungen werden am unteren Rand jeder Marktplatzseite eingeblendet.

Sie können für die Texte Übersetzungen in allen unterstützten Sprachen hinterlegen. Wenden Sie sich an Ihren Plattform-Betreiber, wenn eine zusätzliche Sprache unterstützt werden soll.

Das Branding eines Marktplatzes umfasst auch das Werbebanner und das Layout (Farben, Schriften, Hintergrundbilder, Logos, usw.). Zum Anpassen des Werbebanners, also des Bereiches zur Platzierung von Werbung, Firmenlogos und weiteren Informationen, verwenden Sie die Option Werbebanner anpassen im Menü Marktplatz. Das Layout können Sie über die Option Layout anpassen im Menü Marktplatz ändern.

So passen Sie die Texte an:
  1. Wählen Sie in der Liste Marktplatz den Marktplatz, für den Sie Anpassungen vornehmen möchten.

    Die Liste stellt alle Marktplätze zur Auswahl, auf die Sie als Marktplatz-Manager Zugriff haben.

  2. Speichern Sie die Texte, die aktuell auf der Benutzeroberfläche sichtbar sind, mit Exportieren in einer Microsoft Excel-Datei mit Dateierweiterung .xls.
  3. Öffnen Sie die Datei und bearbeiten oder übersetzen Sie die Texte. Die Datei enhält die Texte in allen aktuell unterstützen Sprachen. Sie finden einen Text, indem Sie in der Datei danach suchen, zum Beispiel auf Deutsch.

    Jeder Text ist durch einen Schlüssel eindeutig gekennzeichnet. Auf den Schlüssel folgt der Text in jeder unterstützten Sprache.

    Schlüssel für E-Mail-Texte sind wie folgt gekennzeichnet:
    • Schlüssel, die auf .subject oder .text enden und denen eine Beschreibung in Großbuchstaben vorausgeht, enthalten den Betreff bzw. den Inhalt der E-Mail. Die Beschreibung in Großbuchstaben gibt den Grund an, warum die E-Mail verschickt wird.
    • Schlüssel, die mit text. beginnen, enthalten die Kopf- und Fußzeile, die Anrede zu Beginn und die Grußformel am Ende der E-Mails.

    Beachten Sie folgendes:

    • Alle beschreibenden Texte (gekennzeichnet durch einen auf .description endenden Schlüssel) können Sie mit HTML-Tags formatieren. Zum Beispiel können Sie mit <b> Text fett darstellen oder <br> für Zeilenumbrüche verwenden.
    • Sie können in den Texten alle Zeichen des Unicode-Zeichensatzes verwenden.
    • Wenn Sie einen Text nicht ändern möchten, können Sie die entsprechenden Tabellenzellen löschen. Dies kann zum Nachvollziehen von Änderungen an der Benutzeroberfläche hilfreich sein.
    • Stellen Sie sicher, dass die Felder keine ungültigen Daten enthalten und somit Fehler im Seitenlayout erzeugen können. Ein zu langer Text wird zum Beispiel an der Benutzeroberfläche auf mehrere Zeilen aufgeteilt. Verwenden Sie zum Testen der Übersetzungen eine separate Umgebung.
  4. Importieren Sie die angepasste oder übersetzte Datei, um Ihre Änderungen sichtbar zu machen. Wählen Sie dazu die Datei aus, die die bearbeiteten Texte enthält, und klicken Sie Importieren. Die Datei muss im Microsoft Excel-Format mit Dateierweiterung .xls sein. Die Schlüssel und Übersetzungen müssen auf dem ersten Arbeitsblatt stehen.

Ihre Anpassungen werden gespeichert und sind für Marktplatzbesucher sofort sichtbar. Um Ihre Änderungen rückgängig zu machen und die Benutzeroberfläche wieder im Lieferzustand anzuzeigen, klicken Sie Zurücksetzen.

Hinweis: Die geänderten Texte der Benutzeroberfläche sind nur im Kontext des Marktplatzes sichtbar, für den Sie die Änderungen durchgeführt haben. Wenn Sie sich mit der URL zur Verwaltung eines bestimmten Marktplatzes anmelden (https://<hostname.fqdn>:8081/oscm-portal?marketplaceId=<mId>), können Sie keine Änderungen für einen anderen Marktplatz sehen. Wenn Sie sich ohne einen Marktplatzkontext anmelden (https://<hostname.fqdn>:8081/oscm-portal), können Sie überhaupt keine angepassten Texte sehen.

So geben Sie die Allgemeinen Geschäftsbedingungen und die Datenschutzerklärung ein:

  1. Wählen Sie die gewünschte Sprache und geben Sie den jeweiligen Text im bereitgestellten Editor ein.
  2. Um den Text in einer weiteren Sprache zu hinterlegen, speichern Sie den Text in der ersten Sprache, wählen die nächste Sprache und geben die Übersetzung ein.
    Hinweis: Verwenden Sie ausschließlich die Funktionen des Editors, um Textformatierungen vorzunehmen. Direkte HTML-Codierungen im Text sowie Verweise auf externe HTML-Stylesheets oder -Inhalte können zu Fehlern im Seitenlayout führen.
  3. Schließen Sie Ihre Eingaben mit Speichern ab.
Ihre Texte werden gespeichert und sind für Marktplatzbesucher sofort sichtbar.

So definieren Sie einen Hyperlink zu einer Webseite mit dem Impressum Ihres Marktplatzes:

  1. Wählen Sie die gewünschte Sprache und geben Sie den Hyperlink auf die Webseite mit dem Impressum im Feld Impressum ein.
  2. Um den Hyperlink in einer weiteren Sprache zu hinterlegen, speichern Sie den Hyperlink in der ersten Sprache, wählen die nächste Sprache und geben den entsprechenden Hyperlink ein.
  3. Schließen Sie Ihre Eingaben mit Speichern ab.
Ihre Änderungen werden gespeichert, und der Hyperlink ist für Marktplatzbesucher sofort sichtbar.
