Service-Preismodell definieren

Auf der Seite Service-Preismodell definieren können Sie ein Preismodell für einen ausgewählten Service definieren oder ändern.

Sie haben einen Service gewählt, für dessen Kostenberechnung ein externes Abrechnungssystem verwendet wird. Die Preismodelle für die Kostenberechnung werden ebenfalls in diesem System definiert und verwaltet. Für den ausgewählten Service müssen Sie das passende Preismodell von dem externen Abrechnungssystem importieren. Sie können das importierte Preismodell auf der Plattform anzeigen, aber nicht ändern. Falls Änderungen erforderlich sind, müssen diese im externen Abrechnungssystem vorgenommen werden, und Sie müssen danach das Preismodell erneut von dort importieren.

So definieren oder ändern Sie ein Service-Preismodell:

  1. In dem Bereich Definieren Sie das Preismodell klicken Sie Importieren, um das Preismodell von dem externen Abrechnungssystem zu importieren. Der Billing-Adapter, der die Plattform mit dem externen Abrechnungssystem verbindet, ist für die Bereitstellung des richtigen Preismodells verantwortlich. Über Anzeigen können Sie sich das Preismodell ansehen.
  2. Ändern Sie nach Bedarf die Lizenzvereinbarung. Die Lizenzvereinbarung muss von Kunden gelesen und akzeptiert werden, wenn sie den Service abonnieren.

    Standardmäßig wird die Lizenzvereinbarung des Herstellers angezeigt. Sie können sie übernehmen oder eine andere eingeben.

    Sie können die Lizenzvereinbarung später in weiteren Sprachen hinterlegen, indem Sie das Preismodell lokalisieren.

    Wiederverkäufer, die den Service anbieten, können Ihre Lizenzvereinbarung übernehmen oder eine eigene hinterlegen. Wenn Sie als Serviceanbieter die Lizenzvereinbarung ändern, wird die Lizenzvereinbarung der Wiederverkäufer nicht aktualisiert.

  3. Schließen Sie Ihre Eingaben mit Speichern ab.
  4. Um die Lizenzvereinbarung des Preismodells in einer anderen Sprache zu hinterlegen, klicken Sie Lokalisieren.

Sobald Sie den Service aktivieren, steht Kunden das Preismodell zur Verfügung. Es gilt für alle neuen Abonnements.

Vermittler und Wiederverkäufer und deren Kunden sehen Ihre Änderungen sofort.
