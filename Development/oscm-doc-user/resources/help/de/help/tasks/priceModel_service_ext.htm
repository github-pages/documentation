Service-Preismodell definieren

Auf der Seite Service-Preismodell definieren können Sie ein Preismodell für einen ausgewählten Service definieren oder ändern.

Sie haben ein Service gewählt das für Nutztung eines externen Abrechnungssystem in Rechnung gestellt ist. Das bedeutet, dass Sie den erforderlichen Preismodell von den Preismodellen die durch den externen Abrechnungssystem angeboten werden auswählen müssen.

Als Serviceanbieter sind Sie dafür verantwortlich den Preismodell aktuell zu halten. Soll es geänderte Preise in dem externen Abrechnungssystem geben, müssen Sie Ihren Servicepreismodell aktualisieren um die aktuellen Preise für die Kunden auf den Markplatz anzuzeigen.

So definieren Sie ein Kunden-Preismodell:

  1. In der Bereich Service-Preismodell definieren wählen Sie den Preismodell den Sie von dem externen Abrechnungssystem hochladen möchten. Die Details des Preismodells werden sofort angezeigt.
  2. Ändern Sie nach Bedarf die Lizenzvereinbarung. Die Lizenzvereinbarung muss von Kunden gelesen und akzeptiert werden, wenn sie den Service abonnieren.

    Standardmäßig wird die Lizenzvereinbarung des Herstellers angezeigt. Sie können sie übernehmen oder eine andere eingeben.

    Sie können die Lizenzvereinbarung später in weiteren Sprachen hinterlegen, indem Sie das Preismodell lokalisieren.

    Wiederverkäufer, die den Service anbieten, können Ihre Lizenzvereinbarung übernehmen oder eine eigene hinterlegen. Wenn Sie als Serviceanbieter die Lizenzvereinbarung ändern, wird die Lizenzvereinbarung der Wiederverkäufer nicht aktualisiert.

  3. Schließen Sie Ihre Eingaben mit Speichern ab.
  4. Um die Beschreibung und die Lizenzvereinbarung des Preismodells in einer anderen Sprache zu hinterlegen, klicken Sie Lokalisieren.

Sobald Sie den Service aktivieren, steht dem Kunden das Preismodell zur Verfügung. Es gilt für alle neuen Abonnements.

Vermittler und Wiederverkäufer sowie ihre Kunden können sofort sich alle Änderungen ansehen, die Sie vorgenommen haben.
