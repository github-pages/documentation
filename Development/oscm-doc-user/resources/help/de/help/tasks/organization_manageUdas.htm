Eigene Attribute verwalten

Auf der Seite Eigene Attribute verwalten können Sie eigene Attribute für Ihre Kunden und deren Abonnements anlegen, ändern oder löschen.

Eigene Attribute sind sinnvoll, wenn Sie zusätzliche Kundeninformationen speichern möchten, beispielsweise Daten zur Kostenkontrolle oder Kostenrechnung. So können Sie zum Beispiel eine Rechnungsstelle angeben, auf die bestimmte Umsätze eines Kunden verbucht werden sollen. Da es keinen Standard für solche Daten gibt, definieren Sie ihr eigenes Attribut für die Rechnungsstelle. Die eigenen Attribute werden in die Rechnungsdaten übernommen, die monatlich für Services generiert werden, die das plattformeigene Abrechnungssystem verwenden. Die Informationen können dann zum Beispiel als Referenz für ein externes Buchhaltungs-, Rechnungs- oder CRM-System verwendet werden.

Wenn Sie ein eigenes Attribut für Kunden oder Abonnements hinzufügen, können Sie entscheiden, ob Sie selbst den Attributwert für einen bestimmten Kunden oder ein bestimmtes Abonnement angeben möchten oder ob der Wert vom Kunden beim Abonnieren Ihres Service angegeben werden soll.

Für jedes Attribut können Sie einen Standardwert festlegen. Dieser Wert wird verwendet, sofern kein spezifischer Wert für einen bestimmten Kunden oder ein bestimmtes Abonnement angegeben wird. Wenn der Wert eines eigenen Attributs vom Kunden eingegeben werden soll, ist das entsprechende Feld beim Abonnieren des Service mit dem Standardwert vorbelegt. Wenn der Administrator des Kunden oder der Abo-Manager den Wert ändert, gilt der neue Wert automatisch für alle Abonnements seiner Organisation.

Um Attributwerte für einen bestimmten Kunden anzugeben, verwenden Sie die Menüoption Kunden verwalten im Menü Kunde. Um Attributwerte für ein bestimmtes Abonnement anzugeben, verwenden Sie die Menüoption Abo-Attribute verwalten im Menü Kunde.

Eigene Attribute sind auch für Kunden Ihrer Vermittler relevant, wenn Sie den Vermittlern die Berechtigung zum Wiederverkauf eines Service erteilen. Eigene Attribute haben keine Auswirkung auf Kunden Ihrer Wiederverkäufer.

So fügen Sie eigene Attribute hinzu:

  1. Um ein eigenes Attribut für alle Ihre Kunden hinzuzufügen, klicken Sie Hinzufügen im Abschnitt Legen Sie eigene Attribute für Kunden fest. Um ein eigenes Attribute für alle Abonnements hinzuzufügen, klicken Sie Hinzufügen im Abschnitt Legen Sie eigene Attribute für Abos fest.
  2. Geben Sie auf der Seite Eigenes Attribut anlegen die erforderlichen Daten ein. Details dazu finden Sie hier.
  3. Schließen Sie Ihre Eingaben auf der Seite Eigenes Attribut anlegen mit Speichern ab. Die vorherige Seite zur Verwaltung von eigenen Attributen wird wieder angezeigt. Sie sehen das neue Attribut.
  4. Mit Hinzufügen können Sie ein weiteres eigenes Attribut definieren. Wenn Sie kein weiteres Attribut hinzufügen möchten, können Sie die Seite verlassen.

So ändern Sie eigene Attribute:

  1. Klicken Sie das Symbol zum Bearbeiten des zu ändernden Attributs in der Spalte Bearbeiten.
  2. Ändern Sie die gewünschten Daten auf der Seite Eigenes Attribut bearbeiten. Details dazu finden Sie hier.
  3. Schließen Sie Ihre Eingaben auf der Seite Eigenes Attribut bearbeiten mit Speichern ab.

    Die vorherige Seite zur Verwaltung von eigenen Attributen wird wieder angezeigt. Sie sehen die geänderte Definition.

So löschen Sie eigene Attribute:

  1. Klicken Sie das Symbol zum Bearbeiten des zu löschenden Attributs in der Spalte Bearbeiten.
  2. Klicken Sie Löschen auf der Seite Eigenes Attribut bearbeiten.
  3. Bestätigen Sie den Löschauftrag.

    Das eigene Attribut wird sofort gelöscht und die vorherige Seite zur Verwaltung von eigenen Attributen wird wieder angezeigt.
