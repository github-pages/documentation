Sprachen verwalten

Die Plattform unterstützt mehrere Sprachen, in denen Benutzer arbeiten können. Nach der Installation sind die Standardsprachen der Plattform, Deutsch, Englisch und Japanisch, als Sprachpakete vorhanden. Sie können zusätzliche Sprachen bereitstellen und die Texte der Standardsprachen anpassen. Informationen, welche Sprachen hinzugefügt werden können, finden Sie hier.

Ein Sprachpaket setzt sich wie folgt zusammen:
  • Benutzeroberfläche: Alle Texte der Benutzeroberfläche und Texte in von der Plattform generierten E-Mails.
  • Online-Hilfe und FAQ-Datei im HTML-Format: Die Online-Hilfe wird im Administrationsportal, die FAQ-Datei als Hilfe zum Marktplatz angezeigt.
  • Texte in Berichten: Texte in von der Plattform generierten Berichten.
Um eine Sprache hinzuzufügen oder existierende Texte zu ändern, sind folgende Schritte erforderlich:
  1. Übersetzen Sie alle Dateien eines Sprachpakets:

    • Die Texte der Benutzeroberfläche werden in einer Datei im Format Microsoft Excel (.xls ) übersetzt oder aktualisiert. Diese Datei kann mit der Funktion Exportieren auf der Seite Sprachen verwalten erzeugt werden. Weitere Informationen erhalten Sie hier.

    • Die Online-Hilfe und FAQs können direkt in den HTML-Dateien übersetzt oder aktualisiert werden. Weitere Informationen erhalten Sie hier.

    • Texte in Berichten werden in Konfigurationsdateien übersetzt oder aktualisiert. Weitere Informationen erhalten Sie hier.

  2. Eine neue Sprache muss auf der Seite Sprachen verwalten registriert werden.
  3. Importieren Sie die übersetzte oder aktualisierte Microsoft Excel-Datei.
  4. Installieren Sie die übersetzten oder aktualisierten Online-Hilfe- und FAQ-Dateien.
  5. Installieren Sie die übersetzten oder aktualisierten Texte in Berichten.
  6. Eine neue Sprache muss auf der Seite Sprachen verwalten aktiviert werden. Benutzer können jede aktivierte Sprache in Ihrem Benutzerprofil einstellen.

Die Seite Sprachen verwalten verwenden Sie für folgende Aufgaben:

  • Anzeigen der Liste der unterstützten Sprachen. Die Standardsprache des Systems ist Englisch.
  • Registrieren einer Sprache bei der Plattform durch Hinzufügen eines Sprachcodes. Dies ist Voraussetzung dafür, dass zusätzlich zu den nach der Installation verfügbaren Sprachen Deutsch, Englisch und Japanisch, Übersetzungen in einer anderen Sprache bereitgestellt werden können.
  • Exportieren der Texte der Benutzeroberfläche und Importieren von Übersetzungen bzw. Anpassungen.
  • Aktualisieren von vorhandenen Übersetzungen.
  • Aktivieren oder Deaktivieren einer Sprache.

So registrieren Sie eine neue Sprache:

  1. Geben Sie im Feld ISO-Sprachcode den zweistelligen ISO-Code der Sprache ein, die unterstützt werden soll. Geben Sie beispielsweise cs für Tschechisch ein.

    Es werden nur Sprachen mit Schreibrichtung von links nach rechts unterstützt. Sprachen mit Schreibrichtung von rechts nach links oder oben nach unten sowie Varianten von Sprachen wie zum Beispiel Britisches und Amerikanisches Englisch werden nicht unterstützt. Eine Liste der zulässigen Sprachcodes erhalten Sie hier.

  2. Klicken Sie Hinzufügen. Die neue Sprache ist standardmäßig deaktiviert und steht Benutzern noch nicht zur Auswahl.

Sie können nun wie folgt fortfahren:

  1. Exportieren Sie die Texte der Benutzeroberfläche und übersetzen Sie sie.
  2. Importieren Sie die Texte.
  3. Übersetzen Sie die Online-Hilfe- und FAQ-Dateien sowie die Texte in Berichten und installieren Sie sie. Informationen zu Online-Hilfe- und FAQ-Dateien erhalten Sie hier, zu Texten in Berichten hier.

    Sie können auch nur einen Teil der Texte übersetzen. Zum Beispiel können Sie nur die Online-Hilfe, FAQs, Texte in E-Mails und Berichten auf Ungarisch zur Verfügung stellen, die Benutzeroberfläche aber auf Englisch beibehalten. Wählt ein Benutzer Ungarisch in seinem Benutzerprofil, sieht er die Benutzeroberfläche auf Englisch, alle weiteren Texte auf Ungarisch.

  4. Aktivieren Sie die neue Sprache, damit Benutzer sie auswählen können.

So exportieren, übersetzen und importieren Sie Texte der Benutzeroberfläche:

  1. Klicken Sie Exportieren und speichern Sie die Texte in eine Microsoft Excel-Datei mit Dateierweiterung .xls auf Ihrem Dateisystem.
  2. Übersetzen Sie die Texte. Weitere Informationen erhalten Sie hier.
  3. Importieren Sie die übersetzte Microsoft Excel-Datei:
    1. Klicken Sie das Feld Dateiname oder Durchsuchen und wählen Sie die Datei mit den Übersetzungen.

    2. Klicken Sie Importieren.

    Stellen Sie sicher, dass die angepasste Datei im Microsoft Excel-Format (.xls ) ist und drei Arbeitsblätter mit jeweils fünf Spalten enthält (Schlüssel (Key), die drei Standardsprachen "system en", "system de", "system ja" sowie die neue Sprache).

    Ist ein Text in der neuen Sprache nicht vorhanden, wird der englische Text verwendet. Existiert der Text auch nicht auf Englisch, wird der technische Schlüssel angezeigt.

  4. Schließen Sie Ihre Änderungen mit Speichern ab.

So aktualisieren Sie existierende Texte:

  1. Klicken Sie die Zeile der Sprache, die Sie aktualisieren wollen.

    Sie können jede aufgelistete Sprache wählen, einschließlich der Standardsprachen (de, en, jp).

  2. Klicken Sie Exportieren und speichern Sie die Texte in eine Microsoft Excel-Datei mit Dateierweiterung .xls auf Ihrem Dateisystem.
  3. Die Spalte mit dem ausgewählten Sprachcode im Titel enthält die bereits vorhandenen Texte. Bearbeiten Sie die Texte. Weitere Informationen erhalten Sie hier.
  4. Importieren Sie die aktualisierte Microsoft Excel-Datei:
    1. Klicken Sie das Feld Dateiname oder Durchsuchen und wählen Sie die aktualisierte Datei.

    2. Klicken Sie Importieren.

  5. Schließen Sie Ihre Änderungen mit Speichern ab.

So aktivieren Sie eine Sprache:

  1. Markieren Sie Aktiviert neben der jeweiligen Sprache.

    Sie können mehrere Sprachen markieren.

  2. Schließen Sie Ihre Änderungen mit Speichern ab.

Benutzer können jede aktivierte Sprache in Ihrem Profil einstellen.

So deaktivieren Sie eine Sprache:

  1. Entfernen Sie die Markierung Aktiviert neben der jeweiligen Sprache. Die Standardsprache Englisch kann nicht deaktiviert werden.

    Sie erhalten eine Meldung, dass die gewählte Sprache nach Bestätigung durch Speichern nicht mehr unterstützt wird.

  2. Schließen Sie Ihre Änderungen mit Speichern ab.

Wenn ein Benutzer die deaktivierte Sprache in seinem Profil eingestellt hat:

  • Wird die Profilsprache des Benutzers auf Englisch umgestellt. Das bedeutet, dass ihm alle Texte zunächst in der Standardsprache Englisch angezeigt werden.
  • Muss der Benutzer sein Profil aktualisieren: Er kann bei der Standardsprache bleiben oder eine andere, aktivierte Sprache wählen.
  • Steht die deaktivierte Sprache nicht mehr zur Auswahl.
