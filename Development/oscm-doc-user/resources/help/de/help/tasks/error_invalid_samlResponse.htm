Probleme mit einem externen Authentifizierungssystem beheben

Wenden Sie sich an Ihren Plattform-Betreiber, der die Verbindung zu dem von Ihrer Organisation verwendeten Authentifizierungssystem überprüfen und gegebenenfalls Fehler beheben kann. Das System ist entweder falsch konfiguriert oder liefert ungültige Benutzerdaten.
