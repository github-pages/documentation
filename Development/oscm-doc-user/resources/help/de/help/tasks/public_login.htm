Anmelden

Auf der Seite Anmelden können Sie sich mit Ihren persönlichen Daten anmelden. Sie erreichen diese Seite, indem Sie die Zugangs-URL in Ihrem Browser eingeben, ein entsprechendes Lesezeichen wählen oder dem Link in Ihrer Registrierungs-E-Mail folgen.

So melden Sie sich an:

  1. Geben Sie Ihren Benutzernamen und Ihr Passwort ein.
  2. Klicken Sie Anmelden oder drücken Sie die Eingabetaste.

    Sie werden anschließend entweder direkt angemeldet oder aufgefordert, Ihr erstes Passwort zu ändern.

Wenn Sie beim Anmelden ein falsches Passwort eingeben, wird Ihr Zugang nach der dritten Falscheingabe möglicherweise gesperrt. Das hängt davon ab, ob ihre Organisation die Benutzerdaten in einem externen Authentifizierungssystem verwaltet. In diesem Fall können Passwörter nur in diesem System geändert oder zurückgesetzt werden. Wird dagegen die Plattform zur Benutzerauthentifizierung verwendet, wenden Sie sich an Ihren Administrator. Dieser kann Ihr Passwort zurücksetzen, und Sie erhalten eine E-Mail mit einem neuen Passwort.

Wenn Sie Ihr Passwort vergessen haben, klicken Sie Passwort vergessen? auf der Seite Anmelden. Sie können anschließend ein neues Passwort für Ihren Benutzernamen festlegen. Ein neues Passwort kann nicht festgelegt werden, wenn Ihr Zugang gesperrt ist oder wenn Ihre Organisation Benutzernamen und Passwörter in einem externen Authentifizierungssystem verwaltet.

Wenn Sie Ihren Benutzernamen vergessen haben, wenden Sie sich an Ihren Administrator. Dieser kann sich die Benutzernamen anzeigen lassen, die für Ihre Organisation registriert sind.

Wenn Ihre Sitzung unterbrochen wird, müssen Sie sich erneut anmelden.
