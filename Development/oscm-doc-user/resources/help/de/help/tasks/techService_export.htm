Servicedefinition exportieren

Auf der Seite Servicedefinition exportieren können Sie die Definition technischer Services, die Sie bereitgestellt haben, exportieren und in einer XML-Datei speichern. Sie können die XML-Datei nach Belieben editieren oder als Vorlage für neue Servicedefinitionen verwenden. Ausführliche Informationen zum Format und zur Verwendung der XML-Elemente in der Datei finden Sie im Handbuch Technology Provider's Guide.

So exportieren Sie die Definition technischer Services:

  1. Markieren Sie die Services, deren Definition Sie exportieren möchten. Beim Export mehrerer Definitionen werden alle Definitionen in einer XML-Datei gespeichert.
  2. Klicken Sie Exportieren.
  3. Entscheiden Sie, ob Sie die Definitionen in einer XML-Datei speichern oder in einem Editor Ihrer Wahl anzeigen lassen möchten.
