Servicedefinition löschen

Auf der Seite Servicedefinition löschen können Sie einen technischen Service löschen. Dies ist nur möglich, wenn kein Serviceanbieter auf Basis des technischen Service marktfähige Services definiert hat und wenn alle Abonnements, die für entsprechende marktfähige Services abgeschlossen wurden, beendet sind.

So löschen Sie einen technischen Service:

  1. Wählen Sie den zu löschenden Service, und klicken Sie Löschen.
  2. Bestätigen Sie den Löschauftrag.
Der Service wird gelöscht. Eine zugehörige XML-Datei mit der Servicedefinition wird nicht gelöscht, so dass Sie sie bei Bedarf wieder importieren können.
