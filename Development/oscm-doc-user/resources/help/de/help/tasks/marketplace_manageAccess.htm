Zugang zu Marktplatz verwalten

Auf der Seite Zugang verwalten legen Sie fest, wen Sie auf einen Marktplatz zulassen.

Nach dem Anlegen ist ein Marktplatz öffentlich und damit zugänglich für alle anonymen und registrierten Benutzer. Als Marktplatzeigentümer können Sie den Zugang auf Benutzer von bestimmten registrierten Organisationen beschränken. Die Benutzer müssen sich in diesem Fall anmelden, um die Services auf dem Marktplatz sehen, abonnieren und verwenden zu können. Neue Kunden können sich auf dem Marktplatz nicht selbst registrieren, sondern müssen von einem Serviceanbieter, Wiederverkäufer, Vermittler oder dem Betreiber registriert werden.

Wenn Sie einen Marktplatz von einem öffentlichen zu einem nicht-öffentlichen ändern, erhalten Organisationen mit bestehenden Abonnements automatisch Zugang. Sie können jeder Organisation den Zugang entziehen, unabhängig davon, ob er automatisch oder explizit von Ihnen erteilt wurde. Bestehende Abonnements werden in diesem Fall ausgesetzt.

Einen Marktplatz mit eingeschränktem Zugang können Sie jederzeit wieder öffentlich und damit zugänglich für alle Benutzer und Organisationen machen.

So verwalten Sie den Zugang zu einem Marktplatz:

  1. Wählen Sie in der Liste Marktplatz den Marktplatz, für den Sie den Zugang verwalten möchten.

    Die Liste enthält alle Marktplätze, auf die Sie als Marktplatz-Manager Zugriff haben.

    Die Option Nur ausgewählte Kunden zeigt, ob der Marktplatz öffentlich oder nur für Benutzer bestimmter Organisationen zugänglich ist. Im letzteren Fall sind alle Organisationen aufgelistet, und Markierungen zeigen, welche davon Zugang zum Marktplatz haben.

  2. Ändern Sie falls gewünscht die Zugangsart für den Marktplatz, indem Sie die Option Nur ausgewählte Kunden markieren oder die Markierung entfernen.
  3. Wählen Sie für einen nicht-öffentlichen Marktplatz die Organisationen aus, die Zugang haben sollen. Entfernen Sie die Markierung für jede Organisation, die keinen Zugang mehr haben sollen. Falls die Organisation bereits Abonnements besitzt, wird eine entsprechende Warnung angezeigt. Wenn Sie der Organisation den Zugang dennoch entziehen, werden die Abonnements ausgesetzt.
