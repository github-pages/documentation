Kunden-Preismodell löschen

Auf der Seite Kunden-Preismodell löschen können Sie Preismodelle löschen, die Sie Ihren Kunden nicht mehr anbieten möchten. Sie können ein oder mehrere Preismodelle löschen, die für einen bestimmten Kunden definiert wurden.

Üblicherweise löschen Sie ein Kunden-Preismodell nur in Absprache mit dem jeweiligen Kunden.

Um ein Kunden-Preismodell zu löschen, muss der entsprechende marktfähige Service deaktiviert sein. Das Löschen wirkt sich nicht auf laufende Abonnements aus. Für neue Abonnements wird automatisch das entsprechende Service-Preismodell verwendet.

So löschen Sie Kunden-Preismodelle:

  1. Wählen Sie in der Liste Kunde den Kunden, für den Sie Preismodelle löschen möchten.

    Die Liste führt alle Kundenorganisationen auf, für die Sie kundenspezifische Preismodelle definiert haben.

  2. Markieren Sie die Services, für die Sie die Kunden-Preismodelle löschen möchten.
  3. Klicken Sie Löschen und bestätigen Sie den Löschauftrag.
Die Kunden-Preismodelle werden sofort gelöscht und stehen nicht mehr zur Verfügung.
