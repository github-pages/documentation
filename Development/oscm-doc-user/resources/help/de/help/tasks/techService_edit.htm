Servicedefinition ändern

Auf der Seite Servicedefinition ändern können Sie die Texte ändern, die Ihren technischen Service beschreiben. Ihre Änderungen wirken sich auf alle neuen marktfähigen Services aus, die auf Basis des technischen Service definiert werden.

Die Texte, die Sie eingeben, werden in der in Ihrem Benutzerprofil eingestellten Sprache gespeichert. Details zum Bearbeiten des Profils finden Sie hier.

So bearbeiten Sie eine Servicedefinition:

  1. Wählen Sie den zu bearbeitenden technischen Service. Die Teile der Servicedefinition, die Sie ändern können, werden angezeigt.
  2. Geben Sie für die folgenden Eigenschaften neue Texte ein oder ändern Sie bestehende Texte:
    • Zugriffsinformation: Dieser Text wird Kunden beim Zugriff auf einen abonnierten Service angezeigt, der auf diesem technischen Service basiert. Der Text wird an der Benutzeroberfläche angezeigt, und er ist Bestandteil der E-Mail, die die einem Service zugeordneten Benutzer erhalten. Geben Sie die Beschreibung für alle technischen Services an, für die Direktzugriff bzw. Benutzerzugriff als Zugangsart definiert ist. Bei diesen Zugriffsarten greifen die Benutzer nicht über die Plattform auf die Anwendung zu. Als Zugriffsinformation können Sie zum Beispiel einführenden Text und die URL der Anmeldeseite bereitstellen, über die die Benutzer direkt auf den Service zugreifen können.

      Beachten Sie folgendes: Wenn zuvor keine Zugriffsinformation für Englisch gespeichert wurde, wird der hier eingegebene Text unabhängig von Ihrer derzeitigen Spracheinstellung auch für Englisch hinterlegt. Englisch ist die Standardsprache des Systems. Das bedeutet, dass der für Englisch gespeicherte Text allen Kunden angezeigt wird, die in einer Sprache arbeiten, für die kein eigener Text hinterlegt ist.

    • Beschreibung: Dieser Text wird Serviceanbietern angezeigt, die für diesen technischen Service einen marktfähigen Service definieren. Der Serviceanbieter kann den Text zur Beschreibung des marktfähigen Service übernehmen oder einen neuen eingeben.
    • Lizenzvereinbarung: Dieser Text wird Serviceanbietern angezeigt, wenn sie ein Preismodell für einen marktfähigen Service auf Basis dieses technischen Service definieren. Der Serviceanbieter kann den Text der Lizenzvereinbarung übernehmen oder einen neuen eingeben.
    • Suchwörter: Mit Suchwörtern können Sie Ihren Service kennzeichnen. Kunden können die Suchwörter auf einem Marktplatz nutzen, um marktfähige Services auf Basis dieses technischen Service zu finden. Voraussetzung dafür ist, dass der Eigentümer des Marktplatzes die Anzeige der Suchwörter eingeschaltet hat.

      Sie können maximal fünf Suchwörter festlegen, jeweils getrennt durch ein Komma. Suchwörter dürfen maximal 20 Zeichen lang sein. Bei der Eingabe der Suchwörter müssen Sie nicht auf Groß-/Kleinschreibung achten.

      Mit Die am häufigsten verwendeten Suchwörter anzeigen sehen Sie die Suchwörter, die am häufigsten von den Herstellern technischer Services definiert wurden.

    Hinweis: Verwenden Sie ausschließlich die Funktionen des Editors, um Textformatierungen vorzunehmen. Direkte HTML-Codierungen im Text sowie Verweise auf externe HTML-Stylesheets oder -Inhalte werden nicht ausgewertet und können zu Fehlern im Seitenlayout führen.
  3. Geben Sie für die Parameter und deren Optionen Beschreibungen ein oder ändern Sie die existierenden Beschreibungen. Die Beschreibungen werden angezeigt, wenn ein Serviceanbieter für den technischen Service einen marktfähigen Service definiert.
  4. Geben Sie für die definierten Ereignisse Beschreibungen ein oder ändern Sie die existierenden Beschreibungen. Die Beschreibungen werden angezeigt, wenn ein Serviceanbieter für den technischen Service einen marktfähigen Service definiert.
  5. Geben Sie für die definierten Servicerollen jeweils einen Namen und eine Beschreibung ein oder ändern Sie die existierenden Namen und Beschreibungen.

    Servicerollen können auf Zugriffsrechte in der zugrunde liegenden Anwendung abgebildet werden. Im Preismodell für einen marktfähigen Service auf Basis dieses technischen Service können für jede Servicerolle unterschiedliche Nutzungsgebühren festgelegt werden.

    Die Namen und Beschreibungen der Servicerollen werden angezeigt, wenn ein Serviceanbieter ein Preismodell für einen marktfähigen Service auf Basis des technischen Service definiert.

  6. Geben Sie für die definierten Operationen jeweils einen Namen und eine Beschreibung ein oder ändern Sie die existierenden Namen und Beschreibungen. Mit Hilfe von Operationen, beispielsweise Datensicherung oder das Starten und Beenden einer Anwendung, kann über das Web auf Systemressourcen der zugrunde liegenden Anwendung zugegriffen werden.

    Die Namen und Beschreibungen der Operationen werden angezeigt, wenn ein Kunde die Anwendung aufruft, die über den technischen Service integriert wurde.

  7. Schließen Sie Ihre Eingaben mit Speichern ab.
Serviceanbieter sehen die Änderungen, sobald sie neue marktfähige Services auf Basis des technischen Service definieren.
