Abrechnungssysteme anzeigen

Auf der Seite Abrechnungssysteme anzeigen können Sie detaillierte Informationen zu den Abrechnungssystemen sehen, die der Plattform-Betreiber für die Plattform definiert hat. Diese Abrechnungssysteme können bei der Definition eines technischen Service angegeben werden.

Wenn Sie die Seite Abrechnungssysteme anzeigen öffnen, werden die Details des Abrechnungssystems angezeigt, das momentan als Standardsystem eingestellt ist. Falls erforderlich, kann der Plattform-Betreiber ein anderes Abrechnungssystem als Standard festlegen.

Die angezeigten Informationen sind abhängig vom jeweiligen Abrechnungssystem. Über Verbindung testen können Sie prüfen, ob die Verbindung zu einem Abrechnungssystem hergestellt werden kann. Falls dies nicht möglich ist, bitten Sie den Plattform-Betreiber, die Verbindungseigenschaften zu überprüfen und zu korrigieren. Eventuell ist das Abrechnungssystem außer Betrieb oder Verbindungsprobleme sind aufgetreten.
