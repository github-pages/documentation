Abrechnungssystemdetails anzeigen

Auf der Seite Abrechnungssysteme anzeigen können Sie die Details des Abrechnungssystems, den der Plattform-Betreiber für die Plattform ausgewählt hat, sich ansehen. Diese Abrechnungssysteme können ausgewählt werden wenn man einen technischen Service registriert.

Wenn die Seite Abrechnungssysteme anzeigen eröffnet wird, werden die Details des Abrechnungssystems, der im Moment als Standard konfiguriert ist, erweitert. Sollte es erforderlich sin, ist Euer Plattform-Betreiber dafür verantwortlich ein anderes System als Standard festzulegen.

Die für die einzelnen Systeme gelisteten Details sind für jedes Abrechnungssystem spezifisch. Klicken Sie Verbindung testen um sicherzustellen, dass die Verbindung zum Abrechnungssystem erstellt werden kann. Wenn keine Verbindung erstellt werden kann, wenden Sie sich an Ihrem Plattform-Betreiber, der die Verbindungseigenschaften überprüfen und korrigieren kann. Das Abrechnungssystem kann abgeschaltet sein oder andere Verbindungsprobleme sind aufgetreten.
