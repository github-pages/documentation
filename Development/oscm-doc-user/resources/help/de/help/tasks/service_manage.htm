Service verwalten

Auf der Seite Service verwalten können Sie die Eigenschaften der Services anzeigen und ändern, für die Sie die Berechtigung zum Wiederverkauf haben.

Folgende Eigenschaften eines Service werden angezeigt:

  • Servicename und Kurzbeschreibung. Diese werden im Servicekatalog auf einem Markplatz angezeigt, wenn Sie den Service veröffentlichen.
  • Ausführliche Beschreibung der Funktionalität des Service. Die Beschreibung wird auf einem Marktplatz angezeigt, wenn Kunden Details zu dem Service anfordern.
  • Lizenzvereinbarung. Diese muss von den Kunden gelesen und akzeptiert werden, wenn sie den Service abonnieren.
  • Details zum Preismodell. Diese werden vom Serviceanbieter festgelegt.
    • Wenn für die Kostenberechnung für den Service ein externes Abrechnungssystem verwendet wird, können Sie die dort definierten Abrechnungsbedingungen und Preise über Anzeigen ansehen.
    • Wenn für die Kostenberechnung für den Service das plattformeigene Abrechnungssystem verwendet wird, kann der Service kostenlos oder kostenpflichtig sein. Wenn für die Nutzung des Service Kosten anfallen, können Sie hier die Abrechnungsbedingungen und Preise sehen. Details dazu finden Sie hier. Alle Preise sind netto ohne Mehrwertsteuer.

Mit Ausnahme der Lizenzvereinbarung können die Informationen nur angezeigt werden. Den Text der Lizenzvereinbarung können Sie nach Bedarf ändern. Dazu muss der Service deaktiviert sein. Laufende Abonnements sind von Ihren Änderungen nicht betroffen. Die Änderungen wirken sich nur auf neue Abonnements aus. Sie müssen den Service aktivieren, damit er den Kunden zum Abonnieren angeboten wird.

So sehen und ändern Sie die Eigenschaften eines Service:

  1. Wählen Sie im Feld Service-ID den Service, dessen Eigenschaften Sie anzeigen oder ändern möchten.

    Die Liste enthält alle Services, für die Sie die Berechtigung zum Wiederverkauf haben.

  2. Im Feld Lizenzvereinbarung ändern Sie den Text der Lizenzvereinbarung nach Bedarf. Das Feld ist mit der Lizenzvereinbarung vorbelegt, die der Serviceanbieter festgelegt hat. Sie können diese entweder übernehmen oder Ihre eigene festlegen.
    Hinweis: Verwenden Sie ausschließlich die Funktionen des Editors, um Textformatierungen vorzunehmen. Direkte HTML-Codierungen im Text sowie Verweise auf externe HTML-Stylesheets oder -Inhalte können zu Fehlern im Seitenlayout führen.
  3. Schließen Sie Ihre Änderungen mit Speichern ab. Der Text wird in der Sprache gespeichert, die in Ihrem Profil festgelegt ist.
  4. Um die Lizenzvereinbarung in einer anderen Sprachen verfügbar zu machen, ändern Sie die Sprache in Ihrem Profil. Verwenden Sie dazu die Menüoption Profil bearbeiten im Menü Benutzerkonto. Anschließend können Sie die gewünschten Übersetzungen bearbeiten.

Der Service wird mit den neuen Eigenschaften angezeigt.

Damit Kunden den Service abonnieren können, müssen Sie den Marktplatz festlegen, auf dem Sie den Service veröffentlichen möchten. Anschließend können Sie den Service aktivieren.
