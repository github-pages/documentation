Passwort zurücksetzen

Sie haben den Vorgang zum Zurücksetzen Ihres Passworts gestartet. Sie erhalten eine Email mit einem Link zu einer Webseite, auf der Sie den Vorgang abschließen können.

Folgen Sie dem Link in der Email, um ein neues Passwort festzulegen. Der Link steht Ihnen 24 Stunden lang zur Verfügung.
