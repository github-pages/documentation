Abo-Details anzeigen

Auf der Seite Abo-Details anzeigen können Sie Informationen zu Abonnements Ihrer Kunden anfordern. Wenn Sie als Serviceanbieter angemeldet sind, gehören dazu auch Abonnements von Kunden Ihrer Vermittler.

Die folgenden Informationen zu einem Abonnement können angezeigt werden:

  • Datum, an dem der Kunde den Service abonniert hat, und Anzahl der Benutzer, die dem Abonnement zugeordnet sind.
  • Details zum marktfähigen Service, den der Kunde abonniert hat.
  • Details zum technischen Service, der dem marktfähigen Service zugrunde liegt.
  • Details zum Preismodell des marktfähigen Service.

So zeigen Sie die Informationen zu einem Abonnement an:

Wählen Sie in der Liste im oberen Bereich der Seite das Abonnement, zu dem Sie Informationen anzeigen lassen möchten.

Sie können jedes laufende Abonnement Ihrer Kunden wählen. Wenn Sie als Serviceanbieter angemeldet sind, können Sie auch ein laufendes Abonnement der Kunden Ihrer Vermittler wählen.

Die Detailinformationen werden sofort im unteren Bereich der Seite angezeigt.
