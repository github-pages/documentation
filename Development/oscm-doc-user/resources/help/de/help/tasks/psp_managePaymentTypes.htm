Zahlungsarten hinzufügen

Auf der Seite Neue Zahlungsart können Sie eine vom ausgewählten Zahlungsdienstleister angebotene Zahlungsart einrichten. Die Zahlungsarten, die Sie hier einrichten, können später Serviceanbietern und Wiederverkäufern zur Verfügung gestellt werden. Die Serviceanbieter und Wiederverkäufer können wiederum Zahlungsarten auswählen, die sie ihren Kunden anbieten möchten.

Sie können eine neue Zahlungsart hinzufügen, jedoch keine bereits für den Zahlungsdienstleister definierte Zahlungsart löschen. Sie können für jede Zahlungsart einen beliebigen Namen vergeben. Die Zahlungsarten selbst werden vom Zahlungsdienstleister vorgegeben.

So fügen Sie eine Zahlungsart hinzu:

  1. Geben Sie im Feld Name einen Namen für die Zahlungsart ein. Serviceanbietern, Wiederverkäufern und Kunden wird die Zahlungsart unter diesem Namen angezeigt.

    Der Name kann maximal 100 Zeichen enthalten.

  2. Geben Sie im Feld Zahlungsart die vom Zahlungsdienstleister definierte ID für die Zahlungsart ein.
  3. Schließen Sie Ihre Eingaben mit Speichern ab.
    Hinweis:

    Eine einmal gespeicherte Zahlungsart kann nicht mehr entfernt werden.

Fahren Sie mit der Konfiguration des Zahlungsdienstleisters auf der vorherigen Seite fort.
