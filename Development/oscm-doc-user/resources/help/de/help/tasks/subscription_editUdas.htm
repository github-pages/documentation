Abo-Attribute verwalten

Auf der Seite Abo-Attribute verwalten können Sie Attributwerte für eigene Abo-Attribute eingeben, vorausgesetzt, es wurden eigene Attribute für Abonnements Ihrer Kunden und die Kunden Ihrer Vermittler definiert und Sie sind berechtigt, die jeweiligen Attributwerte festzulegen. Die Seite listet auch eigene Attribute auf, deren Werte vom Administrator Ihres Kunden oder dem Abo-Manager beim Abonnieren eines Service eingegeben werden sollen. Sie können die Werte dieser Attribute nicht ändern.

So bearbeiten Sie die Werte für Abo-Attribute:

  1. Wählen Sie in der Liste im oberen Bereich der Seite das Abonnement, für das Sie die Attributwerte bearbeiten möchten. Die aktuellen Attributwerte für das gewählte Abonnement werden im unteren Bereich der Seite angezeigt.
  2. Geben Sie im Feld Wert einen neuen Wert an. Ein Attributwert darf maximal 255 Zeichen lang sein.
  3. Schließen Sie Ihre Eingaben mit Speichern ab. Die Option Speichern steht nicht zur Verfügung, wenn die Seite ausschließlich eigene Attribute auflistet, deren Werte vom Administrator oder Abo-Manager Ihres Kunden beim Abonnieren eines Service eingegeben werden sollen.

Die Werte werden bei einer Neuberechnung der Nutzungsgebühren in die Rechnungsdaten des Kunden übernommen. Wenn kein Wert existiert, wird der Attributname ohne einen Attributwert in die Rechnungsdaten übernommen.
