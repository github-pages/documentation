Umsatzbeteiligung von Vermittlern verwalten

Auf der Seite Umsatzbeteiligung von Vermittlern verwalten können Sie für einzelne Services, die Vermittler verkaufen dürfen, eine Umsatzbeteiligung festlegen. Diese Beteiligung pro Service hat Vorrang vor der für einen Marktplatz voreingestellten Umsatzbeteiligung für Vermittler (einstellbar über die Menüoption Marktplatzeigenschaften ändern). Desweiteren können Sie für einzelne Vermittlerorganisationen eine Umsatzbeteiligung festlegen. Diese wiederum hat Vorrang vor der Beteiligung pro Service.

Marktplatz-spezifische Umsatzbeteiligungen legen Sie auf der Seite Marktplatzeigenschaften ändern fest.

Auf der Seite Umsatzbeteiligung von Vermittlern verwalten können Sie auch die Umsatzbeteiligung für Services ändern, für die keine Berechtigung mehr zum Wiederverkauf besteht oder die bereits vom Serviceanbieter gelöscht wurden. Es besteht die Möglichkeit, dass noch Abonnements für diese Services laufen und dass Sie die jeweilige Umsatzbeteiligung ändern möchten.

So legen Sie die Umsatzbeteiligung für einen Vermittler fest:

  1. Wählen Sie in der Liste Service-ID den Service, für den Sie eine Umsatzbeteiligung festlegen möchten.

    Die Liste enthält alle auf der Plattform verfügbaren Services. Dazu gehören auch alle Services, die bereits gelöscht wurden. An den Namen von gelöschten Services wird eine Raute (#) sowie ein UNIX-Zeitstempel angehängt. Dieser gibt an, wann der Service gelöscht wurde. Bei allen Services ist die Organisations-ID des Serviceanbieters, der den Service zur Verfügung stellt, in Klammern angegeben.

  2. In der ersten Tabelle wird die für den Marktplatz eingestellte Umsatzbeteiligung für Vermittler angezeigt, die auch für den gewählten Service voreingestellt ist.

    Im Feld Umsatzbeteiligung (%) können Sie die Umsatzbeteiligung für den gewählten Service ändern: Geben Sie einen Prozentwert an.

    Dieser Wert gilt für alle Vermittler, die den gewählten Service veröffentlichen. Der Wert hat Vorrang vor der Einstellung, die über die Menüoption Marktplatzeigenschaften ändern festgelegt wurde.

  3. In der zweiten Tabelle werden alle Vermittlerorganisationen angezeigt, die den gewählten Service verkaufen dürfen.

    Im Feld Umsatzbeteiligung (%) können Sie einen Prozentwert eingeben, der nur für die entsprechende Vermittlerorganisation gilt.

    Dieser Wert hat Vorrang sowohl vor der Umsatzbeteiligung für den gewählten Service als auch vor der Einstellung, die über die Menüoption Marktplatzeigenschaften ändern festgelegt wurde.

  4. Schließen Sie Ihre Eingaben mit Speichern ab.

    Die Prozentsätze werden berücksichtigt, sobald der Vermittler den Service auf einem Marktplatz veröffentlicht. Die tatsächlichen Umsatzbeteiligungen werden bei der nächsten Neuberechnung der Nutzungsdaten errechnet und mit den Rechnungsdaten ausgegeben.
