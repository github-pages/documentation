Registrieren

Auf der Seite Registrieren geben Sie die Daten ein, die für die Einrichtung Ihres Benutzerkontos erforderlich sind.

So registrieren Sie sich:

  1. Geben Sie Ihre E-Mail-Adresse ein.

    Die E-Mail-Adresse muss eingegeben werden. Sobald Sie registriert sind, erhalten Sie eine Bestätigung an diese Adresse.

    Das System validiert, ob die Syntax der E-Mail-Adresse gültig ist und ob der Domain-Name den Standards der Internet Assigned Numbers Authority (IANA) entspricht.

    Beispiele: user.name@domain.arpa, user.name@domain.org, user@mycompany.lan.uk

  2. Geben Sie Ihre Registrierungsdaten ein.

    Felder, die mit einem Stern (*) gekennzeichnet sind, müssen ausgefüllt werden.

    • Gewünschter Benutzername: Name, mit dem Sie sich anmelden möchten. Benutzernamen dürfen aus maximal 100 Zeichen bestehen. Die folgenden Zeichen sind nicht zulässig:

      ! " # $ % & ' * + , / : ; < = > ? \ ^ `

    • Passwort: Passwort, mit dem Sie sich anmelden möchten. Das Passwort muss aus mindestens 6 Zeichen bestehen. Es darf nicht länger als 100 Zeichen sein.

      Beim Eingeben des Passworts zeigt Ihnen das Feld Passwortsicherheit, wie sicher Ihr Passwort ist. Die Farbe grün bedeutet, dass Sie ein sicheres Passwort eingegeben haben. Ein sicheres Passwort besteht aus mindestens 8 Zeichen und verwendet eine Kombination aus Zahlen, Buchstaben und Sonderzeichen.

    • Anrede: Anrede oder Titel, zum Beispiel Herr oder Frau.
    • Vorname: Ihr Vorname.
    • Nachname: Ihr Nachname.
    • Organisation und Adresse: Name und Adresse Ihrer Organisation.
  3. Wählen Sie im Feld Land das Land, in dem sich Ihre Organisation befindet.
  4. Geben Sie die im Bild Sicherheitsüberprüfung angezeigten Zeichen ein. Achten Sie dabei auf Groß-/Kleinschreibung.
  5. Lesen und akzeptieren Sie die allgemeinen Geschäftsbedingungen sowie die Datenschutzerklärung.
  6. Bestätigen Sie Ihre Registrierung mit Registrieren.

    Per E-Mail erhalten Sie eine URL, über die Sie Ihren Zugang bestätigen und den Registrierungsvorgang abschließen müssen.

  7. Zur Bestätigung Ihres Zugangs folgen Sie der URL in der E-Mail.
  8. Klicken Sie Weiter auf der Registrierungsseite.

    Auf der Seite Anmelden können Sie sich nun mit dem von Ihnen angegebenen Benutzernamen und Passwort anmelden.

    Per E-Mail erhalten Sie die URL für den Zugang zum System. Sie können diese URL für alle zukünftigen Anmeldungen verwenden.

Nach erfolgreicher Registrierung erhalten Sie automatisch die Administratorrolle für Ihre Organisation. Mit dieser Benutzerrolle haben Sie die erforderlichen Zugriffsrechte, um Ihre Organisation und deren Benutzer sowie Abonnements zu verwalten.
