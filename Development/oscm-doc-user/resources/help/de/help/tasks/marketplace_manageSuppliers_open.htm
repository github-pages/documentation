Anbieter verwalten

Auf der Seite Anbieter verwalten können Sie festlegen, wer Services auf dem ausgewählten Marktplatz veröffentlichen darf.

Sie haben einen offenen Marktplatz ausgewählt. Hier kann grundsätzlich jeder Serviceanbieter, Wiederverkäufer und Vermittler Services veröffentlichen.

Als Marktplatzeigentümer können Sie jedoch einem Anbieter jederzeit die Zulassung entziehen, zum Beispiel wenn er mit Zahlungen in Verzug ist. Die auf dem Marktplatz veröffentlichten Services des Anbieters werden automatisch deaktiviert und können nicht mehr von Kunden abonniert werden. Bestehende Abonnements sind nicht betroffen. Alle derzeit gesperrten Anbieter werden auf der Seite Anbieter verwalten angezeigt.

Einen Anbieter, den Sie gesperrt haben, können Sie jederzeit wieder auf dem Marktplatz zulassen, zum Beispiel wenn er seine Rechnungen beglichen hat.

Wenn Sie einen Anbieter sperren oder wieder zulassen, werden die Administratoren seiner Organisation automatisch per E-Mail informiert.

So sperren Sie einen Anbieter auf dem gewählten Marktplatz:

  1. Geben Sie im Feld Organisations-ID die Organisations-ID des Anbieters ein.

    Wenden Sie sich an den Anbieter, wenn Sie seine Organsations-ID nicht kennen.

  2. Klicken Sie Sperren und bestätigen Sie die Sperrung.

    Die Sperre ist sofort wirksam, und der Anbieter kann keine Services mehr auf dem Marktplatz veröffentlichen.

So heben Sie die Sperre eines Anbieters für den gewählten Marktplatz auf:

  1. Markieren Sie in der Liste den Anbieter, dessen Sperre Sie aufheben möchten.

    Sie können einen oder mehrere Anbieter markieren.

  2. Klicken Sie Zulassen.

    Die Zulassung ist sofort wirksam, und der Anbieter kann seine Services wieder auf dem Marktplatz veröffentlichen.
