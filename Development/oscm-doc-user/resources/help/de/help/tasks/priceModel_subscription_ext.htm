Abo-Preismodell definieren

Auf der Seite Abo-Preismodell definieren können Sie für ein ausgewähltes Abonnement ein spezifisches Preismodell definieren oder ändern.

Sie haben ein Abonnement für einen Service gewählt, für dessen Kostenberechnung ein externes Abrechnungssystem verwendet wird. Die Preismodelle für die Kostenberechnung werden ebenfalls in diesem System definiert und verwaltet. Für das ausgewählte Abonnement müssen Sie das passende Preismodell von dem externen Abrechnungssystem importieren. Sie können das importierte Preismodell auf der Plattform anzeigen, aber nicht ändern. Falls Änderungen erforderlich sind, müssen diese im externen Abrechnungssystem vorgenommen werden, und Sie müssen danach das Preismodell erneut von dort importieren.

So definieren oder ändern Sie ein Abo-Preismodell:

  1. In dem Bereich Definieren Sie das Preismodell klicken Sie Importieren, um das Preismodell von dem externen Abrechnungssystem zu importieren. Der Billing-Adapter, der die Plattform mit dem externen Abrechnungssystem verbindet, ist für die Bereitstellung des richtigen Preismodells verantwortlich. Über Anzeigen können Sie sich das Preismodell ansehen.
  2. Schließen Sie Ihre Eingaben mit Speichern ab.
Die Änderungen am Abo-Preismodell sind sofort gültig.
