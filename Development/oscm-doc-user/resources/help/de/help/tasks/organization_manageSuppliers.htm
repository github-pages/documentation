Serviceanbieter verwalten

Auf der Seite Serviceanbieter verwalten legen Sie fest, welche Serviceanbieter berechtigt sein sollen, marktfähige Services für einen bestimmten technischen Service zu definieren.

So verwalten Sie Serviceanbieter:

  1. Wählen Sie den technischen Service, für den Sie Serviceanbieter hinzufügen oder löschen möchten. Die Serviceanbieter, die aktuell berechtigt sind, marktfähige Services für diesen technischen Service zu definieren, werden angezeigt.
  2. Um einen Serviceanbieter zur Liste der berechtigten Serviceanbieter hinzuzufügen, geben Sie die ID des Serviceanbieters im Feld Organisations-ID ein und klicken Hinzufügen. Wenden Sie sich an den Serviceanbieter oder Ihren Plattform-Betreiber, wenn Sie die Organisations-ID nicht kennen.
  3. Um einen Serviceanbieter aus der Liste der berechtigten Serviceanbieter zu löschen, markieren Sie den Serviceanbieter in der Liste und klicken Löschen.

    Wenn Kunden einen auf dem technischen Service basierenden marktfähigen Service des Serviceanbieters abonniert haben, wird eine entsprechende Meldung ausgegeben. Die Meldung listet die IDs der zugrunde liegenden Anwendungsinstanzen auf. Um die IDs der Instanzen Ihrer technischen Services anzuzeigen, können Sie einen Instanzenbericht erstellen.
