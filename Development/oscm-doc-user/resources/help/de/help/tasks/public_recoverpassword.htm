Passwort zurücksetzen

Auf der Seite Passwort zurücksetzen starten Sie den Vorgang, mit dem Sie ein neues Passwort für Ihren Benutzernamen festlegen.

So setzen Sie Ihr Passwort zurück:

  1. Geben Sie Ihren Benutzernamen ein.

    Diese Eingabe müssen Sie vornehmen.

    Falls Sie Ihren Benutzernamen vergessen haben, wenden Sie sich an Ihren Administrator. Dieser kann sich alle Benutzernamen anzeigen lassen, die für Ihre Organisation registriert sind.

  2. Geben Sie die im Bild Sicherheitsüberprüfung angezeigten Zeichen ein. Achten Sie dabei auf Groß-/Kleinschreibung.
  3. Mit Weiter starten Sie den Vorgang zum Festlegen eines neuen Passworts.

Sie erhalten eine Email mit einem Link zu einer Webseite, auf der Sie ein neues Passwort eingeben können. Klicken Sie den Link in der Email, um die Seite anzuzeigen.

Hinweis: Wenn Sie keine Email erhalten, haben Sie einen falschen Benutzernamen eingegeben. Sie erhalten in diesem Fall keine Fehlermeldung. Überprüfen Sie Ihren Benutzenamen und versuchen Sie es erneut.
