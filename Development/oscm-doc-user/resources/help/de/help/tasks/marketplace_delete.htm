Marktplatz löschen

Auf der Seite Marktplatz löschen können Sie einen Marktplatz von Ihrer Plattform löschen.

Wenn Sie einen Marktplatz löschen, werden automatisch alle auf dem Marktplatz veröffentlichten Services deaktiviert. Kunden können diese Services nicht mehr abonnieren. Laufende Abonnements sind nicht betroffen.

So löschen Sie einen Marktplatz:

  1. Wählen Sie in der Liste Marktplatz den zu löschenden Marktplatz.

    Die Liste stellt alle Marktplätze zur Auswahl, die auf Ihrer Plattform angelegt wurden.

  2. Klicken Sie Löschen und bestätigen Sie den Löschauftrag.
