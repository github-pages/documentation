Profil bearbeiten

Auf der Seite Profil bearbeiten können Sie Ihre persönlichen Daten ändern und die Sprache festlegen, in der Sie arbeiten möchten. Wenn Sie für Ihre Organisation die Administratorrolle haben, können Sie zusätzlich die Organisations-ID anzeigen und die Organisationsdaten ändern.

Hinweis: Falls Sie in einigen Feldern auf dieser Seite keine Eingaben machen können, verwendet Ihre Organisation ein externes Authentifizierungssystem zur Benutzerverwaltung. Die Daten können nur im externen System geändert werden.

So bearbeiten Sie Ihre persönlichen Daten:

  1. Ändern Sie die Daten in den entsprechenden Eingabefeldern.
    • Anrede, Vorname und Nachname: Anrede oder Titel, zum Beispiel Herr oder Frau, Ihr Vorname und Ihr Nachname.
    • E-Mail: Ihre E-Mail-Adresse.

      Das System validiert, ob die Syntax der E-Mail-Adresse gültig ist und ob der Domain-Name den Standards der Internet Assigned Numbers Authority (IANA) entspricht.

      Beispiele: user.name@domain.arpa, user.name@domain.org, user@mycompany.lan.uk

    • Sprache: Sprache, in der Ihnen die Benutzeroberfläche, Dokumentation und E-Mails angezeigt werden. Der Plattform-Betreiber definiert, welche Sprachen unterstützt werden.

      Wenn eine von Ihnen ausgewählte Sprache vom Plattform-Betreiber deaktiviert wurde, werden Ihnen die Benutzeroberfläche, Dokumentation und E-Mails in der Standardsprache Englisch angezeigt. Sie erhalten ein Meldung, dass Sie Ihr Benutzerprofil bearbeiten sollten. Sie können entweder die Standardsprache Englisch in Ihrem Profil speichern oder eine andere unterstützte Sprache auswählen.

  2. Schließen Sie Ihre Eingaben mit Speichern ab.
So bearbeiten Sie die Daten Ihrer Organisation (nur Administratoren):
  1. Ändern Sie die Daten in den entsprechenden Eingabefeldern.
    • Organisation: Name der Organisation, zum Beispiel der Firmenname.
    • E-Mail: E-Mail-Adresse der Organisation für den allgemeinen E-Mail-Verkehr, zum Beispiel info@firma.com.

      Das System validiert, ob die Syntax der E-Mail-Adresse gültig ist und ob der Domain-Name den Standards der Internet Assigned Numbers Authority (IANA) entspricht.

      Beispiele: user.name@domain.arpa, user.name@domain.org, user@mycompany.lan.uk

    • E-Mail-Support: Nur für Administratoren von Serviceanbietern, Vermittlern und Wiederverkäufern. E-Mail-Adresse des Kunden-Supports Ihrer Organisation, zum Beispiel support@organisation.com. Wenn Sie diese Adresse nicht angeben, wird bei Kundenanfragen zu abonnierten Services die Adresse für den allgemeinen E-Mail-Verkehr verwendet.
      Hinweis: Als Vermittler können Sie die E-Mail-Adresse des Kunden-Supports Ihrer Organisation in Ihrem Profil speichern. Kundenanfragen zu Ihren Services werden jedoch direkt an den Kunden-Support des jeweiligen Serviceanbieters weitergeleitet.
    • Telefon: Telefonnummer der Organisation.
    • Webseite: URL der Webseite der Organisation. Diese URL ist für Serviceanbieter von Bedeutung. Sie wird Kunden angezeigt, die Details zu dem Anbieter anfordern, dessen Service sie abonnieren möchten.
    • Bild für Profil: Klicken Sie Durchsuchen oder das Feld Bild hochladen und wählen Sie die Datei mit dem gewünschten Bild. Folgende Bildformate werden unterstützt: GIF, JPG und PNG. Die maximal unterstützte Größe ist 154 x 154 Pixel. Die empfohlene Größe ist 154 x 60 Pixel. Das Bild ist nur für Serviceanbieter relevant. Es wird Kunden zusammen mit weiteren Informationen zum Serviceanbieter angezeigt, wenn sie Details zu einem Service dieses Anbieters auf Marktplätzen anzeigen.
    • Adresse: Adresse der Organisation.
    • Land: Land, in dem die Organisation ihren Sitz hat.
    • Beschreibung: Beschreibung der Organisation. Die Beschreibung kann beispielsweise Informationen aus dem Unternehmensprofil bereitstellen. Sie wird in der in Ihrem Benutzerprofil eingestellten Sprache gespeichert. Um den Text in einer anderen Sprache zu hinterlegen, ändern Sie die Sprache in Ihrem Benutzerprofil und geben dann die entsprechende Übersetzung ein. Die Beschreibung ist nur für Serviceanbieter relevant. Sie wird Kunden angezeigt, die Details zu einem Anbieter anfordern, dessen Service sie abonnieren möchten.
  2. Schließen Sie Ihre Eingaben mit Speichern ab.
