Kunden verwalten

Auf der Seite Kunden verwalten können Sie die Organisationsdaten einsehen, die aktuell für Ihre Kunden definiert sind. Sie können Attributwerte für eigene Attribute eingeben, vorausgesetzt, es wurden eigene Attribute für Kunden definiert, und Sie können Kunden Rabatte gewähren.

So verwalten Sie einen Kunden:

  1. Wählen Sie im Feld Organisations-ID den Kunden, dessen Daten Sie bearbeiten möchten.

    Die Liste enthält alle Kunden, die einen Ihrer Services abonniert haben oder die Sie als Serviceanbieter selbst registriert haben, unabhängig davon, ob bereits ein Service abonniert wurde. An den Namen der Kundenorganisation ist jeweils die ID der Organisation angehängt.

  2. Wenn eigene Attribute für Ihre Kunden und die Kunden Ihrer Vermittler definiert sind und Sie die erforderlichen Attributwerte festlegen dürfen, können Sie diese in der Tabelle Attribute eingeben.

    Ein Attributwert darf maximal 255 Zeichen lang sein. Die Werte werden in die Rechnungsdaten für den Kunden übernommen, sobald die Daten neu generiert werden. Wenn Sie ein Feld für einen Attributwert leer lassen, wird der Attributname ohne einen Attributwert in die Rechnungsdaten übernommen.

    Die Tabelle Attribute listet auch eigene Attribute auf, deren Werte vom Administrator oder Abo-Manager des Kunden beim Abonnieren eines Service eingegeben werden sollen. Sie können diese Werte nicht ändern.

  3. Um einem Kunden einen Preisnachlass zu gewähren, geben Sie die erforderlichen Daten im Abschnitt Legen Sie einen Rabatt für den Kunden fest: ein.

    Einem Kunden kann ein Preisnachlass gewährt werden, der für alle seine Abonnements von Services gilt, die das plattformeigene Abrechnungssystem verwenden. Der Rabatt kann ab sofort oder ab einem künftigen Monat gelten und auf einen bestimmten Zeitraum begrenzt werden. Vor Ablauf dieses Zeitraums wird der Kunde per E-Mail informiert, so dass er rechtzeitig reagieren und den Serviceanbieter kontaktieren kann.

    Ein Rabatt wird als Prozentsatz definiert, der von den Gesamtkosten für ein Abonnement abgezogen wird. Er wird auf alle Kosten innerhalb eines Abrechnungszeitraums gewährt, in dem der Rabatt gilt, unabhängig davon, ob der Rabatt für den gesamten oder nur für einen Teil des Abrechnungszeitraums definiert wurde. Wenn Sie einen Rabatt ändern, werden die Preisnachlässe berücksichtigt, sobald neue Rechnungsdaten generiert werden. Üblicherweise ändern Sie einen Rabatt nur in Absprache mit dem jeweiligen Kunden.

    Geben Sie die erforderlichen Informationen wie folgt ein:

    • Legen Sie im Feld Prozent (%) den Rabattsatz fest. Bestimmen Sie also, wieviel Prozent vom regulären Preis für ein Abonnement bei diesem Kunden abgezogen werden sollen.
    • Soll der Rabatt ab dem aktuellen Monat gelten, lassen Sie das Feld Von leer. Ansonsten geben Sie den Monat und das Jahr ein, ab dem der Rabatt gewährt werden soll. Das Format ist MM/JJJJ, z.B. 11/2010. Ein in der Vergangenheit liegender Monat wird nicht ausgewertet. Rabatte werden immer ab dem aktuellen Abtrechnungszeitraum oder ab dem angegebenen Monat gewährt.
    • Soll der Rabatt nur für eine bestimmte Anzahl von Monaten gelten, geben Sie im Feld Bis den entsprechenden Monat und das Jahr ein. Das Format ist MM/JJJJ, z.B. 11/2012. Wenn Sie dieses Feld leer lassen, gilt der Rabatt so lange, bis Sie Ihre Angaben ändern.

    Soll zum Beispiel ein Kunde ab sofort und für einen Zeitraum von einem Jahr von einem Rabatt profitieren, lassen Sie das Feld Von leer und geben im Feld Bis den vorangegangenen Monat und das nächste Jahr an.

  4. Schließen Sie Ihre Eingaben mit Speichern ab.

Die Administratoren des Kunden werden per E-Mail über Ihre Änderungen informiert.
