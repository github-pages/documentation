Service kopieren

Auf der Seite Service kopieren können Sie einen marktfähigen Service kopieren. Einen marktfähigen Service zu kopieren ist sinnvoll, wenn Sie einen existierenden marktfähigen Service mit leicht veränderten Eigenschaften anbieten möchten. So können Sie einen Service mit einem besseren Service-Level zu einem höheren Preis anbieten oder Preisnachlässe für einen Service im Rahmen einer Werbekampagne definieren.

Wenn Sie einen marktfähigen Service kopieren, werden folgende Aktionen ausgeführt:

  • Es wird ein neuer marktfähiger Service angelegt, dem derselbe technische Service zugrunde liegt wie dem kopierten marktfähigen Service.
  • Die Servicedefinition wird kopiert.
  • Das Preismodell wird kopiert.

So kopieren Sie einen Service:

  1. Wählen Sie den Service, den Sie kopieren möchten.
  2. Geben Sie im Feld Service-ID eine Kennung für die Kopie ein.

    Service-IDs dürfen maximal 40 Zeichen lang sein. Die folgenden Zeichen sind nicht erlaubt:

    ! " # $ % & ' * + , / : ; < = > ? \ ^ `

    Die Service-ID wird zum Beispiel verwendet, wenn Sie den Service aktualisieren oder löschen oder wenn Sie Up-/Downgrade-Optionen oder ein Preismodell für den Service festlegen.

  3. Klicken Sie Kopieren.

Die Kopie des Service wird angelegt. Sie können die Definition und das Preismodell des neuen Service nach Bedarf bearbeiten.

Damit Kunden den neuen Service abonnieren können, müssen Sie die Optionen für dessen Veröffentlichung festlegen und ihn aktivieren. Sie können nach Bedarf Up-/Downgrade-Optionen für den Service festlegen. Außerdem können Sie Vermittlern und Wiederverkäufern die Berechtigung zum Wiederverkauf des Service erteilen.
