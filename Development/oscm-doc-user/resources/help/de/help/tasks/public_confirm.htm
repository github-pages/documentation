Anmelden nach Bestätigung der Registrierung

Sobald Sie Ihren Zugang bestätigt haben, können Sie sich anmelden. Klicken Sie dazu Weiter auf der Seite Registrieren.

Per E-Mail erhalten Sie danach die URL für den Zugang zum System. Sie können diese URL für alle zukünftigen Anmeldungen verwenden.
