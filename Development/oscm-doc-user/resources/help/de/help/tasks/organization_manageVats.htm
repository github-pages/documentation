Mehrwertsteuersätze verwalten

Auf der Seite Mehrwertsteuersätze verwalten können Sie folgende Aufgaben durchführen:

  • Sie können festlegen, ob die Berechnung der Nutzungsgebühren mit Mehrwertsteuer erfolgen soll.
  • Sie können einen allgemeinen Mehrwertsteuersatz definieren, der für alle Preise Ihrer Kunden gelten soll.
  • Sie können einen landesspezifischen Mehrwertsteuersatz für jedes Land definieren, in dem Sie Ihre Services anbieten möchten.
  • Sie können einen kundenspezifischen Mehrwertsteuersatz definieren. Dies ist beispielsweise erforderlich, wenn eine Kundenorganisation eine Niederlassung in einem anderen Land hat als Ihre Hauptverwaltung.

Die Mehrwertsteuersätze wirken sich wie folgt auf die Berechnung der Nutzungsgebühren für einen Kunden aus:

  • Wenn Sie die Berechnung mit Mehrwertsteuer deaktivieren, werden die Preise netto berechnet. Es wird keine Mehrwertsteuer auf die Gesamtkosten erhoben.
  • Ein kundenspezifischer Mehrwertsteuersatz hat immer Vorrang vor dem allgemeinen oder einem landesspezifischen Mehrwertsteuersatz.
  • Der Mehrwertsteuersatz des Landes, in dem sich die Kundenorganisation befindet, wird verwendet, wenn kein kundenspezifischer Mehrwertsteuersatz festgelegt wurde.
  • In allen anderen Fällen wird der allgemeine Mehrwertsteuersatz verwendet.

Der Mehrwertsteuersatz wirkt sich nicht auf die Elemente des Preismodells aus. Die Mehrwertsteuer wird zu den Gesamtkosten addiert. Die Summe ergibt den vom Kunden zu zahlende Bruttobetrag. Achten Sie darauf, dass Sie die richtigen gesetzlichen Mehrwertsteuersätze angeben.

Wenn Sie einem Wiederverkäufer die Berechtigung zum Verkauf eines Service erteilen, hat der von Ihnen definierte Mehrwertsteuersatz keine Auswirkung auf die Kunden des Wiederverkäufers. Die Nutzungsgebühren seiner Kunden werden immer ohne Mehrwertsteuer berechnet. Wenn Sie einem Vermittler die Berechtigung zum Verkauf eines Service erteilen, wird die von ihnen definierte Mehrwertsteuer bei der Berechnung der Nutzungsgebühren berücksichtigt.

So verwalten Sie Mehrwertsteuersätze:

  • Markieren Sie Berechnung mit Mehrwertsteuer aktiviert, um die Berechnung mit Mehrwertsteuer für die Kunden Ihrer Organisation einzuschalten. So werden die Nutzungsgebühren für Abonnements Ihrer Kunden als Bruttobeträge berechnet. Diese Einstellung wirkt sich auf alle Preise und alle Ihre Kunden aus.

    Um die Berechnung mit Mehrwersteuer zu deaktivieren, entfernen Sie die Markierung. Wenn die Markierung entfernt ist, können Sie keinen allgemeinen, landes- oder kundenspezifischen Mehrwertsteuersatz festlegen. Alle Nutzungsgebühren werden netto berechnet.

    Beachten Sie, dass alle von Ihnen definierten Mehrwertsteuersätze gelöscht werden, wenn Sie die Markierung entfernen.

  • Geben Sie im Feld Allgemeiner Mehrwertsteuersatz (%) den Wert ein, der für alle Organisationen verwendet werden soll, sofern kein landes- oder kundenspezifischer Satz definiert ist. Geben Sie den Mehrwertsteuersatz in Prozent als Dezimalzahl mit maximal zwei Dezimalstellen an.
  • So legen Sie einen landesspezifischen Mehrwertsteuersatz fest:
    1. Wählen Sie in der Liste Land das Land. Es werden nur die Länder aufgelistet, für die noch kein Mehrwertsteuersatz festgelegt wurde.
    2. Geben Sie im Feld Mehrwertsteuersatz (%) neben dem Namen des Landes den geltenden Wert ein. Geben Sie den Wert in Prozent als Dezimalzahl mit maximal zwei Dezimalstellen an.
  • So legen Sie einen kundenspezifischen Mehrwertsteuersatz fest:
    1. Wählen Sie in der Liste Kunde die Kundenorganisation. Es werden nur die Kunden aufgelistet, für die noch kein Mehrwertsteuersatz festgelegt wurde.
    2. Geben Sie im Feld Mehrwertsteuersatz (%) neben dem Namen des Kunden den geltenden Wert ein. Geben Sie den Wert in Prozent als Dezimalzahl mit maximal zwei Dezimalstellen an.
  • Ändern Sie einen Mehrwertsteuersatz, indem Sie den Prozentsatz im jeweiligen Feld überschreiben.

Schließen Sie Ihre Eingaben mit Speichern ab.
