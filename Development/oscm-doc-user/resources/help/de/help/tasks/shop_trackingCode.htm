Tracking Code hinzufügen

Auf der Seite Tracking Code hinzufügen können Sie Code zu den Seiten Ihres Marktplatzes hinzufügen oder entfernen.

Tracking Code dient zur Verfolgung der Besuche auf Ihrem Marktplatz. Zum Beispiel können Sie damit die Häufigkeit messen, mit der Marktplatzbesucher Ihren Marktplatz aufrufen. Sie können jede beliebige Web-Analyse-Software verwenden. Tracking Code wird als HTML-Code angegeben. Dieser HTML-Code wird auf allen Seiten Ihres Marktplatzes eingefügt. Sobald eine Seite von einem Besucher geladen wird, wird das entsprechende Skript aufgerufen und Sie können die Daten auswerten.

So fügen Sie Tracking Code hinzu:

  1. Wählen Sie in der Liste Marktplatz den Marktplatz, dem Sie Tracking Code hinzufügen möchten.

    Die Liste enthält alle Marktplätze, auf die Sie als Marktplatz-Manager Zugriff haben.

  2. Geben Sie den HTML-Code ein. Falls erforderlich, können Sie JavaScript-Code verwenden.
    Hinweis: Der Code kann zu Fehlern im Seitenlayout des Marktplatzes führen. Zudem können Änderungen am Layout Anpassungen des Marktplatzes zunichte machen.

    Sie sind verantwortlich für ein fehlerfreies Seitenlayout.

  3. Schließen Sie Ihre Eingaben mit Speichern ab.

Der Code wird auf allen Seiten Ihres Marktplatzes eingefügt. Die Änderung wird wirksam, sobald Besucher sich das nächste Mal am Marktplatz anmelden oder den Inhalt ihres Browsers neu laden.

So entfernen Sie Tracking Code:

  1. Wählen Sie in der Liste Marktplatz den Marktplatz, für den Sie Tracking Code löschen möchten.

    Die Liste enthält alle Marktplätze, auf die Sie als Marktplatz-Manager Zugriff haben.

  2. Löschen Sie den HTML-Code.
  3. Schließen Sie Ihre Änderungen mit Speichern ab.

Der Code wird von den Seiten Ihres Marktplatzes gelöscht. Die Änderung wird wirksam, sobald Besucher sich das nächste Mal am Marktplatz anmelden oder den Inhalt ihres Browsers neu laden.
