Organisation verwalten

Auf der Seite Organisation verwalten können Sie folgende Aufgaben durchführen:

  • Der Organisation eine weitere Rolle zuweisen (Hersteller, Serviceanbieter, Wiederverkäufer, Vermittler).
  • Adress- und Kontaktdaten der Organisation aktualisieren.
  • Den Distinguished Name für die Organisation festlegen.

    Dies ist erforderlich, wenn eine Organisation Zertifikate für die vertrauenswürdige Kommunikation zwischen der Betreiberorganisation und sich selbst verwenden will.

    Die Plattform überprüft den Distinguished Name im Client-Zertifikat. Falls er mit dem hier festgelegten übereinstimmt, ist der Aufrufer berechtigt, eine Verbindung zur Plattform herzustellen.

    Den Distinguished Name einer Organisation können Sie aus ihrem Zertifikat mit folgendem Aufruf in Ihrer Java-Laufzeitumgebung ermitteln: 

    <IhrJRE>/bin/keytool -printcert -file <Dateiname.crt>

    Detaillierte Informationen zur Verwendung von Zertifikaten finden Sie im Operator's Guide.

  • Zahlungsarten festlegen, die Serviceanbieter oder Wiederverkäufer ihren Kunden für Services anbieten können, die das plattformeigene Abrechnungssystem verwenden.

    Standardmäßig kann jeder Kunde auf Rechnung bezahlen. Somit ist gewährleistet, dass Kunden jederzeit kostenpflichtige Services abonnieren können.

    Wenn ein Serviceanbieter oder Wiederverkäufer eine weitere Zahlungsart anbieten will, muss er einen Vertrag mit einem kommerziellen Zahlungsdienstleister (PSP) abschließen.

    Detaillierte Informationen zur Integration eines Zahlungsdienstleister finden Sie im Operator's Guide.

So verwalten Sie eine Organisation:

  1. Wählen Sie in der Liste Organisations-ID die Organisation, deren Daten Sie bearbeiten möchten.

    Es werden alle Organisationen angezeigt, die auf Ihrer Plattform verwaltet werden.

  2. Bearbeiten Sie die Organisationsdaten:
    • Organisation. Name der Organisation, zum Beispiel der Firmenname.
    • E-Mail. E-Mail-Adresse der Organisation für den allgemeinen E-Mail-Verkehr. Die Organisation kann später auch eine E-Mail-Adresse für Rechnungen hinterlegen.

      Das System validiert, ob die Syntax der E-Mail-Adresse gültig ist und ob der Domain-Name den Standards der Internet Assigned Numbers Authority (IANA) entspricht.

      Beispiele: user.name@domain.arpa, user.name@domain.org, user@mycompany.lan.uk

    • Telefon. Telefonnummer der Organisation.
    • Webseite. URL der Webseite der Organisation. Kunden können diese Webseite aufrufen, wenn sie Informationen zu der Organisation anzeigen, die einen Service anbietet.
    • Sprache: Sprache, in der die Organisation normalerweise arbeitet.
    • Adresse der Organisation. Der Administrator kann später eine weitere Adresse angeben, die auf Rechnungen gedruckt werden soll.
    • Land, in dem die Organisation ansässig ist.
    • Beschreibung der Organisation. Die Beschreibung kann beispielsweise Informationen aus dem Unternehmensprofil bereitstellen.
      Hinweis: Verwenden Sie ausschließlich die Funktionen des Editors, um Textformatierungen vorzunehmen. Direkte HTML-Codierungen im Text sowie Verweise auf externe HTML-Stylesheets oder -Inhalte können zu Fehlern im Seitenlayout führen.
    • Marktplatz, den der Kunde besuchen soll. Sie können einen der Marktplätze wählen, zu denen Sie Zugang vom jeweiligen Marktplatzeigentümer erhalten haben.
    • Für eine Organisation mit der Rolle Serviceanbieter: Umsatzbeteiligung Betreiber. Diese zwischen Betreiber und Serviceanbieter vereinbarte Umsatzbeteiligung erhält der Betreiber vom Serviceanbieter dafür, dass Services über die vom Betreiber bereitgestellten Marktplätze verkauft werden können. Diese vereinbarte Beteiligung gilt für alle Services, die der Serviceanbieter anbietet, sofern Sie nicht eine Umsatzbeteiligung pro Service festlegen (einstellbar über die Menüoption Umsatzbeteilung des Betreibers verwalten im Menü Betrieb). Die Beteiligung ist auch unabhängig davon, ob der Serviceanbieter seine Services selbst verkauft oder die Dienstleistung von Vermittlern oder Wiederverkäufern in Anspruch nimmt. Rabatte, die ein Serviceanbieter mit seinen Kunden vereinbart, wirken sich auf die Umsatzbeteiligung aus.
      Hinweis: Umsatzbeteiligungen und Rabatte werden nur für Services errechnet, die das plattformeigene Abrechnungssystem verwenden. Services, die ein externes Abrechnungssystem verwenden, werden nicht berücksichtigt.
    • Für eine Organisation mit der Rolle Hersteller, Serviceanbieter, Wiederverkäufer und/oder Vermittler:

      Bild für Profil: Bild, das einen Anbieter auf einem Marktplatz identifiziert. Kunden sehen dieses Bild zusammen mit den Informationen zu der Organisation, die einen Service anbietet.

      Klicken Sie Durchsuchen oder das Feld Bild hochladen und wählen Sie die Datei mit dem gewünschten Bild. Folgende Bildformate werden unterstützt: GIF, JPG und PNG. Die maximal unterstützte Größe ist 154 x 154 Pixel. Die empfohlene Größe ist 154 x 60 Pixel.

  3. Wählen Sie im Bereich Organisationsrolle eine neue Rolle, falls erforderlich. Die Rollen Wiederverkäufer und Vermittler können nicht zusammen oder mit der Hersteller- oder Serviceanbieterrolle zugewiesen werden.
  4. Geben Sie den Distinguished Name der Organisation an.

    Der Distinguished Name ist im Zertifikat der Organisation definiert und besteht aus folgenden Elementen:

    • CN - Common Name
    • OU - Organizational Unit
    • O - Organization
    • L - Location
    • ST - State
    • C- Country

    Die einzelnen Elemente müssen durch Komma getrennt werden.

  5. Legen Sie die Zahlungsarten fest, die die Organisation ihren Kunden anbieten kann. Klicken Sie Hinzufügen und fahren Sie auf der Seite Neue Zahlungsart fort.

    Das Hinzufügen von Zahlungsarten ist nur für Serviceanbieter oder Wiederverkäufer möglich.

    Hinweis: Eine einmal hinzugefügte Zahlungsart kann nicht mehr entfernt werden.
  6. Schließen Sie Ihre Eingaben mit Speichern ab.

Die Administratoren der Organisation werden per E-Mail über Ihre Änderungen informiert.
