Zahlungsarten verwalten

Auf der Seite Zahlungsarten verwalten legen Sie fest, wie Kunden für die Nutzung Ihrer Abonnements bezahlen können, zum Beispiel Zahlung auf Rechnung, per Kreditkarte oder Bankeinzug. Sie können Zahlungsarten für einzelne Kunden und für einzelne Services festlegen.

Hinweis: Zahlungsarten wie auch Rechnungsanschriften sind nur relevant für kostenpflichtige Services, die das plattformeigene Abrechnungssystem verwenden. Auch für diese Services kann der Plattform-Betreiber alle Möglichkeiten für den Kunden zum Angeben solcher Zahlungsinformationen deaktivieren. Falls dies der Fall ist, wird auf der Seite Zahlungsarten verwalten ein entsprechender Hinweis angezeigt und die Definition von Zahlungsarten erübrigt sich.

Es ist Aufgabe Ihres Plattform-Betreibers, die Zahlungsarten einzurichten, die Sie anbieten können. Zahlung auf Rechnung ist automatisch für jeden Serviceanbieter eingerichtet. Das bedeutet, dass Sie Ihren Kunden immer Zahlung auf Rechnung anbieten können ohne zusätzliche Vorbereitungen durch Ihren Plattform-Betreiber. Sie sind bei dieser Zahlungsart für die Erstellung der Rechnungen und die Zahlungsabwicklung zuständig.

Wenn Sie Ihren Kunden eine andere Zahlungsart anbieten möchten, benötigen Sie eine vertragliche Vereinbarung mit einem Zahlungsdienstleister. Ihr Plattform-Betreiber kann dann die entsprechenden Zahlungsarten für Sie einrichten. Die Verantwortung für die Erstellung der Rechnungen und die Zahlungsabwicklung liegt beim Zahlungsdienstleister.

Wenn Sie entscheiden, einem Kunden für einen kostenpflichtigen Service mehrere Zahlungsarten anzubieten, kann sich der Kunde beim Abonnieren des Service seine bevorzugte Zahlungsart aussuchen. Wenn Sie eine Zahlungsart löschen, die bereits von einem Kunden für ein Abonnement ausgewählt wurde, muss Ihr Kunde eine neue Zahlungsart wählen, bevor er den Service weiter nutzen kann.

Beachten Sie die folgenden Hinweise zum Löschen einer Zahlungsart:

  • Wenn Sie eine Zahlungsart während der kostenlosen Nutzungsdauer eines Kunden löschen, wird die kostenlose Nutzungsdauer nicht um den Zeitraum verlängert, in dem der Kunde den Service nicht nutzen kann.
  • Wenn Sie eine Zahlungsart für ein Abonnement löschen, bei dem die Kosten pro Zeiteinheit abgerechnet werden, können dem Kunden bei gestaffelten Preisen zusätzliche Kosten entstehen. Von zusätzlichen Kosten ist jeder Kunde betroffen, der den Service nicht innerhalb der Zeiteinheit weiter nutzt, in der Sie die Zahlungsart löschen. Bei einem Kunden, der einen Service nach erneuter Auswahl einer Zahlungsart weiter nutzt, wird das Abonnement wie ein neues Abonnement behandelt. Dadurch ergeben sich zum Beispiel zusätzliche Kosten für die Summe der Zeiteinheiten, die die Benutzer dem Abonnement zugeordnet sind, oder für die Anzahl der abrechenbaren Ereignisse.

Auf der Seite Zahlungsarten verwalten können Sie Zahlungsarten für Services und Kunden festlegen. Die Schnittmenge aus beidem bestimmt, welche Zahlungsart der Kunde wählen kann, wenn er einen Service abonniert. Wenn die Schnittmenge leer ist, das heißt, die für Services und für Kunden festgelegten Zahlungsarten haben keine Zahlungsarten gemeinsam, steht dem Kunden keine Zahlungsart zur Verfügung. In einem solchen Fall ist es dem Kunden nicht möglich, einen kostenpflichtigen Service zu abonnieren.

So legen Sie eine Zahlungsart fest:

  1. Markieren Sie die Zahlungsart im jeweiligen Bereich. Sie können eine Zahlungsart für alle neuen Services, alle neuen Kunden, einzelne Services oder einzelne Kunden festlegen.
  2. Schließen Sie Ihre Eingaben mit Speichern ab.

Die festgelegten Zahlungsarten stehen Ihren Kunden sofort zur Auswahl.

So entfernen Sie eine Zahlungsart:

  1. Entfernen Sie die Markierung für eine Zahlungsart im jeweiligen Bereich.
  2. Beenden Sie Ihre Eingaben mit Speichern.

Die Zahlungsart wird sofort entfernt und steht nicht mehr zur Auswahl.
