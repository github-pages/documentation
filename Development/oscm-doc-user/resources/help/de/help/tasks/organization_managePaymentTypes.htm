Zahlungsarten hinzufügen

Auf der Seite Neue Zahlungsart können Sie die Zahlungsarten definieren, die dem gewählten Serviceanbieter bzw. Wiederverkäufer zur Auswahl stehen sollen. Später kann der Serviceanbieter bzw. Wiederverkäufer festlegen, welche Zahlungsarten er seinen Kunden zur Verfügung stellen will.

So fügen Sie eine Zahlungsart hinzu:

  1. Wählen Sie in der Liste Zahlungsdienstleister denjenigen, den Sie dem Serviceanbieter bzw. Wiederverkäufer zuweisen möchten.

    Die Liste enthält alle Zahlungsanbieter, die auf Ihrer Plattform registriert und konfiguriert sind.

  2. Geben Sie im Feld PSP-ID die vom Zahlungsdienstleister festgelegte ID für den Serviceanbieter bzw. Wiederverkäufer ein. Der Serviceanbieter bzw. Wiederverkäufer muss ein Vertragsverhältnis mit dem Zahlungsdienstleister haben. Serviceanbieter und Wiederverkäufer werden beim Zahlungsdienstleister im PSP-System registriert. Über eine eindeutige ID können Zahlungen dem Konto des Serviceanbieters bzw. Wiederverkäufers zugeordnet werden.
  3. Wählen Sie die Zahlungsarten, die dem Serviceanbieter bzw. Wiederverkäufer für seine Kunden zur Verfügung stehen sollen.

    Neben Zahlungsart werden alle Zahlungsarten angezeigt, die für den Zahlungsdienstleister definiert sind. Sie sind abhängig von der Konfiguration des Zahlungsdienstleisters.

  4. Schließen Sie Ihre Eingaben mit Speichern ab.
    Hinweis: Eine einmal hinzugefügte Zahlungsart kann nicht mehr entfernt werden.

Fahren Sie mit der Verwaltung der Organisation fort.
