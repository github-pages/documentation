Temporäres Passwort ändern

Auf der Seite Temporäres Passwort ändern müssen Sie das Passwort ändern, das Sie für Ihre erste Anmeldung erhalten haben.

So ändern Sie das Passwort:

  1. Geben Sie das aktuelle (temporäre) Passwort und ein neues Passwort ein und wiederholen Sie das neue Passwort im entsprechenden Eingabefeld.

    Das Passwort muss aus mindestens 6 Zeichen bestehen. Es darf nicht länger als 100 Zeichen sein.

    Beim Eingeben des Passworts zeigt Ihnen das Feld Passwortsicherheit, wie sicher Ihr Passwort ist. Die Farbe grün bedeutet, dass Sie ein sicheres Passwort eingegeben haben. Ein sicheres Passwort besteht aus mindestens 8 Zeichen und verwendet eine Kombination aus Zahlen, Buchstaben und Sonderzeichen.

  2. Schließen Sie Ihre Eingaben mit Ändern ab.

Das neue Passwort wird gespeichert. Mit diesem Passwort können Sie sich ab sofort anmelden.
