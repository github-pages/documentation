Anbieter verwalten

Auf der Seite Anbieter verwalten können Sie festlegen, wer Services auf dem ausgewählten Marktplatz veröffentlichen darf.

Sie haben einen Marktplatz für zugelassene Anbieter ausgewählt. In diesem Fall bestimmen Sie explizit einen oder mehrere Serviceanbieter, Wiederverkäufer oder Vermittler, die Services auf dem Marktplatz veröffentlichen dürfen. Beim Anlegen des Marktplatzes wird die als Marktplatzeigentümer angegebene Organisation automatisch zugelassen, wenn Sie die Rolle Serviceanbieter, Wiederverkäufer oder Vermittler hat. Alle derzeit zugelassenen Anbieter werden auf der Seite angezeigt.

Als Marktplatzeigentümer können Sie einem Anbieter jederzeit die Zulassung entziehen, zum Beispiel wenn er mit Zahlungen in Verzug ist. Die auf dem Marktplatz veröffentlichten Services des Anbieters werden automatisch deaktiviert und können nicht mehr von Kunden abonniert werden. Bestehende Abonnements sind nicht betroffen.

Wenn Sie einen Anbieter auf dem Marktplatz zulassen oder ihm die Zulassung entziehen, werden seine Administratoren automatisch per E-Mail informiert.

So lassen Sie einen Anbieter auf dem gewählten Marktplatz zu:

  1. Geben Sie im Feld Organisations-ID die Organisations-ID des Anbieters ein.

    Wenden Sie sich an den Anbieter, wenn Sie seine Organsations-ID nicht kennen.

  2. Klicken Sie Zulassen.

    Die Zulassung ist sofort wirksam, und der Anbieter kann seine Services auf dem Marktplatz veröffentlichen.

So entziehen Sie einem Serviceanbieter die Zulassung für den gewählten Marktplatz:

  1. Markieren Sie in der Liste den Anbieter, dem Sie die Zulassung entziehen möchten.

    Sie können einen oder mehrere Serviceanbieter markieren.

  2. Klicken Sie Sperren und bestätigen Sie den Auftrag.

    Die Sperre ist sofort wirksam, und der Anbieter kann keine Services mehr auf dem Marktplatz veröffentlichen.
