Umsatzbeteiligung von Wiederverkäufern verwalten

Auf der Seite Umsatzbeteiligung von Wiederverkäufern verwalten können Sie für einzelne Services, die Wiederverkäufer verkaufen dürfen, eine Umsatzbeteiligung festlegen. Diese Beteiligung pro Service hat Vorrang vor der für einen Marktplatz voreingestellten Umsatzbeteiligung für Wiederverkäufer (einstellbar über die Menüoption Marktplatzeigenschaften ändern).

Desweiteren können Sie für einzelne Wiederverkäuferorganisationen eine Umsatzbeteiligung festlegen. Diese wiederum hat Vorrang vor der Beteiligung pro Service.

Auf der Seite Umsatzbeteiligung von Wiederverkäufern verwalten können Sie auch die Umsatzbeteiligung für Services ändern, für die keine Berechtigung mehr zum Wiederverkauf besteht oder die bereits vom Serviceanbieter gelöscht wurden. Es besteht die Möglichkeit, dass noch Abonnements für diese Services laufen und dass Sie die jeweilige Umsatzbeteiligung ändern möchten.

Hinweis: Umsatzbeteiligungen werden nur für Services errechnet, die das plattformeigene Abrechnungssystem verwenden. Services, die ein externes Abrechnungssystem verwenden, werden nicht berücksichtigt.

So legen Sie die Umsatzbeteiligung für einen Wiederverkäufer fest:

  1. Wählen Sie in der Liste Service-ID den Service, für den Sie eine Umsatzbeteiligung festlegen möchten.

    Die Liste enthält alle auf der Plattform verfügbaren Services. Dazu gehören auch alle Services, die bereits gelöscht wurden. An den Namen von gelöschten Services wird eine Raute (#) sowie ein UNIX-Zeitstempel angehängt. Dieser gibt an, wann der Service gelöscht wurde. Bei allen Services ist die Organisations-ID des Serviceanbieters, der den Service zur Verfügung stellt, in Klammern angegeben.

  2. In der ersten Tabelle wird die für den Marktplatz eingestellte Umsatzbeteiligung für Wiederverkäufer angezeigt, die auch für den gewählten Service voreingestellt ist.

    Im Feld Umsatzbeteiligung (%) können Sie die Umsatzbeteiligung für den gewählten Service ändern: Geben Sie einen Prozentwert an.

    Dieser Wert gilt für alle Wiederverkäufer, die den gewählten Service veröffentlichen. Der Wert hat Vorrang vor der Einstellung, die über die Menüoption Marktplatzeigenschaften ändern festgelegt wurde.

  3. In der zweiten Tabelle werden alle Wiederverkäuferorganisationen angezeigt, die den gewählten Service verkaufen dürfen.

    Im Feld Umsatzbeteiligung (%) können Sie einen Prozentwert eingeben, der nur für die entsprechende Wiederverkäuferorganisation gilt.

    Dieser Wert hat sowohl Vorrang vor der Umsatzbeteiligung für den gewählten Service als auch vor der Einstellung, die über die Menüoption Marktplatzeigenschaften ändern festgelegt wurde.

  4. Schließen Sie Ihre Eingaben mit Speichern ab.

    Die Prozentsätze werden berücksichtigt, sobald der Wiederverkäufer den Service auf einem Marktplatz veröffentlicht. Die tatsächlichen Umsatzbeteiligungen werden bei der nächsten Neuberechnung der Nutzungsdaten errechnet und mit den Rechnungsdaten ausgegeben.
