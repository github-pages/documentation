Suchindex aktualisieren

Der Suchindex wird vom System automatisch in regelmäßigen Abständen erstellt. Er wird von Benutzern auf Marktplätzen genutzt, um nach Abonnements oder Services zu suchen.

Es kann Fälle geben, beispielsweise nach einer Datenbank-Wiederherstellung, dass der Suchindex nicht mehr konsistent ist. Um dies zu beheben, können Sie die Index-Dateien entweder manuall löschen und die entsprechende Domände des Application Servers neu starten, oder die Option Suchindex aktualisieren verwenden, um den Suchindex explizit neu zu generieren. Der Suchindex wird anhand der aktuellen Daten in der Datenbank erstellt. Existierende Index-Strukturen werden überschrieben. Der Index wird online neu erstellt, das heißt, sie müssen keine Domäne neu starten.

Auf der Seite Suchindex aktualisieren können Sie die Neuerstellung des Suchindex durch Klicken auf Aktualisieren starten.
