Probleme mit der Verbindung zur Datenbank beheben

Wenden Sie sich an Ihren Plattform-Betreiber, der die Verbindung zur Datenbank überprüfen und gegebenenfalls Fehler beheben kann. In der Datenbank werden die Zugangsdaten der Benutzer gespeichert. Die Datenbank wurde eventuell gestoppt, oder es sind andere Verbindungsprobleme aufgetreten.
