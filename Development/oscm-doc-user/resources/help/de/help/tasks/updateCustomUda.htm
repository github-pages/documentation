Eigenes Attribut bearbeiten

Auf der Seite Eigenes Attribut bearbeiten können Sie ein ausgewähltes Attribut beabeiten oder löschen.

So bearbeiten Sie ein eigenes Attribut:

  1. Ändern Sie die erforderlichen Daten.
    • Attribut-ID: ID oder Name des Attributs. Die ID darf aus maximal 40 Zeichen bestehen. Die folgenden Zeichen sind nicht erlaubt: ! " # $ % & ' * + , / : ; < > \ ^ `
    • Vordefinierter Wert: Standardwert für das Attribut. Sie können maximal 255 Zeichen eingeben. Der Standardwert wird verwendet, sofern kein spezifischer Wert für einen bestimmten Kunden oder ein bestimmtes Abonnement angegeben wird. Wenn der Wert eines eigenen Attributs vom Kunden eingegeben werden soll, ist das entsprechende Feld beim Abonnieren des Service mit dem Standardwert vorbelegt.
    • Benutzeroption: Markieren Sie diese Option, wenn Sie dem Administrator des Kunden oder dem Abo-Manager beim Abonnieren Ihres Service die Möglichkeit bieten möchten, den Attributwert selbst einzugeben.
    • Pflichtangabe: Markieren Sie diese Option, wenn die Eingabe des Attributwerts durch den Administrator des Kunden oder den Abo-Manager obligatorisch sein soll.
  2. Schließen Sie Ihre Eingaben mit Speichern ab.

Die vorherige Seite zur Verwaltung von eigenen Attributen wird wieder angezeigt. Sie sehen die geänderte Definition.

Ein geänderter Name oder ein geänderter Standardwert wird in die Rechnungsdaten übernommen, sobald diese neu generiert werden. Wenn kein Wert für ein Attribut angegeben ist, wird der Attributname ohne zugehörigen Wert in die Rechnungsdaten übernommen.

So löschen Sie ein eigenes Attribut:

  • Klicken Sie Löschen und bestätigen Sie den Löschauftrag.

Das eigene Attribut wird sofort gelöscht und die vorherige Seite zur Verwaltung von eigenen Attributen wird wieder angezeigt. Das Attribut wird bei der nächsten Generierung der Rechnungsdaten nicht mehr berücksichtigt.
