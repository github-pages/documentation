Aktionsprotokoll exportieren

Auf der Seite Aktionsprotokoll exportieren können Sie Informationen zu Benutzeraktionen exportieren, die mit Abonnements, Services und Preismodellen zusammenhängen. Ein Aktionsprotokoll kann hilfreich sein, wenn Sie beispielsweise überprüfen möchten, wann und von wem ein Preismodell geändert wurde oder wann ein Abonnement angelegt wurde.

Hinweis:

Benutzeraktionen werden nur dann protokolliert, wenn die Konfigurationseinstellung AUDIT_LOG_ENABLED auf true gesetzt ist.

So exportieren Sie das Aktionsprotokoll:

  1. Wählen Sie in der Liste Benutzerrolle die Rolle, für die Sie die Protokolldaten exportieren möchten. Wählen Sie Alle, um alle protokollierten Aktionen zu exportieren.

    Abhängig von der gewählten Benutzerrolle wird eine Aktionsliste angezeigt.

    Sie können protokollierte Aktionen von Administratoren und Standardbenutzern, die mit Abonnements zusammenhängen, exportieren sowie protokollierte Aktionen von Service-Managern von Serviceanbietern, die mit Services und Preismodellen zusammenhängen. Die Aktionen von Administratoren schließen Aktionen von OE-Administratoren und Abo-Managern ein. Aktionen von Service-Managern schließen Aktionen von Vermittlern und Wiederverkäufern ein.

  2. Wählen Sie in der Tabelle Aktion die Aktionen, für die Sie Protokolldaten exportieren möchten. Markieren Sie das Kästchen in der Kopfzeile der Tabelle, um alle Aktionen auszuwählen.
  3. Geben Sie das Start- und Endedatum des Zeitraums an, für den Sie die Protkolldaten exportieren möchten. Das Datumsformat ist JJJJ-MM-TT.
  4. Starten Sie den Export mit Exportieren.

Sie können wählen, ob die exportierten Daten angezeigt oder in einer Datei gespeichert werden sollen. Detaillierte Informationen zum Aufbau und Inhalt des Aktionsprotokolls finden Sie hier.
