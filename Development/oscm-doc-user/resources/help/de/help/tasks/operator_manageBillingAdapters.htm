Abrechnungssysteme verwalten

Es ist möglich proprietäre und Fremdabrechnungssysteme i OSCM zu benutzten. Während Hersteller die Anwendungen mit OSCM integrieren, können sie auswählen ob sie den eigenen Abrechnungssystem der Plattform oder einen externen Abrechnungssystem benutzten möchten.

Auf der Seite Abrechnungssysteme verwalten, können Sie ein externes System an OSCM anknöpfen. Um ein System erfolgreich anzuknöpfen, die folgenden Voraussetzungen müssen erfüllt werden:

  • Die durch den Betreiber bestimmten Verbindungseigenschaften müssen auf das externe Abrechnungssystem weisen.

Eins von den unterstützten Systemen muss als Standardsystem eingestellt werden. Der Standardabrechnungssystem ist für die Hersteller, die die Anwendung als einen technischen Service registrieren um sie in OSCM verfügbar zu machen, vordefiniert.

Wenn man die Seite Abrechnungssysteme verwalten besucht, werden die Details von den Abrechnungssystem, der im Moment als Standardsystem konfiguriert ist, angezeigt. Auf dieser Seite können Sie:

  • Sich die Liste der unterstützen Abrechnungssysteme ansehen.
  • Einen neuen Abrechnungssystem hinzufügen.
  • Den Namen und die Verbindungseigenschaften eines Abrechnungssystems aktualisieren.
  • Den Standardabrechnungssystem auswählen.
  • Einen Abrechnungssystem entfernen.

Um ein neues Abrechnungssystem zu definieren:

  1. Klicken Sie Hinzufügen. Das fügt einen leeren Bereich ein, wo man die Abrechnungssystemdetails definieren kann.
  2. Geben Sie das Abrechnungssystem ID von den Abrechnungssystem ein. Dieses ID identifiziert eindeutig das Abrechnungssystem im OSCM. Wenn das neue Abrechnungssystem definiert ist, kann das ID nicht geändert werden.

    Die IDs sind auf 40 Zeichen beschränkt und darf keinen von den folgenden Zeichen beinhalten:

    ! " # $ % & ' * + , / : ; < = > ? \ ^ `

  3. Fügen Sie den Namen des Abrechnungssystems ein.

    Namen sind auf 40 Zeichen beschränkt. Alle Zeichen sind erlaubt.

  4. Fügen Sie die Verbindungseigenschaften ein. Je nach dem Abrechnungsadapter, der implementiert wurde um OSCM mit dem externen Abrechnungssystem zu verbinden, kann die Zahl der Verbindungseigenschaften und ihre Semantik verschieden sein. Sie müssen aber auf ein spezifisches externes Abrechnungssystem deuten und solche Daten wie der Port den der Abrechnungssystem verbindet oder ein Provider-URL festlegen. Der Abrechnungssystem und der Adapter werden durch JNDI-lookup verbunden.

    Fügen Sie den Namen der Eigenschaft in das erste Feld und den entsprechenden Wert in das zweite Feld ein.

    Benutzen Sie den Pluszeichen um eine zusätzliche Eigenschaft hinzuzufügen und den Minuszeichen um eine Eigenschaft zu entfernen.

  5. Klicken Sie Verbindung testen um sicherzustellen, dass die Verbindung zu dem Abrechnungssystem erstellt werden kann. Eine entsprechende Nachricht wird gezeigt, die Sie informiert ob die Verbindung erfolgreich erstellt wurde.
  6. Klicken Sie Als Standard auswählen um das neue Abrechnungssystem als Standardabrechnungssystem zu definieren, wenn erforderlich. Das neue Abrechnungssystem wird sofort als Standard markiert. Der Marker wird von dem vorherigen Standardsystem entfernt.
  7. Um den Abrechnungssystem zu definieren, klicken Sie Speichern.

Hersteller können jetzt dieses Abrechnungssystem auswählen, wenn sie eine Anwendung als technischen Service in OSCM registrieren.

Um einen existierenden Abrechnungssystem zu aktualisieren:

  1. Erweitern Sie die Details des Abrechnungssystem, der aktualisiert werden muss.
  2. Ändern Sie den Namen des Abrechnungssystems, solte es erforderlich sein.
  3. Ändern Sie die Werte der Verbindungseigenschaften. Benutzten Sie den Pluszeichen um eine zusätzliche Eigenschaft hinzuzufügen oder den Minuszeichen um eine Eigenschaft zu entfernen, wenn erforderlich.
  4. Klicken Sie Verbindung testen um sicherzustellen, dass die Verbindung zum Abrechnungssystem erstellt werden kann.
  5. Klicken Sie Als Standard auswählen um das Abrechnungssystem als Standard zu definieren.
  6. Klicken Sie Speichern um Änderungen wirksam zu machen.

Um ein Abrechnungssystem zu entfernen:

  1. Erweitern Sie die Details des Abrechnungssystems, der entfernt werden muss.

    Die folgenden Voraussetzungen müssen erfüllt werden damit ein Abrechnungssystem entfernt sein kann:

    • Das Abrechnungssystem muss als Standard markiert sein.
    • Keine technischen Services, vermarktbaren Services und keine aktiven Abonnements benutzten das Abrechnungssystem.
  2. Klicken Sie Entfernen und bestätigen Sie die Entfernung.

    Das Abrechnungssystem wird sofort entfernt.
