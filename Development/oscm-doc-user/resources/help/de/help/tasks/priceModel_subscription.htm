Abo-Preismodell definieren

Auf der Seite Abo-Preismodell definieren können Sie für ein ausgewähltes Abonnement ein spezifisches Preismodell definieren oder ändern. Wenn der abonnierte Service das plattformeigene Abrechnungssystem verwendet, können Sie das Preismodell direkt auf dieser Seite bearbeiten. Verwendet der Service ein externes Abrechnungssystem, können Sie ein neues oder geändertes Preismodell von diesem System importieren.

Abospezifische Preismodelle können Sie für Ihre eigenen Kunden und die Kunden Ihrer Vermittler definieren.

Ein Abo-Preismodell hat Vorrang gegenüber dem ursprünglichen Service-Preismodell oder einem Kunden-Preismodell. Den Abrechnungsmodus eines bestehenden Abo-Preismodells können Sie nicht mehr ändern. Einführende Informationen zu Preismodellen finden Sie hier.

Wenn Sie ein Abo-Preismodell ändern und der zugrundeliegende Service das plattformeigene Abrechnungssystem verwendet, wirken sich Ihre Änderungen auf den aktuellen und zukünftige Abrechnungszeiträume aus. Dies bedeutet beispielsweise, dass alle Benutzerzuordnungen oder Ereignisse, die durch die Nutzung des Service im aktuellen Abrechnungszeitraum bereits vor Ihren Änderungen angefallen sind, schon auf Grundlage des neuen Preismodells in Rechnung gestellt werden.

Üblicherweise ändern Sie ein Abo-Preismodell nur in Absprache mit dem jeweiligen Kunden. Sie können ein Abo-Preismodell jederzeit ändern, unabhängig davon, ob der entsprechende marktfähige Service aktiviert oder deaktiviert ist.

So definieren Sie ein Abo-Preismodell:

  1. Wählen Sie in der Liste im oberen Bereich der Seite das Abonnement, für das Sie ein Abo-Preismodell definieren möchten.

    Sie können jedes laufende Abonnement wählen, das für einen Ihrer Services abgeschlossen wurde. Dazu gehören Abonnements Ihrer direkten Kunden und der Kunden Ihrer Vermittler.

  2. Abhängig von dem Abrechnungssystem, das für den zugrunde liegenden technischen Service definiert ist, fahren Sie wie folgt fort:
    • Wenn das plattformeigene Abrechnungssystem verwendet wird, definieren Sie das Preismodell in den weiteren Bereichen auf der Seite. Detaillierte Informationen dazu finden Sie hier.
    • Wenn ein externes Abrechnungssystem verwendet wird, importieren Sie das Preismodell von diesem System. Detaillierte Informationen dazu finden Sie hier.
Die Änderungen am Abo-Preismodell sind sofort gültig.
