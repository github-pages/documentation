Einmalige Gebühr

Eine einmalige Gebühr legt den Betrag fest, den ein Kunde im ersten Abrechnungszeitraum für ein Abonnement bezahlen muss.

Dieser Betrag wird zu den anderen Nutzungsgebühren addiert. Die einmalige Gebühr ist unabhängig von der Anzahl der Benutzer, Ereignissen oder anderen Nutzungsdaten.

Ist für einen Service, den Sie als Up-/Downgrade-Option definiert haben, eine einmalige Gebühr festgelegt, wird diese bei der Umstellung auf diesen Service berechnet. Dies gilt auch, wenn für den Service, von dem aus umgestellt wird, bereits eine einmalige Gebühr berechnet wurde.

Beispiel



Im Beispiel werden 50,00 € zu den Nutzungsgebühren des ersten Abrechnungszeitraums addiert.
