Texte der Benutzeroberfläche

Die Texte der Benutzeroberfläche werden in einer Microsoft Excel-Datei übersetzt oder aktualisiert. Sie erstellen solch eine Datei mit der Option Exportieren auf der Seite Sprachen verwalten.

Die exportierte Microsoft Excel-Datei enthält drei Arbeitsblätter:

  • User interface: Alle Texte und Meldungen, die auf der Benutzeroberfläche des Administrationsportals und der Marktplätze sichtbar sind.
  • Email: Betreff und Inhalt aller E-Mails, die von der Plattform automatisch generiert und verschickt werden.
  • Platform objects: Namen von Serviceparametern, Ereignissen, Berichten und Zahlungsarten, die von der Plattform bereitgestellt werden. Die Parameter- und Ereignisnamen sind für Serviceanbieter sichtbar, wenn Sie einen marktfähigen Service definieren, sowie für alle Benutzer, wenn sie einen Service abonnieren.

Jedes Arbeitsblatt enthält fünf Spalten:

  • Key: Schlüssel, über den die Plattform den zugehörigen Text eindeutig ausliest. Der Schlüssel darf nicht verändert werden.
  • de system, en system, ja system: Die Standardsprachen der Plattform, die als Sprachpakete nach der Installation vorhanden sind (Deutsch, Englisch, Japanisch). Die Texte in den Standardsprachen können für die Übersetzung als Referenz verwendet werden.
  • Add your language code here oder <Language Code>: In diese Spalte tragen Sie die Texte der neuen Sprache ein, die unterstützt werden soll.

    Add your language code here wird angezeigt, wenn Sie auf der Seite Sprachen verwalten beim Starten des Exports keine Sprache ausgewählt hatten, d.h. Sie wollen in der Regel eine neue Sprache bereitstellen. Die Spalte ist leer.

    Wenn Sie auf der Seite Sprachen verwalten eine Sprache ausgewählt haben, ist der Sprachcode für diese Sprache als Spaltenüberschrift eingetragen. In diesem Fall bearbeiten Sie die Texte der ausgewählten Sprache. Sie können auch eine Standardsprache auswählen (de, en oder jp) , wenn Sie die Standardtexte anpassen möchten.

    Die Texte können auch von Marktplatz-Managern bearbeitet werden. Marktplatz-Manager verwenden dazu die Menüoption Marktplatz --> Texte anpassen. Solche Änderungen wirken sich auf alle Marktplätze aus, von denen die Organisation der Marktplatz-Manager Eigentümer ist.

Gehen Sie wie folgt vor, um die Datei zu übersetzen:

  1. Geben Sie auf allen drei Arbeitsblättern im Feld für die Überschrift der fünften Spalte den ISO-Sprachcode für die Sprache ein, die Sie bereitstellen möchten.
  2. Geben Sie für jeden Schlüssel in der fünften Spalte Ihre Übersetzung ein.

    Wenn eine Übersetzung fehlt, wird der englische Text verwendet. Wenn auch der englische Text nicht definiert ist, wird der sprachunabhängige technische Schlüssel angezeigt.

    Beachten Sie folgendes:

    • Alle beschreibenden Texte (gekennzeichnet durch einen auf .description endenden Schlüssel) können Sie mit HTML-Tags formatieren. Zum Beispiel können Sie mit <b> Text fett darstellen oder <br> für Zeilenumbrüche verwenden.
    • Sie können in den Texten alle Zeichen des Unicode-Zeichensatzes verwenden.
    • Wenn Sie einen Text nicht ändern möchten, können Sie die entsprechenden Tabellenzellen löschen. Dies kann zum Nachvollziehen von Änderungen an der Benutzeroberfläche hilfreich sein.
    • Stellen Sie sicher, dass die Felder keine ungültigen Daten enthalten und somit Fehler im Seitenlayout erzeugen können. Ein zu langer Text wird zum Beispiel an der Benutzeroberfläche auf mehrere Zeilen aufgeteilt. Verwenden Sie zum Testen der Übersetzungen eine separate Umgebung.
  3. Speichern Sie die Microsoft Excel-Datei im Format .xls .
  4. Importieren Sie die Datei auf der Seite Sprachen verwalten.

    Durch den Import der Datei werden vorhandene Texte in der Sprache des in der fünften Spalte eingegebenen Sprachcodes überschrieben. Die Sprache kann verwendet werden, sobald sie aktiviert ist. Für Benutzer, die die Sprache in ihrem Profil eingestellt haben, sind Ihre Änderungen sofort sichtbar.

Hinweis: Vergewissern Sie sich, dass alle Zellen in allen Arbeitsblättern der Excel-Datei als Text formatiert sind.
