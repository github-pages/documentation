Rabatte

Einem Kunden kann ein Preisnachlass gewährt werden, der für alle seine Abonnements von Services gilt, die das plattformeigene Abrechnungssystem verwenden. Der Rabatt kann ab sofort oder ab einem künftigen Monat gelten und auf einen bestimmten Zeitraum begrenzt werden. Vor Ablauf dieses Zeitraums wird der Kunde per E-Mail informiert, so dass er rechtzeitig reagieren und den Serviceanbieter kontaktieren kann.

Ein Rabatt wird als Prozentsatz definiert, der von den Gesamtkosten für ein Abonnement abgezogen wird. Er wird auf alle Kosten innerhalb eines Abrechnungszeitraums gewährt, in dem der Rabatt gilt, unabhängig davon, ob der Rabatt für den gesamten oder nur für einen Teil des Abrechnungszeitraums definiert wurde.

Ein Serviceanbieter kann seinen direkten Kunden Rabatte gewähren. Rabatte werden für Kunden von Vermittlern und Wiederverkäufern nicht unterstützt.

Rabatte wirken sich auf alle vereinbarten Umsatzbeteiligungen aus.
