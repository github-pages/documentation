Preis für Benutzer

Für die Benutzer, die einem Abonnement zugeordnet sind, kann eine Gebühr definiert werden, die in regelmäßigen Abständen bezahlt werden muss. Unterstützt werden die Zeiträume Monat, Woche, Tag und Stunde. Die Gebühr ist abhängig von der Anzahl der Zeiteinheiten, die ein oder mehrere Benutzer einem Abonnement zugeordnet sind. Diese Gebühr kann nur für Services mit Login-Zugriff oder Benutzerzugriff definiert werden.

Der Preis für Benutzer ist unabhängig von einem Preis pro Abo oder anderen Nutzungsdaten.

Der Preis für Benutzer kann gestaffelt sein: Abhängig von der Summe aller Zeiteinheiten, in denen Benutzer einem Abonnement zugeordnet sind, können unterschiedliche Preise definiert werden. Weitere Informationen finden Sie hier.

Wenn die Nutzungsgebühren pro Zeiteinheit berechnet werden, werden die Zeiteinheiten, in denen ein Benutzer einem Abonnement zugeordnet ist, immer vollständig abgerechnet. Dabei wird nicht zwischen Benutzern unterschieden, die während der ganzen Zeiteinheit zugeordnet waren, und solchen, die nur für eine kürzere Zeit zugeordnet waren. Wenn die Zuordnung eines Benutzers zu einem Abonnement innerhalb einer Zeiteinheit zunächst aufgehoben und danach wieder vorgenommen wird, wird die Zeiteinheit nur einmal berechnet. Wenn jedoch eine Zuordnung aufgehoben, der Benutzer gelöscht und danach ein neuer Benutzer mit demselben Benutzernamen zugeordnet wird, werden diese Benutzer als zwei verschiedene, dem Abonnement zugeordnete, Benutzer behandelt. Deshalb wird die Zeiteinheit zweimal berechnet.
