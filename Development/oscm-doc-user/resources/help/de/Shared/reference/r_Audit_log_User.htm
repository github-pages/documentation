Aktionen von Standardbenutzern

Im folgenden werden Meldungen erläutert, die in das Aktionsprotokoll geschrieben werden, wenn Standardbenutzer auf einem Marktplatz Aktionen ausführen.

Folgende weitere <Name/Wert-Paare> werden im Anschluss an den Header und die für alle Meldungen gültigen Name/Wert-Paare ausgegeben:

  • serviceId=<Service-ID>: ID des marktfähigen Service, wie sie beim Anlegen angegeben wurde.

  • serviceName=<Name>: Servicename für Kunden des marktfähigen Service, wie er beim Anlegen angegeben wurde.

  • subscriptionName=<Abo-Name>: Name des Abonnements, wie er beim Abonnieren angegeben wurde.

Abhängig von der protokollierten Aktion können zusätzliche Name/Wert-Paare ausgegeben werden.

Aktion: Execute service operation

Beschreibung: Ein Benutzer führte eine Operation für den abonnierten Service aus. Operationen und ihre Parameter werden von dem technischen Service zur Verfügung stellt, der dem marktfähigen Service zugrunde liegt.

Weitere Name/Wert-Paare:

  • serviceOperation=<Operation>: ID der vom Benutzer ausgeführten Operation.

  • <Parameter-ID>=<Parameterwert>: Liste der Operationsparameter. Die Listeneinträge sind durch einen senkrechten Strich (|) voneinander getrennt. Jeder Parameter-ID ist der entsprechende Wert des Parameters zugeordnet.
