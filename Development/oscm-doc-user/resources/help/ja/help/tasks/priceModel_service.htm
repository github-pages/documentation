サービスの料金設定

サービスの料金設定ページでは、プラットフォームの請求システムを使用するサービスに対して、料金設定の定義と編集を行うことができます。また、外部の請求システムを使用するサービスに対して新しい料金設定、または編集した料金設定をインポートすることができます。

料金設定では、あるサービスを申し込むサービス利用部門が課金されるかどうか、いくら課金されるのかを設定します。料金設定の基本的な情報については、ここをクリックしてください。

すべてのサービスに対して、サービスを有効化しサービス利用部門が購入できるようにするには、サービスの料金設定が定義されている必要があります。サービスに対して定義できる料金設定は1つだけです。ただし、サービス利用部門固有の料金設定または購入済サービス固有の料金設定を定義することもできます。

サービスの料金設定ページでサービス料金設定を更新するには、対象となるサービスを無効化する必要があります。既存の料金設定を更新する場合、変更内容で現在の値が上書されます。更新しても既存の購入済サービスには影響しません。変更は、サービスが再度公開されると新しい購入済サービスに対して有効になります。

サービス仲介部門またはサービス提供代行部門にサービスを再提供設定を付与するのに際して、サービスの料金設定が必要となります。サービス仲介部門およびサービス提供代行部門はサービス提供部門が登録した料金設定を変更できません。ただし、サービス提供代行部門はサービス利用規約のみ独自のものを提供することができます。

再提供設定を既に与えているサービスの料金設定を更新する場合、サービス仲介部門、サービス提供代行部門およびそれらのサービス利用部門に対してはすぐに変更した新しい料金が表示されます。公開されているサービスはすぐに新しい料金で提供されます。購入済サービスには更新による影響はありません。いずれの変更もそれ以降の新しい購入済サービスにのみ反映されます。

注: アクセス種別がエクスターナルアクセスの技術サービスに準拠したサービスポータルへ登録済のサービスでは、料金設定を定義する必要はありません。エスクターナルアクセス型では、アプリケーションがサービスを利用するサービス利用部門への課金方法を管理します。

サービス料金設定を定義するには：

  1. 料金設定を定義する対象サービスをサービスID一覧から選択します。
  2. 基になっている技術サービスに対して定義された請求システムに従って次のようにしてください:
    • プラットフォームの請求システムを使用している場合、 このページ後の部分で料金設定を定義してください。詳細は ここをクリックしてください。
    • 外部の請求システムを使用している場合、当該システムから料金設定をインポートし、サービス利用規約を指定してください。詳細は ここをクリックしてください。

このサービスを再度公開すると、サービス利用部門は料金設定を参照できるようになります。すべての購入済サービスで有効になります。

サービス仲介部門またはサービス提供代行部門およびそれらのサービス利用部門に対してはサービス提供部門が変更した新しい料金が表示されます。
