サービス利用部門向け料金設定の削除

サービス利用部門向け料金設定の削除ページでは、サービス利用部門への提供を中止するサービス利用部門向け料金設定を削除できます。特定のサービス利用部門に対して定義されている料金設定を削除できます。

通常は、当該のサービス利用部門の合意がある場合のみ、サービス利用部門向け料金設定を削除します。

サービス利用部門向け料金設定の削除では、該当するサービスを無効化する必要があります。サービス利用部門向け料金設定を削除しても、既存の購入済サービスには影響しません。新しい購入済サービスに対応するサービス料金設定が自動的に適用されます。

サービス利用部門向け料金設定を削除するには：

  1. サービス利用部門一覧から、削除する料金設定の対象のサービス利用部門を選択します。

    一覧には、サービス利用部門向け料金設定が登録されているすべてのサービス利用部門が含まれます。

  2. サービス一覧から、対応するチェックボックスを選択し、サービス利用部門向け料金設定を削除する対象サービスを選択します。
  3. サービス利用部門向け料金設定を削除するには、削除をクリックし、削除します。
料金設定は削除され、ただちに利用できなくなります。
