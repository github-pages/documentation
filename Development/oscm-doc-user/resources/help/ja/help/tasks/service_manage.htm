サービスの管理

サービスの管理ページでは、ログインしている利用者がサービスポータルに公開する権限を与えられているサービスの詳細を確認または変更できます。

閲覧可能なサービスの情報は以下のとおりです:

  • サービスを公開した際にサービスポータルのサービスカタログに表示されるサービス名と要約。
  • サービスが提供する機能の詳細に関する説明。この内容はサービス利用部門がサービス詳細を確認する際にサービスポータルで表示されます。
  • サービス利用部門がサービスを購入する際に読むべきサービス利用規約。
  • サービス提供部門がサービスに設定した料金設定の詳細。サービスには無料または有料が設定されています。サービス利用部門がサービスをして課金される際に、この基本料金や利用料金が使用されます。詳細な説明はこちらをクリックしてください。

サービス利用規約のみ変更が可能です。ログインしている利用者のサービス利用部門向けのテキストを変更できます。これらを変更するにはサービスが有効化されていない状態でなければなりません。サービス利用部門の既存の購入済サービスにはこの変更は反映されません。いずれの変更もそれ以降に作られた購入済サービスにのみ反映されます。変更後、サービス利用部門がそのサービスを購入するには再度有効化する必要があります。

サービスの詳細を確認または変更するには:

  1. サービスID一覧から詳細を確認または変更するサービスを選択します。

    一覧にはログインしている利用者が提供する権限を与えられているすべてのサービスが含まれます。

  2. サービス利用規約入力項目でサービスのサービス利用規約の内容を変更します。この項目にはサービス提供部門が入力した内容がプリセットされており、必要に応じて変更できます。
    注: 注:テキストをフォーマットするために提供されたエディタを用いて編集してください。HTMLを手作業で編集し、外部のスタイルシートやコンテンツを参照した場合、画面のレイアウトにエラーが生じる可能性があります。
  3. 保存をクリックして、変更した内容を保存します。 テキストはログインしている利用者が現在使用している言語に対して保存されます。
  4. サービス利用規約を別の言語で提供する場合、アカウントメニューのプロフィールの編集ページを使用して言語の設定を変更した上で、翻訳後の内容を入力してください。

変更後のサービスの詳細が画面に表示されます。

サービス利用部門がこのサービスを購入できるようにするには、まずそのサービスをサービスポータルに公開する必要があります。その後、サービスを有効化してください。
