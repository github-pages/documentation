Rechnungsdatenvorschau

Sie können Rechnungsdaten für die Abonnements eines Kunden von Services, die das plattformeigene Abrechnungssystem benutzen, für einen bestimmten Zeitraum vorab anzeigen. Die Daten können in einer XML-Datei gespeichert oder einem Editor Ihrer Wahl angezeigt werden. Sie können die Daten bearbeiten und zum Beispiel an ein Buchhaltungssystem weiterleiten.

Die Rechnungsdatenvorschau sammelt die Daten, die für die Rechnungen eines bestimmten Kunden relevant sind, und kumuliert sie für jeden Tag des angegebenen Zeitraums. Die Daten werden nicht in der Datenbank gespeichert; die Ergebnisse sind lediglich eine Kostenvorschau für den Kunden.

So starten Sie die Rechnungsdatenvorschau:

  1. Wählen Sie in der Liste Organisations-ID die Organisation, deren Rechnungsdaten Sie anzeigen möchten.

    Es werden alle Organisationen angezeigt, die auf Ihrer Plattform verwaltet werden.

  2. Geben Sie das Anfangs- und Enddatum des Zeitraums an, für den Sie Rechnungsdaten anzeigen möchten.

    Es werden die genauen Kosten innerhalb dieses Zeitraums berechnet, unabhängig von den festgelegten Abrechnungszeiträumen. Wenn Sie als Anfangs- und Enddatum den ersten und den letzten Tag eines Abrechnungszeitraums angeben, entsprechen die für die Vorschau errechneten Nutzungsgebühren genau denen für den Abrechnungszeitraum.

    Geben Sie das Anfangs- und Enddatum im Format JJJJ-MM-TT an.

  3. Klicken Sie Exportieren.
  4. Entscheiden Sie, ob Sie die Daten in einer Datei speichern (<datum>BillingData.xml) oder direkt in einem Editor Ihrer Wahl anzeigen lassen.

Eine detaillierte Beschreibung der Elemente und Attribute in der erzeugten XML-Datei finden Sie im Handbuch Developer's Guide.
