請求データのプレビュー

特定の期間におけるサービス利用部門の購入済サービスの請求データをプレビューすることができます。請求データはXMLファイルに保存するか、任意のエディタで直接表示することもできます。データを経理システムに回送するなどの場合に、請求データを編集した上で作業することができます。 指定した期間における、サービス利用部門の購入済サービスに対する請求データを確認することができます。 プラットフォームの 請求システムを使用している購入済サービスだけが対象となります。請求データはXMLファイルとして保存、またはお好みのエディタで開くことができます。データを経理システムに回送する場合など、必要に応じて請求データを編集した上で扱うことができます。

請求データのプレビューは特定のサービス利用部門の請求データに関するデータを収集し、特定の期間に対する計算をおこないます。ここで作成された請求データは特定のサービス利用部門のコスト見積りのみに利用され、データベースには保存されません。

請求データをプレビューするには：

  1. 請求データを表示する組織を組織ID一覧から選択します。

    一覧にはプラットフォームで管理されるすべての組織が含まれます。

  2. 請求データを表示する期間の開始日と終了日を指定します。

    システムは設定されている請求期間とは別に、ここで指定された期間に対して計算をおこないます。開始日と終了日に請求期間と同じ日付を設定した場合、請求データ作成処理と同じ結果が出力されます。

    日付はYYYY-MM-DD形式で指定します。

  3. エクスポートをクリックします。
  4. ファイル（<日付>BillingData.xml）にデータを保存するか、選択したエディタで直接閲覧するかを選択します。

出力されるXMLファイルの項目の詳細な説明や属性については運用ガイド（技術サービス開発部門編）を参照してください。
