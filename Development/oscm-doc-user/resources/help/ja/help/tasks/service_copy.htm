サービスのコピー

サービスのコピーページでは、既存のサービスポータルへ登録済のサービスのコピーを作成することができます。サービスポータルへ登録済のサービスのコピーは、料金の高い上位サービス、またはプロモーションキャンペーンの特別割引など、既存のサービスを若干異なるプロパティで提供する際に便利です。

サービスポータルへ登録済のサービスのコピーとは、以下を意味します。
  • コピーされたサービスとして、同じ技術サービスに準拠した新しいサービスポータルへ登録済のサービスが作成されます。
  • サービス定義が新しいサービスにコピーされます。
  • 料金設定が新しいサービスにコピーされます。

サービスをコピーするには：

  1. コピーするサービスを一覧から選択します。
  2. サービスIDフィールドにサービスコピーの 識別子を入力します。

    サービスIDは40文字以内で以下の文字を含めることはできません。

    !" # $ % & ' * + , / :; < = > ?\ ^ `

    このサービスIDは、サービスを更新または削除するとき、あるいはオプションのアップグレード／ダウングレードや料金設定を指定するときなどに選択できます。

  3. コピーをクリックします。

サービスのコピーが作成されます。必要に応じて、新しいサービスの定義および料金設定の更新を行うことができます。

サービス利用部門が購入できるように新しいサービスを提供するには、そのサービスの公開設定をおこない有効化する必要があります。任意の操作として、サービスのアップグレード／ダウングレードオプション用の定義やサービス仲介部門およびサービス提供代行部門への再提供設定の付与なとがおこなえます。
