購入済サービスの料金設定変更

購入済サービスの料金設定変更ページでは、サービス利用部門の購入済サービスに対して料金設定を定義、編集を行うことができます。

外部の請求システムを使用して料金を請求が行われる購入済サービスを選択しています。請求額を計算するための料金設定は当該請求システムにおいて定義、管理されます。プラットフォームでは、購入済サービスに対する適切な料金設定をインポートする必要があります。プラットフォームでは料金設定の参照のみすることができ、編集することはできません。料金設定を変更するには、該当する情報を外部の請求システムで編集した後に再度インポートする必要があります。

購入済サービスの料金設定の定義、および編集方法:

  1. 料金設定を入力してくださいにおいて、外部のシステムから料金設定をインポートするためにインポートをクリックします。 請求システムアダプターがプラットフォームを外部の請求システムと接続し、正しい料金設定を使用可能な状態である必要があります。表示ボタンをクリックすると料金設定を参照することができます。
  2. 料金設定を保存するには、保存をクリックします。
購入済サービスに対する料金設定の変更はただちに有効になります。
