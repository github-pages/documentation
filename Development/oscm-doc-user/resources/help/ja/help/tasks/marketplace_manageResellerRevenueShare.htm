サービス提供代行部門の売上配分設定

サービス提供代行部門の売上配分設定ページでは、サービス提供代行部門に提供する権利が与えられている個々のサービスに対してサービス提供代行部門向け売上配分率を設定できます。 特定のサービスの売上配分率は公開されているサービスに対してサービスポータルで設定されている売上配分率より優先して使用されます。また、特定のサービス提供代行部門に対する売上配分率も設定でき、これはサービスポータルで設定された値や特定のサービスに対して設定された値よりもさらに優先して使用されます。

特定のサービスポータルに対する売上配分率はサービスポータルの変更画面から設定できます。

また、サービス提供代行部門の売上配分設定ページでは、サービスの再提供設定がすでに削除されているサービスやサービス提供部門が削除したサービスの売上配分率を更新することができます。それらのサービスの購入済サービスが存在している場合、その売上配分率を変更するために利用できます。

注: 売上配分はプラットフォームの請求システムを使用しているサービスに対してだけ計算されます。外部の請求システムを使用しているサービスは売上配分の計算には含まれません。

サービス提供代行部門の売上配分率を設定するには:

  1. 売上配分率を設定するサービスを選択します。

    一覧にはプラットフォームでその時点で使用可能なすべてのサービスおよびすでに削除されたサービスが含まれます。削除されたサービスは名前の後ろに"#"およびUNIX時間が付加され、いつサービスが削除されたかがわかります。すべてのサービスは、括弧内にサービス提供部門の組織IDが付加されており、どのサービス提供部門が提供したサービスかを認識できます。

  2. 最初のテーブルには、サービスが公開されているサービスポータルに対する標準のサービス提供代行部門向け売上配分率と特定のサービスに対する標準のサービス提供代行部門向け売上配分率が表示されます。

    売上配分率 (%)の入力項目では、特定のサービスに対する標準のサービス提供代行部門向け売上配分率を設定できます。: パーセントの値を入力してください。

    このパーセント値はサービス提供代行部門が選択されたサービスを公開している全てのサービスに対して有効です。この値はサービスポータルの変更画面で設定された標準のサービス提供代行部門向け売上配分率よりも優先して使用されます。

  3. 2番目のテーブルには再提供設定が与えられているすべてのサービス提供代行部門および再提供設定がすでに削除されたすべてのサービス提供代行部門が表示されます。

    売上配分率 (%)を入力してください。それぞれのサービス提供代行部門に対してパーセント値を設定できます。

    この値は特定のサービスに対するサービス提供代行部門向け売上配分率およびサービスポータルの変更画面で設定された標準のサービス提供代行部門向け売上配分率よりも優先して使用されます。

  4. 保存をクリックします。

    この値は即時に適用されます。 配分金額は次回の請求書作成処理が開始されたときに計算されます。
