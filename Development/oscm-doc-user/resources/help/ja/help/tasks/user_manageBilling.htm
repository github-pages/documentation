請求期間の設定

請求期間の設定ページでは、請求期間の開始日を設定できます。

請求期間は、サービス利用部門に対して購入済サービスの請求データを作成する対象期間です。請求期間は常に一カ月です。

請求期間はデフォルトでは毎月1日の00:00:00.000に開始し、月の最終日の23:59:59.999に終了します。しかしながら、実際のビジネスでこれが常に適しているとは限りません。このため、請求期間の開始日を自由に設定できます。例えば、請求期間を毎月8日に開始する設定とした場合、請求期間は8日の00:00:00.000から、翌月7日の23:59:59.999までとなります。

請求期間の開始日を変更した場合、それ以降に購入したサービスおよび再度有効化した購入済サービスに対して変更が適用されます。既存の購入済サービスには影響がありません。

請求期間の情報を含んでいるすべてのレポートに、請求期間の開始日時と終了日時が記載されます:

  • サービス提供部門の サービス利用部門の請求書明細 レポート

  • サービス利用部門の 請求書明細 レポート

  • サービス利用部門の 支払いプレビュー レポート

  • サービス提供部門、サービス提供代行部門、サービス仲介部門およびサービスポータル運用部門の 売上配分明細 レポート

請求期間の開始日を設定するには:

  1. 開始日欄で請求期間の開始日を選択します。 値には1から28までが選択可能です。これは、すべての月に適用可能とするためです。
  2. 保存をクリックして変更した内容を保存します。
