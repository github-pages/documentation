ロールごとの料金の定義

ロールごとの料金の定義ページでは、技術サービス定義で指定したサービスロールの料金を指定できます。サービスロールを使用して異なる利用者に特定の権限を付与できます。これらはアプリケーションで対応する許可にマッピングできます。

サービスロールにはそれぞれ料金を定義できます。請求期間の料金計算の際に、利用者ごとの基本料金にこの料金が加算されます。

注: ロールごとの料金はプラットフォームの請求システムを使用して料金請求されるサービスに対して定義した料金設定に対してだけ有効です。外部の請求システムの料金設定はサービスに対する料金がいかに計算されるかを決定する独自の要素を定義しています。

サービスロールに料金を定義するには：

  1. 料金を定義するサービスロールを選択します。
  2. 利用者単価フィールドに、選択したロールの各利用者に対して利用期間ごと料金が発生する場合に、その単価を入力します。
  3. 利用できるすべてのサービスオプションに対して料金を指定します。
  4. 入力内容を確認し、別のサービスロールに料金を定義するには、適用をクリックします。 入力内容を確定し、サービスの料金設定、サービス利用部門向け料金設定または購入済サービスの料金設定変更ページに戻るには、OKをクリックします。
  5. 料金設定を保存するには、サービスの料金設定、サービス利用部門向け料金設定または購入済サービスの料金設定変更ページで保存をクリックします。
