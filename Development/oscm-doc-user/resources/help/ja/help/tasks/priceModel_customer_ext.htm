サービス利用部門向け料金設定

サービス利用部門向け料金設定ページで、サービスとサービス利用部門に対して料金設定の定義と編集を行うことができます。

外部の請求システムを使用して料金を請求が行われるサービスを選択しています。請求額を計算するための料金設定は当該請求システムにおいて定義、管理されます。サービスとサービス利用部門に対する適切な料金設定をプラットフォームにインポートする必要があります。プラットフォーム上では当該料金設定を参照することができますが、編集を行うことはできません。料金設定を変更するには外部の請求システムにおいて該当する情報を変更した後再度インポートする必要があります。

サービス利用部門向け料金設定の定義、および編集方法:

  1. 料金設定を入力してくださいにおいて、 外部のシステムから料金設定をインポートするため、インポートをクリックします。 請求システムアダプターがプラットフォームを外部の請求システムと接続し、正しい料金設定を使用可能な状態である必要があります。表示ボタンをクリックすると料金設定を参照することができます。
  2. サービス利用規約を変更します。サービス利用部門はサービスを購入する際にサービス利用規約を読んで同意する必要があります。

    デフォルトでは、サービス利用規約は技術サービス開発部門が設定したものが使用されます。既にサービスに設定されているものを使用するか独自のものを設定するかは任意です。

    他の言語のサービス利用規約は登録したのち、料金設定のローカライズから設定できます。

  3. 料金設定を保存するには、保存をクリックします。
  4. 別の言語でサービス利用規約を提供するには、 ローカライズをクリックします。
サービスを有効化すると選択したサービス利用部門で料金設定が使用可能になります。これは当該サービス利用部門のすべての購入済サービスに対して有効になります。
