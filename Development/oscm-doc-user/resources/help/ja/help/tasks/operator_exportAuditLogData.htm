監査ログ情報のエクスポート

監査ログのエクスポートページでは、 料金設定も含めて購入済サービスやサービスに関するあらゆる種類のサービス利用者の操作をエクスポートすることができます。エクスポートしたデータは監査ログとなります。エクスポートしたこれらのデータは、例えば、いつ、誰が料金設定を変更したか確認したり、購入済サービスがいつ作成されたかを確認するのに有用です。

注:

AUDIT_LOG_ENABLEDシステム設定がtrueに設定されている場合にだけ、システムは利用者の操作を記録します。

監査ログをエクスポートするには:

  1. エクスポートするログ情報の利用者のロールを利用者のロール一覧から選択します。すべてのログをエクスポートするには、すべてを選択します。

    指定された利用者のロールに応じて操作の一覧が表示されます。

    管理者および一般の利用者による購入済サービスに関連した操作の情報ををエクスポートできます。また、サービス提供部門のサービス管理者によるサービスと料金設定に関連した操作の情報もエクスポートできます。管理者の操作には、サービス購入管理者による操作も含みます。サービス管理者の操作には、サービス提供代行部門およびサービス仲介部門による操作も含みます。

  2. 表操作では、エクスポートするログ情報の操作を選択します。テーブルヘッダーのチェックボックスをクリックすると、すべての操作を選択できます。
  3. ログ情報をエクスポートしたい期間の開始日と終了日を指定します。 日付はYYYY-MM-DDの形式で指定します。
  4. エクスポートをクリックします。

エクスポートされたデータを表示するか、ファイルに保存するかを選択できます。詳細についてはここをクリックしてください。
