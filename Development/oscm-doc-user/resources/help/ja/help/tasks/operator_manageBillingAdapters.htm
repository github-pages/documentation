請求システムの管理

外部の請求システムとの連携をサポートしています。技術サービス開発部門はデフォルトの請求システム、または外部の請求システムと、アプリケーションを統合して技術サービスを提供することができます。

請求システムの管理ページで外部の請求システムの設定を行います。正しく関連付けを行うには次の条件を満たす必要があります:

  • システム運用管理部門によって指定される接続プロパティにより、請求システムが参照可能である。

登録された請求システムの一つがデフォルトに設定されている。デフォルトの請求システムは、技術サービス開発部門が技術サービスを登録できるように、事前に定義されている必要があります。

請求システムの管理ページにアクセスすると、デフォルトに設定されている請求システムが展開されて表示されます。このページでは次の操作を行うことができます:

  • 登録された請求システムの閲覧
  • 請求システムの追加
  • 請求システムの簡単な説明、および接続プロパティの更新
  • デフォルトの請求システムの設定
  • 請求システムの削除

新しく請求システムを定義する:

  1. 追加をクリックします。 一覧に空の項目が追加され、詳細を定義することができるようになります。
  2. 請求システムIDを入力します。 このIDはプラットフォーム全体で一意である必要があります。請求システムの定義後にIDを変更することはできません。

    IDは40文字以内で次の文字を利用することはできません:

    ! " # $ % & ' * + , / : ; < = > ? \ ^ `

  3. 簡単な説明を入力します。

    簡単な説明は40文字以内で、すべての文字を利用することができます。

  4. 接続プロパティを入力します。 請求システムに依って、接続プロパティの項目数や設定の意味が異なりますが、外部の請求システムのポート番号やURLなどを指定することによって特定の請求システムを参照可能である必要があります。請求システムとの接続はJNDIを用いて行われます。

    1つめの入力欄にプロパティの名称、2つめの入力欄に設定値を入力します。

    プロパティを追加するには＋ボタンをクリックします。プロパティを削除するには－ボタンをクリックします。

  5. 請求システムとの接続確認を行うには接続テストをクリックします。 接続の確認結果がメッセージ出力されます。
  6. この請求システムをデフォルトに設定する場合、デフォルトに設定をクリックします。 請求システムはただちにデフォルトとして表示されます。デフォルト表示が以前のデフォルトから移動されます。
  7. すべての入力が完了したら保存をクリックします。

技術サービス開発部門が技術サービスを登録するときに、この請求システムを選択することができるようになりました。

既存の請求システムを更新するには:

  1. 更新したい請求システムをクリックして展開します。
  2. 簡単な説明を変更します。
  3. 接続プロパティを変更します。プロパティを追加するには＋ボタンをクリックします。プロパティを削除するには－ボタンをクリックします。
  4. 請求システムとの接続確認を行うには接続テストをクリックします。
  5. デフォルトの請求システムに設定するには、デフォルトに設定をクリックします。
  6. 変更内容を保存するには保存をクリックします。

請求システムを削除するには:

  1. 削除する請求システムをクリックして展開します。

    削除する請求システムは次の条件を満たしている必要があります:

    • デフォルトに設定されていない。
    • 技術サービス、サービス、および有効な購入済サービスに使用されていない。
  2. 削除をクリックします。

    請求システムはただちに削除されます。
