レポートの作成

レポートの作成ページでは、 各種レポートを作成できます。

以下のようなレポートがあらかじめ定義されています。なお、選択できるレポートは所属組織のロールによって異なります。

  • 技術サービス開発部門向けレポートは、アプリケーションの高い負荷を検出するなど、技術的な内容を中心としています。アプリケーションや購入済サービスに関する情報を利用すると、売れ行きの悪い技術サービスを特定できます。サービス利用部門レポートに加えて、技術サービス開発部門は以下のレポートを利用できます。:
    • 技術サービス使用情報レポート: サービス利用中に収集されたイベントの種別と回数も含めて、技術サービス開発部門のすべての技術サービスを提示します。
    • サービス提供部門情報レポート: 技術サービスに対して技術サービス開発部門が登録しているサービス提供部門を示し、サービス提供部門が技術サービスをベースに定義したサービスポータルへ登録済のサービスを一覧表示します。
    • 購入済サービス情報レポート: 技術サービス開発部門の技術サービスをベースに作成されたサービスポータルへ登録済のサービスのサービス提供部門、およびサービスポータルへ登録済のサービスの購入済サービス数を示します。
    • インスタンス情報レポート: 技術サービス開発部門の技術サービスのすべてのアプリケーションインスタンスとそれらの設定内容（パラメーターや値）を示します。
  • サービス提供部門向けレポートは、例えば売れ残ったサービスを特定できるよう、ビジネス関連データに焦点を当てています。また、レポートは収集された請求データをベースに生成することができます。サービス利用部門レポートに加えて、サービス提供部門は以下のレポートを利用できます。:
    • サービス情報レポート: サービスポータルへ登録されているサービス提供部門のすべてのサービスと、その購入済サービスを示します。サービス仲介部門によって提供されたサービスの購入済サービスについても一覧表示されます。
    • サービス利用部門情報レポート: サービス提供部門およびそのサービス仲介部門のすべてのサービス利用部門を、サービス利用部門の基本データとその契約サービスと共に示します。レポートは購入済サービスが有効なのか、終了しているのかを出力します。
    • 請求書情報レポート: サービス提供部門およびそのサービス仲介部門の各サービス利用部門すべての請求データの概要を示します。請求データには、各購入済サービスの請求書キー(Billing data key)が含まれます。
    • 支払処理ステータスレポート: サービス提供部門およびそのサービス仲介部門のすべてのサービス利用部門に対する請求データ計算の状況、購入済サービス、および購入元のサービスを示します。
    • サービス利用部門の請求書明細レポート: 選択されたサービス利用部門の購入済サービスに対する現在の請求期間の請求データを示します。従量課金の計算では、サービスの使用時間に対する料金はミリ秒単位で計算されます。定額料金では、請求期間終了時の単位時間に対する購入済サービスの料金がレポートに記載されます。請求書レポートに含まれる請求書キー(Billing data key)を指定して購入済サービスを選択します。請求書には、サービス利用部門の購入済サービスの請求書キーが印刷されます。
    • 売上配分明細レポート: サービス提供部門が提供したサービスの使用に関する売上配分用請求データを示します。 それぞれのサービスポータルに対して、売上配分用請求データはカレンダーの一月前の期間に購入済サービスに対して課された料金から計算されます。 売上配分用請求データは組織単位および個々のサービス単位での収入を提示します。 また、サービス提供部門がその他の組織に支払うべき金額も計算されます。 このデータは他の組織との契約形態に応じて売上配分の収集または支払の情報として使用できます。 サービス提供部門がサービス利用部門に適用する割引は、システム運用管理部門およびサービスポータル運用部門の売上配分率から差し引かれます。 サービス提供部門は作成するレポートの対象請求期間を指定します。 月はMMのフォーマットで入力し、年はYYYYのフォーマットで入力してください。
  • サービス仲介部門向けレポートはサービス仲介部門の売上配分収入に焦点を当てています。サービス仲介部門は以下のレポートを利用できます。:
    • サービス仲介部門の売上配分明細レポート: サービス仲介部門が提供したサービスの使用に関する売上配分情報を示します。それぞれのサービスポータルに対して、売上配分用請求データはカレンダーの過去の期間に購入済サービスに対して課された料金から計算されます。売上配分用請求データはサービスを供給しているサービス提供部門単位での収入を提示します。このデータは契約形態に応じてサービスポータル運用者およびサービス提供部門に対する売上配分の収集の情報として使用できます。利用者は作成するレポートの対象請求期間を指定します。月はMMのフォーマットで入力し、年はYYYYのフォーマットで入力してください。
  • サービス提供代行部門向けレポートはサービス提供代行部門の売上配分収入に焦点を当てています。サービス仲介部門は以下のレポートを利用できます。:
    • サービス提供代行部門の売上配分明細レポート: サービス提供代行部門が提供したサービスの使用に関する売上配分情報を示します。それぞれのサービスポータルに対して、売上配分用請求データはカレンダーの過去の期間に購入済サービスに対して課された料金から計算されます。売上配分用請求データはサービスを供給しているサービス提供部門単位での収入を提示します。このデータは他の組織との契約形態に応じて売上配分の収集または支払の情報として使用できます。利用者は作成するレポートの対象請求期間を指定します。月はMMのフォーマットで入力し、年はYYYYのフォーマットで入力してください。
  • システム運用管理者向けレポートはプラットフォームのサービスポータルにサービスを公開しているサービス提供関連の部門に焦点を当てています。システム運用管理者は以下のレポートを利用できます。:
    • サービス提供部門収入情報レポート: プラットフォームに登録されているすべてのサービス提供部門およびサービス提供代行部門の収入情報を示します。作成するレポートの期間を指定します。日付はYYYY-MM-DDのフォーマットで入力してください。

      レポートには、名前、ID、サービス提供部門およびサービス提供代行部門ごとの収益、通貨当たりの累積値が示され、すべての通貨に対する収益の合計が表示されます。サービス提供部門の売上配分には再提供設定を付与したすべてのサービス仲介部門による収入も含まれます。

    • 外部サービス情報レポート: 該当するプラットフォームでサービスポータルへエスクターナルアクセス型サービスを公開しているサービス提供関連の部門(サービス提供部門、サービス仲介部門、サービス提供代行部門)を示します。

      各サービス提供部門に対して、レポートは該当するサービスポータルへ登録済のサービスとその公開日・非公開日を示します。サービスが複数回にわたって公開・非公開となる場合、すべてのタイムスタンプが一覧表示されます。

    • サービス提供部門の売上配分明細レポート: プラットフォームに登録されているすべてのサービス提供部門の売上配分情報を示します。 それぞれのサービスポータルに対して、売上配分用請求データはカレンダーの過去の期間に購入済サービスに対して課された料金から計算されます。 それぞれのサービス提供部門について、組織単位および個々のサービス単位での収入を提示します。 また、サービス提供部門がその他の組織に支払うべき金額も計算されます。 サービス提供部門がサービス利用部門に適用する割引は、売上配分率から差し引かれます。 利用者は作成するレポートの対象請求期間を指定します。 月はMMのフォーマットで入力し、年はYYYYのフォーマットで入力してください。
    • サービス仲介部門/サービス提供代行部門の売上配分明細レポート: プラットフォームに登録されているすべてのサービス仲介部門およびサービス提供代行部門の売上配分情報を示します。それぞれのサービスポータルに対して、売上配分用請求データはカレンダーの過去の期間に購入済サービスに対して課された料金から計算されます。それぞれのサービス仲介部門およびサービス提供代行部門について、売上配分用請求データはサービスを供給しているサービス提供部門単位での収入を提示します。このデータはクレームが発生した場合の確認などに利用できます。利用者は作成するレポートの対象請求期間を指定します。月はMMのフォーマットで入力し、年はYYYYのフォーマットで入力してください。
    • サービス情報 (サービス提供部門)レポート: サービスポータルへ登録されているサービス提供部門のすべてのサービスと、その購入済サービスを示します。サービス仲介部門によって提供されたサービスの購入済サービスについても一覧表示されます。サービス提供部門の組織IDを指定するすることでサービス提供部門を選択します。
    • サービス利用部門情報 (サービス提供部門)レポート: サービス提供部門およびそのサービス仲介部門のすべてのサービス利用部門を、サービス利用部門の組織IDとその契約サービスと共に示します。レポートは購入済サービスが有効なのか、終了しているのかを出力します。サービス提供部門の組織IDを指定するすることでサービス提供部門を選択します。
    • 請求書情報 (サービス提供部門)レポート: サービス提供部門およびそのサービス仲介部門の各サービス利用部門すべての請求データの概要を示します。サービス提供部門の組織IDを指定するすることでサービス提供部門を選択します。請求データには、各購入済サービスの請求書キー(Billing data key)が含まれます。
    • サービス利用部門の請求書明細 (サービス提供部門)レポート: 選択されたサービス提供部門についてサービス利用部門の購入済サービスに対する現在の請求期間の請求データを示します。従量課金の計算では、サービスの使用時間に対する料金はミリ秒単位で計算されます。定額料金では、請求期間終了時の単位時間に対する購入済サービスの料金がレポートに記載されます。請求書レポートに含まれる請求書キー(Billing data key)を指定して購入済サービスを選択します。各サービス利用部門の購入済サービスの請求書キーは、請求書情報 (サービス提供部門)レポートに出力されます。

レポートを作成するには：

リストから作成する対象レポートを選択します。レポートの種別によって、追加のレポートパラメーターを入力する必要があります。
下部のページ領域にレポートの内容がただちに表示されます。レポートが表示されているウィンドウのアイコンから印刷または保存ができます。保存する際は複数のフォーマットを選択できます。
