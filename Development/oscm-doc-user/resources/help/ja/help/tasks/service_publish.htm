サービスの公開設定

サービスの公開設定ページでは、以下の作業がおこなえます。:

  • サービスを公開するサービスポータルを設定または変更できます。サービスを有効化するにはサービスポータルが設定されている必要があります。

    サービスポータルを変更するには有効化されていない状態でなければなりません。サービス利用部門の既存の購入済サービスにはこの変更は反映されません。

  • サービス仲介部門またはサービス提供代行部門に対するサービスの再提供設定を付与または除去できます。サービス提供部門のみこの操作をおこなえます。

    サービス仲介部門またはサービス提供代行部門に再提供設定を与えるには、料金設定およびサービスポータルが設定されている必要があります。

    サービス仲介部門またはサービス提供代行部門にサービスの再提供設定を付与した場合、サービス仲介部門またはサービス提供代行部門はサービスポータルにそのサービスを公開できます。サービス仲介部門またはサービス提供代行部門からサービスの再提供設定を除去した場合、サービス仲介部門またはサービス提供代行部門によって公開されているそのサービスはすぐにサービスポータルから撤去され、それ以降サービス利用部門はそのサービスを購入することはできません。一方、サービス仲介部門またはサービス提供代行部門からサービス利用部門が購入した既存のサービスは継続して使用できます。

サービスのサービスポータルを設定または変更するには:

  1. サービスID一覧からサービスポータルを設定または変更するサービスを選択します。

    ログインしている利用者がサービス提供部門の利用者の場合、一覧にはそのサービス提供部門が登録したすべてのサービスが含まれます。プラットフォームの請求システムを使用しているサービスに対しては、システム運用管理部門との間で合意されたシステム運用管理部門の売上配分率が表示されます。この値はサービス提供部門、サービス仲介部門またはサービス提供代行部門がそのサービスを提供した際に使用されます。

    ログインしている利用者がサービス仲介部門またはサービス提供代行部門の利用者の場合、一覧には組織が再提供設定を与えられたすべてのサービスが含まれます。

  2. サービスポータルのオプションを設定してくださいセクションでサービスポータルの情報を設定します:
    • サービスを一般に公開するかどうかを設定します。デフォルトで一般サービスチェックボックスはチェックされています。

      一般サービスはサービスポータルにアクセスしたすべての利用者に表示されます。一般として設定されていないサービスはサービス利用部門のみが購入できます。

    • サービスポータルドロップダウンリストから、サービスを公開するサービスポータルを選択します。サービスポータル運用部門によってアクセス権が与えられているサービスポータルにサービスを公開することができます。

      サービス仲介部門またはサービス提供代行部門でログインしている場合、組織に適用される売上配分が表示されます。サービス提供部門がとしてログインしている場合、サービスポータルを選択するとサービスポータルで定義されている売上配分設定(システム運用管理部門向け、サービスポータル運用部門向け、サービスポータル仲介部門向け、サービスポータル提供代行部門向け)が表示されます。売上配分設定は外部の請求システムを使用しているサービスでは有効になりません。

      選択したサービスポータルにカテゴリが設定されている場合、サービスにカテゴリを割り当てることができます。 選択対象であるカテゴリはサービスポータル運用部門によって作成されます。 カテゴリを使用することで、サービス利用部門はサービスカタログを閲覧しサービスポータル上でサービスのフィルタリングをおこなうことができます。 サービスポータル運用者は、サービスをカテゴリ毎にグルーピングしてサービスポータルのホームページに表示するように設定することもできます。

  3. 保存をクリックして変更内容を保存します。

変更した内容はすぐに適用され、そのサービスは有効化することが可能となります。また、有効化することによってサービス利用部門がサービスを購入できるようになります。

サービスの再提供設定を付与または除去するには (サービス提供部門のみ):

  1. サービスID一覧からサービス仲介部門またはサービス提供代行部門に再提供設定を付与するサービスを選択します。

    一覧にはログインしている利用者の所属擦る組織が登録したすべてのサービスが含まれます。

  2. サービスポータル一覧から、システム運用管理部門、サービス運用管理部門、サービス提供代行部門およびサービス仲介部門との間で合意がとれている適切な売上配分を設定済みのサービスポータルを選択します。この時、選択したサービスポータルのデフォルトの売上配分がプラットフォームの請求システムを使用するサービスに対して表示されます。

    これらのデフォルトの値は、サービス固有または組織固有の売上配分が設定されていない場合に、再提供設定を付与したサービス仲介部門およびサービス提供代行部門に、そのサービス仲介部門またはサービス提供代行部門がサービスを公開する・しないに関わらず適用されます。

    デフォルトの売上配分の下に、プラットフォームに存在するすべてのサービス仲介部門およびサービス提供代行部門が売上配分設定とともに表示されます。

  3. サービス提供代行部門およびサービス仲介部門の選択欄で、サービスの再提供設定を付与または除去したいサービス仲介部門またはサービス提供代行部門の1列目のチェックボックスを選択または除去します。
  4. 保存をクリックして変更内容を保存します。

再提供設定はただちに付与または除去されます。
