カスタム属性の管理

カスタム属性の管理ページでは、サービス利用部門および購入済サービスに対してカスタム属性を追加、編集および削除することができます。

カスタム属性は費用の管理や会計作業など特定用途で必要なサービス利用部門のデータを保存するために使用します。例えば、特定のサービス利用部門の収益の多くを占めるプロフィットセンターを指定したいとします。このようなデータの基準はないため、プロフィットセンターの保存のためにカスタム属性を定義することができます。サービス利用部門または購入済サービスにカスタム属性を定義することができます。特定のサービス利用部門または特定の購入済サービスにカスタム属性を指定すると毎月生成される請求データにも含まれます。この情報は、外部会計、請求またはCRMシステムの参照などに使用することができます。

カスタム属性をサービス利用部門や購入済サービスに設定する際に、サービス利用部門または購入済サービスに自分で属性の値を設定するのか、またはサービスを購入する際にサービス利用部門が属性の値を設定するようにするかを決めることができます。

すべてのカスタム属性に対して、デフォルト値を入力できます。個別のサービス利用部門または個別の購入済サービスに特定の値が設定されていない場合にこの値が使用されます。カスタム属性の値をサービス利用部門が入力できる設定の場合、サービスを購入する手続きにおいて既定値が該当する入力項目にプリセットされます。サービス利用部門の管理者またはサービス購入管理者がこの値を変更した場合、新しい値が自動的にサービス利用部門の購入済サービスに反映されます。

個別のサービス利用部門にカスタム属性を指定するには、サービス利用部門メニューのサービス利用部門の登録メニューオプションを使用します。個別の購入済サービスにカスタム属性を指定するためには、サービス利用部門メニューのカスタム属性の管理メニューオプションを使用します。

サービスの再提供設定が与えられたサービス仲介部門のサービス利用部門はカスタム属性を使用することができます。サービス提供代行部門のサービス利用部門は使用できません。

カスタム属性を追加するには：

  1. すべてのサービス利用部門に共通のカスタム属性を追加するには、サービス利用部門のカスタム属性の定義セクションで追加をクリックします。すべての購入済サービスに共通のカスタム属性を追加するには、購入済サービスのカスタム属性の定義セクションで追加をクリックします。
  2. カスタム属性の追加ページで必要な情報を入力します。詳細な説明はこちらをクリックしてください。
  3. カスタム属性の追加ページで保存をクリックして入力内容を保存します。 カスタム属性の管理ページに戻り、新しい属性が画面に表示されます。
  4. さらにカスタム属性を追加するには、追加をクリックします。追加しない場合は、そのままページを離れます。

カスタム属性を編集するには:

  1. 編集する属性の編集欄のアイコンをクリックします。
  2. カスタム属性の編集ページで、入力内容を編集します。詳細な説明はこちらをクリックしてください。
  3. カスタム属性の編集ページで保存をクリックして入力内容を保存します。

    カスタム属性の管理が表示され、更新されたカスタム属性の内容が画面に表示されます。

カスタム属性を削除するには:

  1. 削除する属性の編集欄のアイコンをクリックします。
  2. カスタム属性の編集ページで、削除をクリックします。
  3. 削除されたことを確認します。

    カスタム属性は即座に削除され、カスタム属性の管理ページが表示されます。
