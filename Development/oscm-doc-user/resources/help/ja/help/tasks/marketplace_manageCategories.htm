カテゴリの管理

カテゴリの管理ページでは、サービスポータルのカテゴリを作成、更新、削除できます。

カテゴリを使用すると、サービスポータル上のサービスを整理・分類することができます。 例えば、ワープロソフト、DTPソフト、表計算ソフトなど異なる種類のアプリケーションをグループ化できます。

サービスポータル運用部門は、サービスポータルにいくつでもカテゴリを作成できます。サービス提供部門、サービス仲介部門およびサービス提供代行部門は、サービスポータルに公開するサービスにこれらのカテゴリを割り当てることができます。サービスポータルでカテゴリの表示が有効になっている場合、サービスポータルの利用者はカテゴリを使用してサービスカタログを参照したり、サービスをフィルタリングすることができます。カテゴリはまたサービスポータルの全文検索機能にも使用されます。

サービスポータル運用部門は、サービスポータルのホームページで表示されるサービスをグルーピングするのにカテゴリを使うかどうかを選択することができます。 詳細についてはここをクリックしてください。

カテゴリ名にはサポートされているすべての言語を指定できます。各言語ごとに言語に独立したIDとその値を設定します。他の言語で名前をローカライズしていない場合、デフォルトでは英語でユーザーインターフェースに表示されます。英語による指定もない場合、言語に依存しないカテゴリ名が使用されます。

カテゴリを作成するには：

  1. カテゴリを作成するサービスポータルをサービスポータル一覧から選択します。 一覧には、サービスポータル運用者がアクセスできるすべてのサービスポータルが含まれます。
  2. カテゴリを提供する言語を選択します。

    一覧にはサポートされているすべての言語が含まれます。後から更新することによって、異なる言語のカテゴリも追加できます。

  3. [追加]をクリックして新しいカテゴリを設定します。
  4. [カテゴリ名]欄に言語依存しないカテゴリ名を入力します。

    カテゴリ名は40文字以内、サービスポータルに定義されるカテゴリ内で一意でなければなりません。

  5. 選択された言語で表示するためのカテゴリの名称を入力します。

    カテゴリの名称は100文字以内で指定します。また、その選択言語内で一意でなければなりません。

  6. カテゴリを追加するには、[追加]をクリックします。追加しない場合は、[保存]をクリックします。

カテゴリを更新するには：

カテゴリの更新をする場合には、サポートされるすべての言語でカテゴリに表示されるカテゴリ名およびその値を変更することができます。

  1. カテゴリを更新するサービスポータルをサービスポータル一覧から選択します。

    一覧には、サービスポータル運用者がアクセスできるすべてのサービスポータルが含まれます。

  2. 言語依存しないカテゴリ名を更新するには、[カテゴリ名]欄に入力します。

    カテゴリ名は40文字以内、サービスポータルに定義されるカテゴリ内で一意でなければなりません。

  3. カテゴリをローカライズするか、ローカライズされた名称を更新するには、必要な言語を選択して、対応する欄にローカライズされた名称を入力します。

    カテゴリの名称は100文字以内で指定します。また、その選択言語内で一意でなければなりません。

  5. さらにカテゴリを更新する場合は手順2または3を繰り返します。
  6. 保存をクリックして変更を保存します。

カテゴリはただちに更新されます。サービスポータルの訪問者には、次回サービスポータルにログインしたときに変更が反映されます。

この更新によってサービス提供部門、サービス仲介部門またはサービス提供代行部門のサービスへのカテゴリの割り当てが影響を受けることはありません。通常、カテゴリの変更の通知は変更者からサービス提供部門、サービス仲介部門およびサービス提供代行部門におこないます。通知は自動的には行われません。

カテゴリを削除するには：

  1. カテゴリを削除するサービスポータルをサービスポータル一覧から選択します。

    一覧には、サービスポータル運用者がアクセスできるすべてのサービスポータルが含まれます。

  2. 削除するカテゴリの削除欄のチェックボックスを選択します。カテゴリは1つまたは複数選択できます。
    注: サービスをカテゴリ毎にグルーピングしてサービスポータルのホームページに表示することを選んだ場合、そのサービスポータルの最初の３つのカテゴリは削除しないでください。
  3. 保存をクリックします。削除するカテゴリを割り当てられているサービスがサービスポータルに公開されている場合、警告が表示されます。
  4. 削除されたことを確認します。

カテゴリはただちに削除されます。 サービスをフィルタリングするのにそのカテゴリを使っていたサービスポータルの訪問者には、空のサービス一覧が表示されます。 サービスポータルに次回ログインしたときには、そのカテゴリーをフィルターとして選択することはもうできなくなります。

サービス提供部門、サービス仲介部門およびサービス提供代行部門がカテゴリをサービスに割り当てていた場合、そのカテゴリの割り当ては削除されます。 サービス提供提供部門、サービス仲介部門およびサービス提供代行部門には、カテゴリが削除されたことがメールで通知されます。
