新しいパスワードの設定

新しいパスワードの設定ページでは、新しいパスワードを入力し、現在のパスワードを変更することができます。

新しいパスワードを設定するには:

  1. 新しいパスワードを繰り返し入力します。

    パスワードは6文字から100文字の間で設定できます。

    パスワードを入力すると、そのパスワードの安全性がパスワードの強度フィールドに示されます。パスワードの強度が高く安全な場合、表示色は緑色になります。パスワードの安全性と強度を確保するには、最低8文字で数字、文字、特殊文字を含める必要があります。

  2. 変更をクリックし、新しいパスワードを保存します。

新しいパスワードが保存されます。次回ログイン時にはそのパスワードを使用します。

画面がログインに切り替わり、利用者にパスワードが変更されたことを確認するEメールが送信されます。
