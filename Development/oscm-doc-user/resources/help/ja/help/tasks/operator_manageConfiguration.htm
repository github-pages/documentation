システム設定の変更

システム設定の変更ページでは、データベースに格納されている現時点のシステム設定の一覧を参照できます。サーバ名、ポート、パス、利用者IDなどについてはインストール時に環境に合わせて設定されています。必要に応じて現在の設定を更新および編集できます。

このシステム設定は、システムのインストール時にoscm-install-pack.zipファイルの抽出先ディレクトリの以下のサブディレクトリにあるconfigsettings.propertiesファイルでも提供されます：

<install_pack_dir>/databases/bes_db
