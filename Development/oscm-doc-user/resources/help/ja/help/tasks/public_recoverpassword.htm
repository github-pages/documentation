パスワードの復元

パスワードの復元ページでは、利用者IDに新しいパスワードを設定するための処理を開始できます。

新しいパスワードを設定するには:

  1. 利用者IDを入力します。

    利用者IDを忘れた場合には、管理者に連絡してください。管理者は所属組織に登録されているすべての利用者のIDを検索することができます。

  2. 画像認証に表示されている文字列を入力します。文字列は大文字と小文字を区別します。
  3. 続行をクリックし、新しいパスワードを設定する処理を開始します。

利用者に新しいパスワードを入力するためのURLが記載されたメールが送信されます。メールに記載された新たなパスワードを定義するためのURLをクリックし、パスワードの復元を完了します。

注: もしこのEメールが送られてこない場合、誤った利用者IDが入力されたことが考えられます。誤った利用者IDが入力された場合でも、エラーメッセージは送信されません。利用者IDを確認し、正しい利用者IDを使用して再度試してください。
