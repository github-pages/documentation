購入済サービスの料金設定変更

購入済サービスの料金設定変更ページでは、選択したサービス利用部門の特定の購入済サービスに料金設定を定義または更新できます。 プラットフォームの請求システムを使用する購入済サービスは料金設定をこのページで編集することができます。また、外部の請求システムを使用するサービスに対して新しい料金設定、または編集した料金設定をインポートすることができます。

購入済サービス料金設定はそのサービス提供部門から直接もしくはそのサービス仲介部門からそのサービスを購入しているサービス利用部門に対して設定できます。

定義または更新する購入済サービス料金設定が、定義されている料金設定またはサービス利用部門向け料金設定に優先されます。ただし、サービスの無料または有料の切替えはできません。料金設定の基本的な情報については、ここをクリックしてください。

プラットフォームの請求システムを使用する購入済サービスの料金設定を更新する場合、現在および将来の請求期間に対して変更された設定が有効になります。つまり、変更前の請求期間に利用したサービスで発生したすべてのサービス利用者の割り当てまたは請求対象イベントは、新しい料金設定に従って課金されることになります。

通常は、当該のサービス利用部門の合意がある場合のみ購入済サービスの料金設定を変更します。購入済サービスの料金設定を変更は該当するサービスが有効化されていても、されていなくてもおこなえます。

サービス利用部門向け料金設定を定義するには：

  1. ページ上部の領域の一覧から、料金設定を定義する購入済サービスを選択します。

    任意の有効化されている購入済サービスを選択することができます。 直接サービスを利用している部門、および、サービス仲介部門を介してサービスを利用している部門の両方が一覧に表示されます。

  2. 基になっている技術サービスに対して定義された請求システムに従って次のようにしてください:
    • プラットフォームの請求システムを使用している場合、 このページ後の部分で料金設定を定義してください。詳細は ここをクリックしてください。
    • 外部の請求システムを使用している場合、当該システムから料金設定をインポートしてください。詳細は ここをクリックしてください。
購入済サービスに対する料金設定の変更はただちに有効となります。
