サービス利用部門の設定

サービス利用部門の設定ページでは、サービス利用部門に対して現時点で定義されている組織データを参照することができます。カスタム属性のような属性値が定義されている場合、値を入力することによってサービス利用部門に割引を提供することができます。

サービス利用部門を管理するには：

  1. 編集するデータの対象のサービス利用部門を組織ID一覧から選択します。

    一覧には現在のサービス提供部門が提供したサービスを購入しているか、もしくはサービスの購入有無に関係なく現在のサービス提供部門によって登録されたすべてのサービス利用部門を表示します。一覧ではサービス利用部門の組織IDが組織名に追加されて表示されます。

  2. カスタム属性がサービス利用部門またはサービス仲介部門のサービス利用部門に定義されており、値を入力する権限がある場合、属性テーブルに必要な値を設定します。

    255文字まで指定できます。次回請求データ生成時にサービス利用部門への請求データにこの値が含まれます。属性値が設定されていない場合、属性名のみが請求データに挿入されます。

    属性テーブルにはサービス利用部門の管理者またはサービス購入管理者が購入済サービスに対して入力した値も表示されます。これらの値は変更することはできません。

  3. サービス利用部門に割引を提供するには、割引の情報のセクションで必要な情報を入力します。

    特定のサービス利用部門に対して、 プラットフォームの 請求システムを使用している購入済サービスすべてに適用される割引を設定できます。現在あるいは将来の月に対して割引を設定することも可能です。一定の期間に制限することもできます。期間が終了する前になると、サービス利用部門がサービス提供部門に連絡するなどの対応ができるように、通知メールが配信されます。

    割引は、購入済サービスの定価から差し引くパーセントとして定義されます。割引が有効な請求期間内に発生したサービス利用部門の費用すべてに対して適用されます。請求期間のすべての期間で割引が有効なのか一部の期間で割引が有効なのかは関係ありません。 割引を変更すると、次回請求データ生成時に新しい割引が有効になります。通常は、当該サービス利用部門の合意がある場合のみ、割引を変更します。

    割引情報は以下のように入力します。

    • 割引率フィールドで、サービス利用部門の購入済サービスの定価から何パーセント割り引くかを入力します。
    • 割引を現在の月に適用する場合、開始月フィールドは空白のままにします。それ以外の場合は、割引を開始する年月をMM/YYYY形式で入力します（例：11/2010）。過去の月を入力すると、これは無視されます。現在の請求期間、または指定した月から割引の適用が始まります。
    • 割引を特定数の月のみに適用する場合、割引を終了する年月を終了月フィールドにMM/YYYY形式で入力します（例：11/2010）。このフィールドを空白にした場合、このフィールドの値を変更するまで割引が適用されます。

    例えば、割引を今から1年間にわたって適用したい場合、開始月フィールドを空白、終了フィールドに前月と来年の数字を入力します。

  4. 保存をクリックして変更した内容を保存します。

変更内容についてサービス利用部門の登録者にメールが送信されます。
