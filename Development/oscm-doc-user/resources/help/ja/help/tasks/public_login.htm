ログイン

ログインページに必要な情報を入力してログインします。ログインページを開くには、Webブラウザのアドレスバーにアクセス用URLを入力するか、対応するブックマークを選択します。また、メールのリンクをクリックして、受け取ったログインデータでアクセスすることもできます。

ログインするには：

  1. 利用者IDとパスワードを入力します。
  2. ログインをクリックするかリターンキーを押します。

    直接ログインするか、初回ログイン時に初期パスワードを変更するようにプロンプトが表示されます。初期パスワードを変更することを強くお勧めします。

間違ったパスワードで3回ログインに失敗すると、アカウントがロックされる場合があります。これは、利用者情報を外部認証システムで維持管理しているかどうかによります。利用者情報を外部認証システムで維持管理している場合、そのシステムでのみパスワードの変更やリセットができます。利用者情報をプラットフォームで維持管理している場合は、パスワードをリセットできる管理者に連絡してください。次回ログインするための新しい一時パスワードが与えられます。

パスワードを忘れた場合は、ログインページのパスワードを忘れた場合をクリックしてください。利用者IDに対して新しいパスワードを設定できます。利用者アカウントがロックされている場合もしくは外部認証システムを用いて利用者情報を組織が管理している場合、新しいパスワードを設定することはできません。

利用者IDを忘れた場合には、管理者に連絡してください。管理者は所属組織に登録されているすべての利用者のIDを検索することができます。

セッションの有効期限が過ぎている場合は、ログインし直してください。
