サービス提供部門の管理

サービス提供部門の管理ページでは、選択したサービスポータルに誰がサービスを公開できるかを決定できます。 サービスポータルを許可されたサービス提供関連の部門のみにアクセスを許可するかしないかで、オプションの意味が変わります。

  • 許可されたサービス提供関連の部門のみ: サービスポータルが許可されたサービス提供関連の部門のみ公開を許可した場合、1つまたは複数のサービス提供部門、サービス仲介部門またはサービス提供代行部門にサービスポータルへのサービス公開を許可する必要があります。サービスポータルが登録されると、サービスポータル運用部門として指定された組織は自動的にサービスを公開できるようになり、サービス提供関連部門のロールが割り当てられます。

    サービス提供関連の部門に支払遅延があるなどの場合に、そのサービス提供関連部門に対する公開許可を取り消すことができます。そのサービス提供関連の部門がすでにサービスポータルにサービスを公開している場合、これらのサービスは自動的に無効化され、サービス利用部門は以後これらのサービスを購入できなくなります。既存の購入済サービスには影響はありません。

  • すべてのサービス提供関連の部門: すべてのサービス提供部門、サービス仲介部門またはサービス提供代行部門がサービスポータルにサービスを公開できる場合、特定のサービス提供関連の部門に対して、サービスポータルへのサービス公開を無効化することができます。これは、サービス提供関連の部門の支払いが遅延している場合などに必要とされます。サービス提供関連の部門がすでにサービスポータルにサービスを公開している場合、これらのサービスは自動的に非公開となり、サービス利用部門は以後これらのサービスを利用できなくなります。既存の購入済サービスには影響はありません。

    サービスポータルから除外されたサービス提供関連の部門に対してはいつでもサービス公開を再度許可することができます。これは、サービス提供関連の部門の支払遅延が解消した場合などに必要とされます。

サービスポータルのサービス提供関連の部門を管理するには:

  1. サービスポータル一覧からサービス提供関連の部門を管理するサービスポータルを選択します。

    一覧には、サービスポータル運用者がアクセスできるすべてのサービスポータルが含まれます。

    サービスポータルの種類にはすべてのサービス提供関連の部門がサービスを公開可能かまたはサービスポータルが許可された部門のみがサービスを公開可能かが表示されます。

  2. 選択したサービスポータルの設定に応じて、以下の作業を行います。:
    • 許可されたサービス提供関連の部門のみがサービスを公開可能: サービスの公開を許可するサービス提供関連の部門の登録およびサービスの公開許可の解除をおこないます。詳細はこちらを参照してください。
    • すべてのサービス提供関連の部門がサービスを公開可能: サービスの公開を許可しないサービス提供関連の部門の登録およびサービス提供関連の部門に対するサービス公開の再許可をおこないます。詳細はこちらを参照してください。
