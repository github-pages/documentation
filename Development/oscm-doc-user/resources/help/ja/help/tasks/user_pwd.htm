パスワードの変更

パスワードの変更ページでは、プラットフォームに保存されているパスワードを変更できます。

注意：利用者認証に外部のシステムを使用している場合は、パスワードの変更方法を組織内の管理者に問い合わせてください。

パスワードを変更するには：

  1. 該当するフィールドに、現在のパスワードと新しいパスワード（新しいパスワードは２回）を入力します。

    パスワードは6文字から100文字の長さで指定してください。

    パスワードを入力すると、そのパスワードの安全性がパスワードの強度フィールドに示されます。パスワードの強度が高く安全な場合、表示色は緑色になります。パスワードの安全性と強度を確保するには、最低8文字で数字、文字、特殊文字を含める必要があります。

  2. 新しいパスワードを保存するには、変更をクリックします。

新しいパスワードが保存されます。再度ログインする際に使用します。
