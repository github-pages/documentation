請求システムの詳細を参照

請求システムの参照ページでは、システム運用管理部門がプラットフォームに対して設定した請求システムを参照することができます。技術サービスを定義するときにこれらの請求システムを選択することができます。

請求システムの参照ページを最初に表示したとき、デフォルトに設定された請求システムの詳細が展開表示されます。システム運用管理部門は必要に応じて別の請求システムをデフォルトに設定します。

詳細情報は請求システムごとに表示されます。接続テストをクリックすると請求システムとの接続を確認することができます。接続できない場合は、システム運用管理部門に連絡して請求システムの動作確認、および接続性の確認を依頼してください。
