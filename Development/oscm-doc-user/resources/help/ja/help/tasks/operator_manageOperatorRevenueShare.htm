システム運用管理部門の売上配分設定

システム運用管理部門の売上配分設定ページでは、システム運用管理部門が管理しているサービスポータル上でサービス提供部門が提供しているサービスの売上配分率を設定できます。サービス固有の売上配分率はサービス提供部門が登録または更新される際に設定された標準のシステム運用管理部門の売上配分率よりも優先されます。

このサービス固有の売上配分率はサービス提供部門が直接サービスを提供したかどうか、つまりサービス仲介部門やサービス提供代行部門をとおしてサービスを提供したかどうかにかかわらず適用されます。 サービス提供部門がサービス利用部門に適用する割引は、システム運用管理部門の売上配分率から差し引かれます。

注: 売上配分と割引はサービスを使用しているサービス利用部門に対する請求データに基づいて計算されます。 請求データはプラットフォームの請求システムを使用しているサービスに対してのみ計算されます。外部の請求システムを使用しているサービスは計算には含まれません。

サービス固有のシステム運用管理部門の売上配分率を設定するには:

  1. 売上配分率を設定するサービスを選択します。 一覧にはプラットフォーム上のすべてのサービスが含まれます。サービス名にはそのサービスを提供しているサービス提供部門の組織IDが合わせて表示されます。
  2. 表にはサービス提供部門に設定されている標準のシステム運用管理部門の売上配分率が表示されます。

    売上配分率 (%)の項目に、サービス固有の売上配分率をパーセント値で入力します。:

    このパーセント値は選択されているサービスに対して有効であり、サービス提供部門だけでなくサービス仲介部門やサービス提供代行部門がこのサービスを提供した場合にもこの値が適用されます。この値は組織の登録や組織の管理画面で設定された標準の値や以前に設定されていたサービス固有の売上配分率よりも優先されます。

  3. 保存をクリックします。

    設定された値はサービス利用部門がそのサービスを購入した際に適用されます。配当金額は次回以降の請求書作成処理で計算されます。
