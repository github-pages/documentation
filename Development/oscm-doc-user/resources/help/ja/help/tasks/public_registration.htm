登録

登録ページでは、アカウントの作成に必要なデータを入力します。

登録方法：

  1. メールアドレスを入力します。

    この情報は必須項目です。登録するとメールで通知されます。

    指定されたメールアドレスの書式が有効な構文か確認され、ドメイン名がIANA（Internet Assigned Numbers Authority）によって定義、管理されている標準に適合しているか確認されます。

    例: user.name@domain.arpa, user.name@domain.org, user@mycompany.lan.uk

  2. 利用者と組織のアカウントに関するデータを指定します。

    アスタリスク（*）のついたフィールドは必須項目です。

    • 希望する利用者ID：プラットフォームにログインするためのID。利用者IDは100文字以内で以下の文字を含めることはできません。

      !" # $ % & ' * + , / :; < = > ?\ ^ `

    • パスワード：プラットフォームにログインするためのパスワード。6文字から100文字の長さで指定してください。

      パスワードを入力すると、そのパスワードの安全性がパスワードの強度フィールドに示されます。パスワードの強度が高く安全な場合、表示色は緑色になります。パスワードの安全性と強度を確保するには、最低8文字で数字、文字、特殊文字を含める必要があります。

    • 性別：男性または女性。
    • 名：名。
    • 姓：姓。
    • 組織と所在地：所属組織の名称と所在地。
  3. 所属組織が所在する国名を選択します。
  4. 文字の確認ボックスにある文字列を読み取って入力します。大文字・小文字を区別しますので、注意してください。
  5. 条件とプライバシーポリシーを読み、該当するチェックボックスを選択して同意します。
  6. 登録内容を確認するには、登録をクリックします。

    登録プロセスが完了し、アカウントが作成されると、URLを含む確認メールが送信されます。

  7. アカウントを確認するには、確認メールの中にあるURLをクリックします。
  8. 登録ページで続行をクリックします。

    ログインページが表示され、指定した利用者IDとパスワードでログインできます。

    さらに、アカウントにアクセスするためのURLが記載されたメールが送信されます。今後のログインに使用できます。

登録が終了すると、登録者が所属組織の管理者ロールに自動的に割り当てられます。管理者ロールには、所属組織や利用者、購入済サービスの管理に必要な権限が与えられます。
