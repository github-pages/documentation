支払種別の追加

新しい支払種別ページでは、サービス提供部門またはサービス提供代行部門が利用できる支払種別を追加できます。ここで定義された支払情報を後でサービス利用部門に提供することができます。

支払種別を追加するには：

  1. サービスプロバイダの選択一覧から、選択されたサービス提供部門またはサービス提供代行部門に割り当てる支払種別を提供するPSPを選択します。

    この一覧には、利用しているプラットオームに登録され設定されているすべてのPSPが含まれます。

  2. [PSP識別子]フィールドで、PSPの識別子またはPSPから提供されたサービス提供部門またはサービス提供代行部門のアカウント番号を入力します。 サービス提供部門およびサービス提供代行部門はPSPと契約関係を結んでいる必要があります。PSPは、サービス提供部門またはサービス提供代行部門のチャネルを設定し、支払いがすぐにサービス提供部門またはサービス提供代行部門の口座にリンクされるように、通常アカウント番号などのIDを発行します。
  3. サービス提供部門またはサービス提供代行部門がサービス利用部門に提供することができる支払種別を選択します。 支払いの横に、選択されたPSPに対して提供される支払種別が表示されます。利用できる支払種別はPSPの設定によって異なります。
  4. 入力内容を保存するには、保存をクリックします。
    注: サービス提供部門またはサービス提供代行部門に割り当てられた支払種別は削除できなくなります。

組織を管理するページに戻ります。
