カスタム属性の追加

カスタム属性の追加ページでは、サービス利用部門や購入済サービスのためのカスタム属性を追加できます。

カスタム属性を追加するには:

  1. 属性IDに属性の識別子または名前を入力します。 属性IDは40文字以内に制限され、以下の文字は使用できません。:

    ! " # $ % & ' * + , / : ; < > \ ^ `

  2. 既定値に属性の標準として用いる値を入力します。

    255文字まで入力できます。

    既定値は個々のサービス利用部門や購入済サービスに値が設定されていない場合に使用されます。サービス利用部門の管理者によって値が設定された場合(下記、手順3を参照)、その値はサービスの購読時にデフォルトの値としてプリセットされます。

  3. 利用者による入力チェックボックスで、サービス利用部門の管理者またはサービス購入管理者が提供しているサービスを購入する際にそのカスタム属性を表示するかどうかを選択します。属性を表示した場合、管理者またはサービス購入管理者はサービス購入時に属性に対して値を設定できます。
  4. 必須チェックボックスで、サービス利用部門の管理者またはサービス購入管理者が提供しているサービスを購入する際にそのカスタム属性の値を入力必須にするかどうかを選択します。 上記手順3の利用者による入力が選択されている場合のみ、このチェックボックスが選択可能となります。
  5. 保存をクリックして入力内容を保存します。

カスタム属性の管理ページが表示され、新しいカスタム属性が画面に表示されます。

新しい属性の名前と値は、次回サービス利用部門の請求書作成処理がおこなわれる際に請求書に記載されます。属性の値が設定されていない場合、属性名のみが値無しで請求書に記載されます。
