ステージのカスタマイズ

ステージのカスタマイズページでは、広告、アイキャッチャー、詳細情報などを配置できるサービスポータルのホームページの領域をカスタマイズできます。

サービスポータルのステージはイメージまたはアニメーションにリンクするHTMLコードなどとして提供されます。サポートされる各言語に対して別々のステージを使用できます。プレビュー機能を使用するとステージが公開される前に参照できます。プレビューを確定するとただちにステージが公開され、サービスポータルの全訪問者に表示されます。

サービスポータルのブランド設定もレイアウト（色、フォント、背景イメージ、ロゴなど）とテキスト（ユーザーインターフェース要素のラベル、メールメッセージのテキストなど）で構成されます。サービスポータルのレイアウトは、サービスポータルメニューのレイアウトのカスタマイズメニューオプションでカスタマイズできます。ユーザーインターフェースのテキストは、サービスポータルメニューのテキストのカスタマイズメニューオプションでカスタマイズできます。

サービスポータルのステージをカスタマイズするには：

  1. ステージをカスタマイズするサービスポータルをサービスポータル一覧から選択します。

    一覧には、サービスポータル運用者がアクセスできるすべてのサービスポータルが含まれます。

  2. カスタマイズされたステージを提供する言語を選択します。

    ステージはサポートされているすべての言語で提供できます。

  3. 対象イメージを参照するHTMLコードを入力します。特定言語にイメージを指定しない場合、デフォルト言語のステージが使用されます。デフォルト言語にカスタマイズされたステージがない場合には、インストール後に利用できるデフォルトイメージが使用されます。

    ステージの推奨される幅は100%で、推奨される高さは200～300ピクセルです。

    例： <IFRAME FRAMEBORDER\="0" SRC\="yourstage.htm" NAME\="stage" WIDTH\="100%" HEIGHT\="280px" SCROLLING\="no"></IFRAME>

    HTMLコードを直接入力するか、サービスポータルホームページのHTMLコードで必要なステージを組み込むインラインフレームを使用できます。

    注意事項：インラインフレームの使用をお勧めします。インラインフレームを使用しない場合、HTMLコードとしてスニペットのみを指定します。<html>、<head>、または<body>などの要素を含む完全HTMLページは使用しないでください。プレビュー機能でページレイアウトを確認します（以下の次の手順参照）。スニペットにスクリプト（JavaScriptsなど）が含まれている場合には、サービスポータルホームページ、プレビューダイアログ、またはステージのカスタマイズページのレイアウトが崩れることがあります。

    ページレイアウトが正しいか注意してください。

  4. プレビューをクリックして、変更されたサービスポータルのホームページを表示します。
  5. 問題なければ、はいをクリックして、新しいサービスポータルのホームページの公開を確定します。利用者が次回サービスポータルにアクセスした際に、新しいステージが表示されます。

    いいえをクリックすると、ステージのカスタマイズページに戻り、さらに追加変更を行うことができます。既存のステージは変更されません。
