サービスポータルへのアクセス権を管理する

アクセス権の管理ページでは、どのユーザーがサービスポータルへアクセスできるかを設定することができます。

サービスポータルが登録されると、サービスポータルは公開され、匿名および登録ユーザーすべてがアクセスすることができます。サービスポータルへのアクセスを特定の登録組織に所属するユーザーに制限することができます。 アクセス制限されたサービスポータルに公開されているサービスの参照、購入、および使用にはログインが必要です。新規に登録を行うサービス利用者は当該サービスポータルにおいては自分自身で登録を行うことはできません。サービス提供部門、サービス提供代行部門、サービス仲介部門、またはシステム運用管理部門が登録する必要があります。

公開されたサービスポータルを非公開にする場合、当該サービスポータルに既存の購入済サービスを持つ組織は自動的に当該サービスポータルへのアクセス権が与えられます。当該組織のアクセス権は、明にアクセス権を与えた組織と同様にいつでも無効にすることができます。この場合既存の購入済サービスは停止されます。

アクセス権の設定されたサービスポータルはいつでも公開してすべてのユーザーがアクセスできるようにすることができます。

サービスポータルのアクセス権を管理するには:

  1. サービスポータルドロップダウンリストよりアクセス権を設定したいサービスポータルを選択します。

    リストにはサービスポータル運用者としてアクセス権のあるすべてのサービスポータルが含まれます。

    アクセスを制限するオプションはサービスポータルが公開されているか、特定の組織だけにアクセスを制限しているかを設定します。アクセス制限を行っている場合、組織のリストが表示され、チェックボックスの選択された組織にアクセス権が与えられていることを意味します。

  2. アクセスを制限するオプションでサービスポータルのアクセスポリシーを設定します。
  3. サービスポータルへのアクセスを制限する場合、アクセス権を与える組織を選択します。 アクセス権を無効にする場合、対応する組織のチェックボックスを無効にします。当該組織が当該サービスポータルに購入済サービスを所有している場合は警告メッセージが表示されます。続行した場合購入済サービスは停止されます。
