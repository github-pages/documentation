サービス利用部門の確認

サービス利用部門の確認ページでは、サービス利用部門に現在設定されているデータを見ることができます。

閲覧可能な組織の情報は以下のとおりです:

  • 組織の名前
  • 組織の一般連絡用メールアドレス
  • 組織の電話番号
  • 組織のWebサイトのURL
  • 組織の所在地
  • 組織が所在する国
  • 組織の説明

サービス利用部門の情報を確認するには:

組織ID一覧から情報を確認するサービス利用部門を選択します。

一覧にはログインしている利用者の組織からサービスを購入しているサービス利用部門、または、サービスを購入しているしていないに関わらずログインしている利用者が登録したサービス利用部門が表示されます。

サービス提供部門としてログインしている場合、サービスの再提供設定を付与したサービス仲介部門からサービスを購入しているサービス利用部門も表示されます。

サービス利用部門の組織IDは組織名と組み合わせて表示されます。

組織のデータが表示されます。
