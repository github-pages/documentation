サービス提供部門の管理

サービス提供部門の管理ページでは、選択したサービスポータルに誰がサービスを公開できるかを決定できます。

公開されたサービスポータルを選択した場合、基本的にはすべてのサービス提供部門、サービス仲介部門またはサービス提供代行部門がそのサービスポータルに対してサービスを公開できます。

しかし、サービスポータル運用部門はブラックリストにサービス提供関連の部門を設定し、そのサービス提供関連の部門によるサービスの公開を禁止できます。例えば、サービス提供関連の部門に支払遅延があるなどの場合が挙げられます。すでにそのサービス提供関連の部門によってサービスが公開されている場合、それらのサービスは自動的に無効化され、サービス利用部門は以後これらのサービスを購入できなくなります。既存の購入済サービスには影響はありません。サービス提供部門の管理ページには公開を禁止されているサービス提供関連の部門がすべて一覧表示されます。

サービスポータルから除外されたサービス提供関連の部門に対してはいつでもサービス公開を再度許可することができます。これは、サービス提供関連の部門の支払遅延が解消した場合などに必要とされます。

サービス提供関連の部門に対してサービスポータルへのサービス公開を禁止した場合およびサービス公開を再度許可した場合、そのサービス提供関連の部門の管理者にその旨を知らせるメールが自動的に送信されます。

サービス提供関連の部門に対してサービスポータルへのサービス公開を禁止するには:

  1. 組織IDにサービス提供関連の部門の組織IDを入力します。 サービス提供関連の部門に連絡をとり組織IDを入手してください。
  2. 禁止をクリックし、禁止されたことを確認します。 公開の禁止はすぐに反映され、それ以降そのサービス提供関連の部門はそのサービスポータルにサービスを公開することはできません。

サービス提供関連の部門に対してサービス公開を再度許可するには:

  1. サービス公開を再度許可するサービス提供関連の部門のチェックボックスを選択します。

    一つまたは複数のチェックボックスが選択可能です。

  2. 許可をクリックします。

    変更はすぐに反映され、サービス提供関連の部門はそのサービスポータルにサービスを再度公開することができます。
