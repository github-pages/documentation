支払種別の追加

新規支払種別 ページでは、選択された支払代行システム（PSP）が提供する支払種別を定義することができます。ここで定義された支払情報は後でサービス提供部門およびサービス提供代行部門に対して提供することができます。これに対して、サービス提供部門およびサービス提供代行部門はサービス利用部門に対して提供する支払種別を選択できます。

支払種別を追加することはできますが、PSPにすでに定義された方法を削除することはできません。支払種別の名前は自由に定義できます。支払種別自体はPSPが提供します。

支払種別を追加するには：

  1. 名前 フィールドで、サービス提供部門、サービス提供代行部門およびサービス利用部門に表示する支払種別を入力します。

    名前には最大100文字まで指定できます。

  2. 支払種別 フィールドで、PSPが指定する支払種別のIDを入力します。
  3. 支払種別を保存するには、保存 をクリックします。
    注: 支払種別を保存すると削除できなくなります。

PSPを設定するページに戻ります。
