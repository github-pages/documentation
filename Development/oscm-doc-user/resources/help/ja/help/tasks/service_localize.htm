サービスのローカライズ

サービスのローカライズページでは、利用者向けサービス名、概要、説明をサポートされているすべての言語で提供できます。サービス向けに定義した場合、利用者が詳細や購入済みサービスを表示した際に表示されるカスタムタブの名前をローカライズすることができます。

要素をローカライズするためには、サービスを有効化されていない状態にする必要があります。ローカライズは、既存の購入済サービスには影響しません。変更内容はそれ以降に新しく購入されたサービスに対してのみ有効になります。サービス利用部門がサービスを購入できるようにするには、再度そのサービスを有効化する必要があります。

サービス仲介部門、サービス提供代行部門およびそのサービス利用部門にはただちに変更された言語の内容でサービスの内容が表示されます。サービス仲介部門およびサービス提供代行部門から購入された既存の購入済サービスには変更内容は反映されません。おこなったすべての変更はそれ以降に新しく購入された場合にのみ反映されます。

注: 料金設定の説明を英語以外の言語にローカライズしない場合、そのローカライズされていない言語に言語設定されているサービス利用部門に対し英語のテキストが表示されます。サービス利用規約も上記と同様に表示されます。

要素をローカライズするには：

  1. ソース言語を選択します。利用者向けサービス名、簡単な説明、詳細説明、カスタムタブの名前がソース言語で表示されます。
  2. ターゲット言語を選択します。 一覧にはサポートされているすべての言語が含まれます。
  3. 翻訳文を入力します。
    注: 書式を変更する際は、エディタが提供するコントロールのみを使用してください。手作業でHTMLタグを入力したり外部のスタイルシートやコンテンツを参照すると、ページレイアウトがエラーになる場合があります。
  4. OKをクリックして翻訳を確定します。

サービスを有効化すると、サービス利用部門の利用者はそれぞれに設定しているプロフィールの言語でローカライズされた項目を見ることができます。

サービス仲介部門、サービス提供代行部門およびそれらのサービス利用部門には変更後すぐにローカライズされた情報が表示されます。
