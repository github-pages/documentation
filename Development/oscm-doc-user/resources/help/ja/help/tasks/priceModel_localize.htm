料金設定要素のローカライズ

料金設定の説明のローカライズページでは、 すべてのサポートされている言語で料金設定要素を設定できます:

  • プラットフォームの請求システムで料金請求されるサービスに対して料金設定を定義する場合、説明とサービス利用規約をローカライズすることができます。
  • 外部の請求システムで料金請求されるサービスに対して料金設定を定義する場合、ローカライズすることができるのはサービス利用規約だけです。 その他の要素は外部の請求システムでローカライズする必要があります。

料金設定要素をローカライズしない場合、英語で表記が行われます。

料金設定の説明を英語以外の言語にローカライズしない場合、そのローカライズされていない言語に言語設定されているサービス利用部門に対し英語のテキストが表示されます。サービス利用規約も上記と同様に表示されます。

翻訳対象のサービスポータルへ登録済のサービスが無効化されている場合のみ、要素をローカライズできます。ローカライズは、既存の購入済サービスには影響しません。サービス利用部門は新しい購入済サービスに対してのみ変更内容を参照できます。

サービスを再提供設定を付与しているサービス仲介部門およびそのサービス利用部門はすぐに提供した言語で変更された料金設定を見ることができます。サービス仲介部門から提供されたサービスの既存の購入済サービスには影響はありません。いずれの変更も新しく購入されるサービスに対してのみ反映されます。

サービス提供代行部門は独自のサービス利用規約を提供することができることから、サービス提供代行部門に対してはローカライズされたサービス利用規約は反映されません。

料金設定の要素をローカライズするには：

  1. ソース言語を選択します。ローカライズ可能な要素がこの言語で表示されます。
  2. ターゲット言語を選択します。 一覧にはサポートされているすべての言語が含まれます。
  3. 要素の翻訳文を入力します。
    注: 書式を変更する際は、エディタが提供するコントロールのみを使用してください。手作業でHTMLタグを入力したり外部のスタイルシートやコンテンツを参照すると、ページレイアウトがエラーになる場合があります。
  4. OKをクリックして翻訳を確定します。
  5. 変更内容を保存するには、ローカライズをおこなったサービスの料金設定、サービス利用部門向け料金設定または購入済サービスの料金設定変更ページで保存をクリックします。

サービスが公開された後に、プロフィールにその言語が設定されているサービス利用部門はローカライズされた要素を参照できます。

サービス仲介部門、サービス提供代行部門およびサービス利用部門はすぐに変更された内容を見ることができます。
