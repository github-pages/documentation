購入済サービスの解約

購入済サービスの解約ページでは、サービス利用部門の購入済サービスを明示的に解約できます。これは、サービス利用部門がサービス利用規約を順守しなかったり、サービスに対する支払いを行わない場合などに必要となることがあります。購入済サービスの解約は、当該サービスに利用者が割り当てられている場合でも可能です。

購入済サービスを解約するには：

  1. ページ上部の領域の一覧から、解約する購入済サービスを選択します。 任意の有効な購入済サービスを選択することができます。 一覧には、直接サービスを利用している部門とサービス仲介部門を介してサービスを利用している部門の購入済サービスが含まれています。

    選択された購入済サービスの詳細が、ページ下部の領域に表示されます。

  2. 理由フィールドに、購入済サービスを解約する理由を記入します。

    購入済サービスが解約され次第、この旨を記載したメールがサービス利用部門の管理者に送信されます。また、サービス購入管理者が設定されている場合は、サービス購入管理者にもメールが送信されます。

  3. 解約をクリックします。
  4. 解約を確認します。

購入済サービスがただちに解約されます。解約されたことと解約理由についての通知メールがサービス利用部門の管理者およびその購入済サービスに設定されているサービス購入管理者に送信されます。その購入済サービスに割り当てられている利用者はサービスを利用できなくなります。
