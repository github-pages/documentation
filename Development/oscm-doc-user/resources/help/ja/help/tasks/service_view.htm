サービスの確認

サービスの確認ページでは、ログインしている利用者がサービスポータルに公開する権限を与えられているサービスの詳細を確認できます。

閲覧可能なサービスの情報は以下のとおりです:

  • サービスを公開した際にサービスポータルのサービスカタログに表示されるサービス名と要約。
  • サービスが提供する機能の詳細に関する説明。この内容はサービス利用部門がサービス詳細を確認する際にサービスポータルで表示されます。
  • サービス利用部門がサービスを購入する際に読むべきサービス利用規約。
  • サービス提供部門が定義した料金設定の詳細:
    • サービスの料金計算に外部の請求システムが使用されている場合、当該システムで定義されている請求条件と料金は表示ボタンをクリックすることで確認できます。
    • プラットフォームの請求システムが使用されている場合、サービスには無料または有料です。サービス利用部門がサービスをして課金される際に、この基本料金や利用料金が使用されます。詳細な説明はこちらをクリックしてください。すべての価格に付加価値税は含まれていません。

サービスを確認するには:

サービスID一覧から詳細を確認するサービスを選択します。

一覧にはログインしている利用者が提供する権限のあるサービスすべてが含まれます。

サービスの詳細が表示されます。

サービス利用部門がこのサービスを購入できるようにするには、まずそのサービスをサービスポータルに公開する必要があります。その後、サービスを有効化してください。
