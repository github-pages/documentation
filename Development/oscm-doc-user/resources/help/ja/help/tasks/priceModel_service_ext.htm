サービスの料金設定

サービスの料金設定ページで、サービスに対して料金設定の定義と編集を行うことができます。

外部の請求システムを使用して料金請求されるサービスを選択しています。料金計算のための料金設定は当該請求システムで定義、管理されています。プラットフォームでは、サービスに対する適切な料金設定をインポートする必要があります。プラットフォームでは料金設定の参照のみすることができ、編集することはできません。料金設定を変更するには、該当する情報を外部の請求システムで編集した後に再度インポートする必要があります。

サービス料金設定の定義、および編集方法:

  1. 料金設定を入力してくださいにおいて、外部のシステムから料金設定をインポートするためにインポートをクリックします。 請求システムアダプターがプラットフォームを外部の請求システムと接続し、正しい料金設定を使用可能な状態である必要があります。表示ボタンをクリックすると料金設定を参照することができます。
  2. 必要に応じて、サービス利用規約を編集します。サービス利用部門はサービスを利用する際にサービス利用規約を読んで、同意する必要があります。

    デフォルトでは、技術サービス開発部門のサービス利用規約が使用されます。このサービス利用規約を使用するか、別の規約を指定するか選択できます。

    料金設定をローカライズすることによって、後で他の言語でサービス利用規約を指定することもできます。

    サービス提供代行部門は独自のサービス利用規約を設定してサービスを提供することができます。サービス提供部門が設定したサービス利用規約を使用するか、サービス提供代行部門が設定した別の文を使用するかを選択します。サービス利用規約を変更しても、サービス提供代行部門のサービス利用規約は変更されません。

  3. 料金設定を保存するには、保存をクリックします。
  4. 別の言語でサービス利用規約を提供するには、 ローカライズをクリックします。

サービスを有効化すると選択したサービス利用部門で料金設定が使用可能になります。これは当該サービス利用部門のすべての購入済サービスに対して有効になります。

変更はサービス利用部門、サービス仲介部門、およびサービス提供代行部門に対してただちに反映されます。
