技術サービスのエクスポート

技術サービスのエクスポートページでは、設定した技術サービスの定義をエクスポートし、そのデータをXMLファイルに保存できます。必要に応じてXMLファイルを編集するか、ファイルを新しい技術サービス定義のサンプルとして使うことができます。ファイル内のXMLタグのフォーマットや使用法の詳細については、運用ガイド（技術サービス開発部門編）を参照してください。

技術サービスの定義をエクスポートするには：

  1. 定義をエクスポートする技術サービスのチェックボックスを選択します（複数選択可）。 複数の技術サービスを選択すると、それらの定義が同じファイルに書き込まれます。
  2. エクスポートをクリックします。
  3. XMLファイルに定義を保存するか、選択したエディタで直接開くかを選択します。
