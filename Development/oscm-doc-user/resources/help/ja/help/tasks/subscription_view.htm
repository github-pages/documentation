購入済サービスの確認

購入済サービスの確認ページでは、サービス利用部門が提供したサービスに関する購入済サービスの詳細を参照できます。サービス提供管理部門はそのサービス提供部門のサービスをサービス仲介部門が提供したサービス利用部門の購入済サービスについても詳細を参照できます。

購入済サービスに関する以下の情報を参照できます。

  • 利用開始日および購入済サービスに割り当てられている利用者数。
  • サービス利用部門が購入したサービスの詳細。
  • 利用している技術サービスの詳細。
  • サービスポータルへ登録済のサービスに定義されている料金設定の詳細。

購入済サービスの詳細を参照するには：

ページの上部領域の一覧から、詳細を参照する購入済サービスを選択します。

提供しているサービスを利用するサービス利用部門の任意の有効な購入済サービスを選択できます。サービス提供部門としてログインしている場合、そのサービス提供部門が提供するサービスを仲介しているサービス仲介部門のサービスを利用するサービス利用部門について、有効な購入済サービスも選択することができます。

購入済サービスの詳細がページの下部領域にただちに表示されます。
