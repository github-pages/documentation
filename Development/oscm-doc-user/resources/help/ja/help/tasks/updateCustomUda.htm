カスタム属性の編集

カスタム属性の編集ページでは、選択したカスタム属性を編集または削除できます。

カスタム属性を編集するには:

  1. 属性の内容を変更します。
    • 属性ID: 属性の識別子または名前。属性IDは40文字以内に制限され、以下の文字は使用できません。: ! " # $ % & ' * + , / : ; < > \ ^ `
    • 属性名および言語: 選択した言語でのカスタム属性の名前。当該言語でサービス利用部門に表示されます。
    • 既定値: 属性の標準として用いる値。255文字まで入力できます。既定値は個々のサービス利用部門や購入済サービスに値が設定されていない場合に使用されます。サービス利用部門によって値が設定された場合、その値はサービスの購読時にデフォルトの値としてプリセットされます。
    • 利用者による入力: サービス利用部門の管理者またはサービス購入管理者がサービスを購入する際にそのカスタム属性を表示するかどうかを選択するためのチェックボックス。属性を表示した場合、管理者またはサービス購入管理者はサービス購入時に属性に対して値を設定できます。
    • 必須: サービス利用部門の管理者またはサービス購入管理者がサービスを購入する際にそのカスタム属性の値を入力必須にするかどうかを選択するためのチェックボックス。
    • コントローラーID: OpenStackやAmazon Web Services (AWS)といったIaaSサービスを提供するために使うサービスコントローラーのID。
    • 暗号化: ログ、レポート、およびDBにおける属性値の暗号化有無。有効にするとサービス利用部門の管理者が入力した値が隠蔽されます。チェックボックスがあらかじめ選択されている場合、設定を変更することはできません。
  2. 保存をクリックして、変更を保存します。

カスタム属性の管理ページが表示され、更新されたカスタム属性の内容が画面に表示されます。

更新された属性の名前と値は、次回サービス利用部門の請求書作成処理がおこなわれる際に請求書に記載されます。属性の値が設定されていない場合、属性名のみが値無しで請求書に記載されます。

カスタム属性を削除するには:

  • 削除をクリックして削除されたことを確認します。

カスタム属性は即座に削除され、カスタム属性の管理ページが表示されます。それ以降にサービス利用部門の請求書作成処理がおこなわれる際には属性の内容は請求書に記載されません。
