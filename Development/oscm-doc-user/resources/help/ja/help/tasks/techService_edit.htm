技術サービス説明の編集

技術サービス説明の編集ページでは、技術サービスの説明文を変更できます。変更は、その技術サービスに準拠している、新たに登録されたサービスに対して有効になります。

入力したテキストは、利用者プロフィールで設定した言語で保存されます。利用者プロフィールの編集方法の詳細については、プロフィールの編集を参照してください。

技術サービス定義を更新するには：

  1. 更新する技術サービスを選択します。 技術サービス定義の編集可能な部分が表示されます。
  2. 以下の情報を入力または編集します。
    • アクセス情報：このテキストは、この技術サービスに準拠したサービスをサービス利用部門が使い始めるときに表示されます。テキストはユーザーインターフェースに表示されます。 また、技術サービスに準拠したサービスが利用者に配信するメールにも記載されます。技術サービスがダイレクトアクセスもしくはユーザアクセスの場合には、利用者はプラットフォーム経由でサービスにアクセスできないため、説明が必要になります。紹介文やログインページのURLなどを提供するために使用します。

      注意事項: 以前に英語のアクセス情報が設定されていない場合、現在使用している言語の設定に関係なく、入力されたアクセス情報は英語にも設定されます。英語はプラットフォームの標準の言語であり、利用者の使用言語に対して説明が設定されていない場合、利用者には英語の説明が表示されます。

    • 説明：このテキストは、この技術サービスに準拠したサービスを登録するサービス提供部門に表示されます。サービス提供部門は提示される説明を使用するか、別の説明を定義することができます。
    • サービス利用規約：このテキストは、この技術サービスに準拠した料金設定をサービス提供部門が設定するときに表示されます。サービス提供部門は提示されるサービス利用規約を使用するか、別のサービス利用規約を定義することができます。
    • タグ：タグはアプリケーションを特徴付ける検索用語です。サービスポータルでは、サービスポータル運用者によってタグ表示がサービスポータルで有効になっている場合に、サービス利用部門はタグを使用して、特定の技術サービスに準拠したサービスを検索することができます。

      コンマで区切った最大5つまでの用語を入力できます。タグは大文字と小文字を区別しません。20文字以内です。

      最も人気のあるタグの表示リンクから、利用できるすべての技術サービスで多く使用されるタグを参照できます。

    注: 書式を変更する際は、エディタが提供するコントロールのみを使用してください。手作業でHTMLタグを入力したり外部のスタイルシートやコンテンツを参照すると、ページレイアウトがエラーになる場合があります。
  3. 技術サービスに定義されたパラメーターやオプションの説明文を入力または編集します。 このテキストは、サービス提供部門が技術サービスに準拠したサービスを登録するときに表示されます。
  4. 技術サービスに定義されたイベントの説明文を入力または編集します。 このテキストは、サービス提供部門が技術サービスに準拠したサービスを登録するときに表示されます。
  5. 技術サービスにロールが定義されている場合、その名前や説明文を入力または編集します。 ロールはアプリケーションで特定の許可にマッピングすることができます。この技術サービスに準拠したサービスの料金設定では、ロールに応じた料金を定義できます。

    このテキストは、サービス提供部門が技術サービスに準拠したサービスの料金設定を定義するときに表示されます。

  6. 技術サービスに操作が定義されている場合は、操作の名前や説明文を入力または編集します。 システムのバックアップや特定アプリケーションの起動・停止など、Web経由でアプリケーションのシステムリソースにアクセスすることができます。

    このテキストは、この技術サービスに準拠したサービスをサービス利用部門が使い始めるときに表示されます。

  7. 変更を保存するには、保存をクリックします。
その技術サービスに準拠したサービスを新しく登録する際に、サービス提供部門に対してこれらの更新が反映されます。
