プロフィールの編集

プロフィールの編集ページでは、個人の利用者データの編集や使用言語を指定できます。また、所属組織の管理者ロールを割り当てられている場合、組織のIDを参照し、組織のデータを変更することもできます。

注意：このページのいくつかのフィールドが無効になっている場合、所属組織は利用者の管理認証に外部のシステムを使用していることを意味します。外部システムで管理されるデータを変更できるのは、その外部システムのみです。

利用者の情報を編集するには：

  1. 各フィールドを編集してデータを更新します。
    • 性別、名、姓：男性または女性、名、姓。
    • メールアドレス：メールアドレス。

      指定されたメールアドレスの書式が有効な構文か確認され、ドメイン名がIANA（Internet Assigned Numbers Authority）によって定義、管理されている標準に適合しているか確認されます。

      例: user.name@domain.arpa, user.name@domain.org, user@mycompany.lan.uk

    • 言語：ユーザーインターフェースや参考ドキュメント、メールなどを表示する言語。 利用可能な言語については、システム運用管理部門が定義するサポートする言語に依存します。

      指定した言語がシステム運用管理部門によって無効化された場合、ユーザーインターフェース、利用可能なドキュメント、メール、およびレポートのラベルはデフォルトの言語である英語で表示されます。 また、プロフィールを更新する必要がある旨のメッセージが表示されます。 この場合、デフォルトの言語を選択して保存するか、もしくは他のサポート済み言語を選択して保存する必要があります。

  2. 変更を保存するには、保存をクリックします。
所属組織の情報を編集するには（管理者のみ）：
  1. 各フィールドを編集してデータを更新します。
    • 組織名：会社名などの所属組織の名称。
    • 代表メールアドレス：所属組織と通信するためのメールアドレス。例：info@mycompany.com。

      指定されたメールアドレスの書式が有効な構文か確認され、ドメイン名がIANA（Internet Assigned Numbers Authority）によって定義、管理されている標準に適合しているか確認されます。

      例: user.name@domain.arpa, user.name@domain.org, user@mycompany.lan.uk

    • サポート用メールアドレス: サービス提供部門、サービス仲介部門およびサービス提供代行部門の管理者に対してのみ設定できます。サービス利用部門が問い合わせをおこなう場合の送信先となるEメールアドレスを入力します。例: support@mycompany.com. Eメールアドレスを指定しない場合、利用者が問い合わせをおこないたい場合などの組織に対する一般的な通信は代表Eメールアドレスを通じておこなわれます。
      注: サービス仲介部門はプロフィール情報に組織のサポート用メールアドレスを設定できます。しかし、サービス利用部門がサービス関する問い合わせをおこなう場合、その問い合わせはサービスを提供しているサービス提供部門のサポート用メールアドレスに直接送られます。
    • 電話番号：所属組織の担当者の電話番号。
    • Webサイト：所属組織のWebサイトのURL。このWebサイトはサービス提供部門として機能する場合に参照されます。サービス利用部門が申し込むサービスを提供するサービス提供部門に関する詳細を表示する際、本Webサイトにアクセスされます。
    • プロフィール画像：画像のアップロードフィールドまたは参照をクリックし、そのイメージを含むファイルを選択します。サポートされるファイル形式はGIF、JPG、PNGです。プロフィールイメージの最大サイズは154x154ピクセルです。推奨サイズは154x60ピクセルです。サービス提供部門として機能する場合、プロフィールイメージはサービス利用部門にとって便利です。サービス利用部門がサービスに関する情報を要求すると、プロフィールイメージが富士通ブランドのサービスポータルにサービス提供部門情報の一部として表示されます。
    • 所在地：所属組織の所在地。
    • 国：所属組織が所在する国名。
    • 説明：所属組織に関する説明を入力します。入力したテキストは、利用者プロフィールで設定した言語で保存されます。テキストをローカライズするには、利用者プロフィールで言語を変更し、該当する翻訳を入力します。この説明は組織がサービス利用部門の場合に意味を持ちます。サービス利用部門は、申し込むサービスを提供するサービス提供部門に関する詳細を表示する際、本説明を参照できます。
  2. 変更を保存するには、保存をクリックします。
