推奨サービスの設定

推奨サービスの設定ページでは、サービスポータルのホームページで強調して表示したいサービスや宣伝したいサービスを指定できます。サービスポータル一覧から希望のサービスポータルを選択した後に、サービスポータルのホームページでどのようにサービスを表示するかをまず定義します:

  • 注目のサービスを選択: デフォルトでは、最も新しく作成された６個のサービスをサービスポータルのホームページに表示します。 残りのサービスは、サービスポータルの訪問者がサービスカタログを参照したときに表示されます。 ホームページでは、サービスポータルの訪問者が提供されているすべてのサービスにアクセスできます。

    サービスポータルのホームページで宣伝したいサービスを個別に選択したい場合に注目のサービスを選択オプションを使います。 表示するサービスの数、および、どのサービスを表示するかを指定することができます。 サービスの順番を定義することで、例えばホームページの一番最初に最も魅力的な商品を置くことによって、サービスをランク付けすることもできます。

    サービスを選択しランク付けする方法の手順については、下記を参照ください。

  • サービスと購入済サービスをカテゴリ毎にグルーピング: サービスポータルのホームページに表示するサービスと購入済サービスを選択することができます。 この機能を利用するには、ホームページの訪問者がそれぞれのアカウントでサービスポータルにログインする必要があります。 ログインすると直ぐに、利用者が所属する組織で利用できるサービスと既存の購入済サービスが表示されます。

    サービスと購入済サービスは、そのサービスポータルで定義されている最初の３つのカテゴリでグルーピングされます。カテゴリは作成された順番に表示されます。 最後に作成されたサービスが最初に表示されます。 サービスポータルに訪問する利用者が利用できるサービスと購入済サービスは、カテゴリでフィルタされます。 タグクラウドによるサービスのフィルタリングは利用できません。.

    注: サービスポータルでカテゴリが有効になっているか無効になっているかに関係なく、サービスポータルのホームページにサービスと購入済サービスを表示することができます。 サービスと購入済サービスをカテゴリ毎にグルーピングオプションは、サービスポータルを作成または更新する際に指定できるカテゴリ機能より優先されます。

    サービスと購入済サービスをホームページに表示する場合、そのサービスの基礎となるアプリケーションのリモートインターフェースのURLを技術サービス定義に指定することを推奨します。 さらに、サービス利用部門はそのサービスを一度しか購入できないようにすることを推奨します。 技術サービス定義に定義できる対応する属性の詳細については、運用ガイド　技術サービス開発部門編を参照ください。

サービスを選択してランク付けするには:

  1. 推奨サービスとして表示するサービスの数を選択します。標準では6が設定されています。
  2. トップページに表示するサービスを推奨可能なサービス一覧から推奨サービス一覧に移動させて設定します。その逆も同様です。

    サービスを一覧から一覧へ移動するには、サービスをクリックし追加または除去をクリックします。全て追加または全て除去をクリックすることで一度に全てのサービスを移動することができます。

    以下の点に注意してください:

    • 推奨可能なサービス一覧にはサービスは名前の昇順で表示されます。推奨サービス一覧にはサービスは利用者が指定した順序で表示されます（以降の手順を参照）。
    • 推奨可能なサービス一覧は有効化されたサービス（緑色で表示）と有効化されていないサービス（赤色で表示）の両方が表示されます。一方、サービスポータルのトップページには有効化されたサービスのみが表示されます。有効化されていないサービスが有効化された際には（例: サービス提供部門がサービスの定義更新を終えた時など）、そのサービスは即時にトップページに表示されます。
    • 推奨可能なサービス一覧には停止中のサービスは表示されません（例: サービス諸条件の違反などの理由でサービス提供部門がサービスを停止した場合など）。サービスポータル上でサービスが有効にされた場合は再度推奨サービスへの追加が必要です。
    • 推奨可能なサービス一覧には特定のサービス利用部門向け料金設定については表示されません。
    • 推奨可能なサービス一覧には一般サービスおよび一般に設定されていないサービスの両方が表示されます。一般サービスはサービスポータルにアクセスしたすべての利用者が見ることができます。一般に設定されていないサービスは登録されているサービス利用部門のみが見ることができます。詳細はこちらをご覧ください。
    • サービス一覧にはサービス提供部門、サービス仲介部門、サービス提供代行部門によってサービスカタログから削除されたサービスは表示されません。サービスポータルのトップページでは、サービス利用部門がサービスカタログに登録されているサービスのみをみることができます。詳細はこちらをご覧ください。
  3. 推奨サービス一覧上で推奨サービスの順序を設定します。

    位置を変更したいサービスを一覧からクリックします。サービスの位置を一つあげる場合や下げる場合には、一つ上に移動または一つ下に移動をクリックします。 サービスの位置を最初や最後に移動する場合には、一番上に移動または一番下に移動をクリックします。

  4. サービスポータルの推奨サービスの補完を設定します。これは有効化され推奨されているサービスの数が推奨サービスに表示するサービスの数に満たない場合に、サービスポータルのトップページに代わりに表示するサービスの設定です（2つ上の手順を参照）。選択肢は以下のとおりです:

    • 登録日付の新しいものを表示は定義された日付が最も新しいサービスで一覧を補完します。デフォルトではこの設定が選択されています。

    • 名前の昇順で表示は名前がアルファベットの順序で最初となるサービスで一覧を補完します。

    • レビュー評価の高い順で表示はレビュー評価が一番高いサービスで一覧を補完します。この設定はサービスポータルでレビュー機能が有効になっている場合のみ表示されます。もしこの設定を選択した後にレビュー機能が無効にされた場合、登録日付が新しいサービスで一覧が補完されます。

    • 他のサービスは表示しないは上記の手順2で設定されたサービスのみがサービスポータルのトップページに表示されます。

  5. 変更を保存するため、保存クリックします。サービスポータルの推奨サービス一覧の設定を初期化し標準の設定に戻す場合は、リセットクリックします。

推奨サービスの設定後、サービスポータルに訪れた利用者がサービスポータルに再度ログインするかもしくはウェブブラウザを再読み込みした際に、設定した順序でサービスが表示されます。
