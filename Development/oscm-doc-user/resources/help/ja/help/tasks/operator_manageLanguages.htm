言語の管理

本プラットフォームでは、利用者向けに複数の言語をサポートしています。インストール直後では、英語、ドイツ語、そして日本語を利用可能です。 システム運用管理者は、新たに言語を追加したり、既存の英語、ドイツ語、日本語のテキストをカスタマイズすることができます。追加可能な言語の一覧は、ここをクリックしてください。

言語バンドルは、以下のリソースで構成されています:
  • ユーザーインターフェースのリソース: ユーザーインターフェースおよび生成されたメールに現れるすべてのテキスト
  • オンラインヘルプ、および FAQのHTMLファイル: オンラインヘルプは管理ポータルから参照でき、FAQはサービスポータルから参照できます。
  • レポートのラベル: 生成されたレポートのコンテンツのラベルとテキスト。
言語を追加する場合および既存のテキストをカスタマイズする場合は、以下の手順に従います:
  1. 言語バンドルのすべてのリソースを翻訳する:

    • ユーザーインターフェースのリソースは言語の管理ページのエクスポート機能によって生成される Microsoft Excel形式のファイル上で翻訳、更新します。 詳細についてはここをクリックしてください。

    • オンラインヘルプとFAQファイルは、HTMLファイルを直接編集して翻訳します。詳細についてはここをクリックしてください。

    • レポートのラベルは、プロパティファイルを更新して翻訳します。詳細についてはここをクリックしてください。

  2. 新規言語を追加する場合: 本プラットフォームの言語の管理ページにて追加する言語を登録します。
  3. 翻訳および更新したMicrosoft Excel形式のファイルをインポートします。
  4. 翻訳および更新したオンラインヘルプおよびFAQのファイルを配備します。
  5. 翻訳および更新したレポートのラベルを配備します。
  6. 新規言語を追加する場合: 本プラットフォームの 言語の管理 ページにて、追加する言語を有効化します。プロフィールを編集する際には、有効化された言語すべてが選択肢として利用可能になります。

言語の管理 ページでは、以下のことができます:

  • 対応している言語の一覧を参照。デフォルトの言語は英語です。
  • プラットフォームに言語コードを追加して、言語を登録。これは、インストールした直後に利用できる言語(英語、ドイツ語、日本語)以外の言語で翻訳を提供するための前提条件です。
  • ユーザーインターフェースのリソースをエクスポートし、翻訳したものをプラットフォームにインポート。
  • 既存の翻訳を更新。
  • 言語の有効化または無効化。

言語を登録するには:

  1. ISO言語コード フィールドには、追加する言語のISOコードを2文字で入力します。 例えば、チェコ語であれば　cs　を入力します。

    本プラットフォームでは、左から右に綴られる言語をサポートしています。右から左に綴られる言語、縦書きの言語、およびイギリス英語やアメリカ英語のような言語の地域差には対応していません。追加可能な言語の一覧については、ここをクリックしてください。

  2. 追加をクリックします。 デフォルトでは、新規に追加した言語は有効になっていません。したがって、利用者はまだ追加した言語を選択することはできません。

以下の作業を行うことができます:

  1. ユーザーインターフェースのリソースをエクスポートし、それらを翻訳。
  2. 翻訳されたテキストおよび名前のインポート。
  3. オンラインヘルプとFAQ、そしてレポートのラベルを翻訳し、それらをオンラインヘルプとFAQ および レポートのラベル に記述されている方法で提供。

    リソースのサブセットだけを翻訳して提供することもできます。例えば、オンラインヘルプ、FAQ、メール、およびレポートのテキストはハンガリー語にするが、ユーザーインターフェースは英語のままにしておくことができます。利用者がプロフィールでハンガリー語を選んだ場合、ユーザーインターフェースは英語のままとなりますが、その他のテキストはハンガリー語で表示されます。

  4. 新規に追加した言語を有効化し、利用者が作業用の言語として選択できるようにします。

ユーザーインターフェースのリソースをエクスポート、翻訳、そしてインポートするには:

  1. エキスポート をクリックし、リソースを.xls形式のMicrosoft Excelファイルでファイルシステムに保存します。
  2. 必要に応じてリソースを翻訳します。詳細についてはここをクリックしてください。
  3. 翻訳したMicrosoft Excel形式のファイルをインポートします:

    1. ファイル名フィールド、または 参照をクリックし、翻訳が入力されているファイルを選択します。

    2. インポートをクリックします。

    カスタマイズしたファイルが、Microsoft Excel形式のファイルであり、３つのワークシートを含んでおり、各ワークシートには５つの列（キー、３つのシステムの標準言語 de, en, ja および新規に追加する言語）があることを確認してください。

    追加する言語に対してラベルや文字列が入力されていない場合、英語の用語が使われます。英語の用語も定義されていない場合、キーの名前が使われます。

  4. 保存 をクリックして、変更を有効にします。

既存の言語の翻訳を更新するには:

  1. 更新したい言語の行をクリックします。

    利用可能な言語であれば、システムのデフォルト言語（de, en, またはjp) も含めてどれでも選択することができます。

  2. エキスポート をクリックし、リソースを.xls形式のMicrosoft Excelファイルでファイルシステムに保存します。
  3. 選択した言語のコードがヘッダーに入力されている列には、既存の翻訳が入力されています。必要に応じて文字列を編集します。詳細については、ここをクリックしてください。
  4. 更新したMicrosoft Excel形式のファイルをインポートします:

    1. ファイル名フィールド、または 参照をクリックし、更新した翻訳が入力されているファイルを選択します。

    2. インポートをクリックします。

  5. 保存 をクリックして、変更を有効にします。

言語を有効化するには:

  1. 有効化したい言語の横の 有効 チェックボックスを選択します。

    一度に複数の言語を選択することができます。

  2. 保存をクリックし、変更を有効にします。

有効化されたすべての言語が、プロフィールを編集する際に選択肢として利用可能になります。

言語を無効化するには:

  1. 無効化したい言語の横の 有効 チェックボックスのチェックを外します。デフォルトの言語である英語を無効化することはできません。

    保存をクリックして変更を確定すると直ちに指定された言語は利用できなくなる旨のメッセージが表示されます。

  2. 保存をクリックし、変更を有効にします。

無効化した言語を利用者がプロフィールで選択していた場合:

  • 利用者の言語は、デフォルトの言語である英語に設定されます。したがって、すべてのテキストはデフォルトの言語である英語で表示されます。
  • 利用者はプロフィールを更新する必要があります: デフォルト言語のままとするか、または他の利用可能な言語を選択できます。
  • 無効化された言語は、もう選択することはできません。
