無料試用期間

無料試用期間はサービスを無料で提供する期間を定義します。サービス提供者は、例えば、サービス利用者にサービスのテストをするために無料試用期間を定義します。この期間が終了すると、サービス利用者は料金設定に従って自動的に課金されます。

無料試用期間中に購入済サービスを一時停止した後に再開した場合、停止している間はサービスを利用することはできませんが、無料試用期間は延長されません。

サービスの料金設定が定額となっている場合、無料試用期間終了後の最初の定額料金期間は、無料試用期間がどれだけ定額料金の期間にかかっていようとも、定額料金の時間単位分が全額課金されます。この期間の課金は、無料試用期間終了時に適用される料金設定に従って計算されます。無料試用期間終了時に適用される料金設定とは、例えば、その時点での割り当てられている利用者の数、定義されているパラメーターの値、または利用者に割り当てられたロールなどです。

アップグレードやダウングレードオプションとして定義されているサービスの無料試用期間は、アップグレードやダウングレードした時点で無料試用期間が開始します。移行前のサービスの無料試用期間とは独立しています。
