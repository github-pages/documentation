サービス契約単価

各サービスに対して、サービス利用部門が利用期間に応じて支払う料金を定義できます。月、週、日、時間などの期間がサポートされています。サービス契約単価は、利用者数、イベント数、その他の利用データ量などの単価の影響を受けません。

例

サービス利用部門が月曜日正午にサービスに申し込んだとします。 購入済サービスは木曜日の午前0時に終了します。

料金設定には1日単位で10000円のサービス契約単価が設定されています。

この場合、購入済サービスの料金は下図のように計算されます。



従量課金の場合、サービス利用部門の料金は30000円となります。また、定額料金の場合、それぞれの日が全体として計算されるため料金は40000円となります。
