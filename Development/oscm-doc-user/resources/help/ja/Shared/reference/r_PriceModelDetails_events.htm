イベントの料金

料金設定では、請求対象イベントごとに料金を定義できます。例えば、サービスに対する利用者のログインやログアウト、特定のトランザクションの完了、特定のデータの作成・削除などが含まれます。サービスの実装や統合により、利用できるイベントが異なります。

このタイプでは次のような段階的な料金を適用できます。請求期間ごとの発生回数に応じてイベントに異なる単価を設定できます。 詳細については、ここをクリックしてください。

イベントの料金は選択された計算方式(従量課金、定額)とは別で扱われます。
