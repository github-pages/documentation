割引

特定のサービス利用部門に対して、 プラットフォームの 請求システムを使用している購入済サービスすべてに適用される割引を設定できます。現在あるいは将来の月に対して割引を設定することも可能です。一定の期間に制限することもできます。期間が終了する前になると、サービス利用部門がサービス提供部門に連絡するなどの対応ができるように、通知メールが配信されます。

割引は、購入済サービスの定価から差し引くパーセントとして定義されます。割引が有効な請求期間内に発生したサービス利用部門の費用すべてに対して適用されます。請求期間のすべての期間で割引が有効なのか一部の期間で割引が有効なのかは関係ありません。

割引はサービス提供部門がそのサービス利用部門に対して設定できます。それらの割引はそのサービス提供部門がサービスの再提供設定を付与しているサービス提供代行部門およびサービス仲介部門のサービス利用部門には適用されません。

割引は、同意済みの売上配分率から差し引かれることに注意してください。
