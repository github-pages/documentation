Adapting the Log Configuration

You may need to change the warning and error level shown in the logs. To do so you will need to adapt the log configuration level for the APP.

Proceed as follows:

  1. Open the <GLASSFISH_HOME>/domains/<DOMAIN_NAME>/config/log4j.app.core.properties configuration file.
  2. Find log4j.logger.org.oscm.app in the configuration file.
  3. Change the log level as desired to one of the below:
    • TRACE
    • DEBUG
    • INFO
    • WARN
    • ERROR
    Example: 
    log4j.logger.org.oscm.app=INFO
