Configuration Settings

The configuration settings for APP are provided in the following files in subfolders of the directory to which you extracted the oscm-app-install-pack.zip file contained in the oscm-integration-app-pack.zip package (<install_pack_dir>):

  • databases/app_db/db.properties
  • domains/app_domain/glassfish.properties
  • databases/app_db/configsettings.properties
The settings are described in detail in
  • Database Configuration Settings
  • APP Configuration Settings
