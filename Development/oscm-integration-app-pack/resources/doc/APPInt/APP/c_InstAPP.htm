Configuring APP

APP comes with a default configuration that can be adjusted BEFORE deployment; it can still be changed once the oscm-app container has been deployed.

Prerequisites and Preparation

Consider the following before you install APP:

  • Hardware and Operating Systems
  • Java and Ant
  • Application Server
  • Relational Database
  • Mail Server

Installation

The installation of APP consists of the following main steps:

  • Preparing the Software and Setup Utilities
  • Configuring APP
  • Setting up the Database
  • Setting up the Application Server Resources
  • Exchanging Certificates

The descriptions assume that you are using the APP setup utilities for setting up the database and the application server resources.
