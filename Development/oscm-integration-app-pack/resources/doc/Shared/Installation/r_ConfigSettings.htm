Configuration Settings

The configuration settings for APP are provided in the following files in subfolders of the directory to which you extracted the oscm-app-install-pack.zip file (<install_pack_dir>):

  • domains/app_domain/glassfish.properties

    Refer to GlassFish Configuration Settings for details.

  • databases/app_db/db.properties

    Refer to Database Configuration Settings for details.

  • databases/app_db/configsettings.properties

    Refer to APP Configuration Settings for details.
